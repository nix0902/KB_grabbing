- [ Help Center ](/) - / - / [ Paper Trading: general questions ](/support/folders/43000549214-paper-trading-general-questions/) - / [ What is a “non-tradable” symbol notification ](/support/solutions/43000719856-what-is-a-non-tradable-symbol-notification/) # What is a “non-tradable” symbol notification Non-tradable symbols are those you cannot trade in [Paper Trading](https://www.tradingview.com/support/solutions/43000516466-paper-trading-main-functionality/). Currently, these include: - Synthetic symbols calculated by TradingView, which have a purple label next to the market status - Indices, as they are generally not tradable. For example NASDAQ:NDX If you want to trade indices, you can select other instruments that track index performance. You can check the symbol type in the symbol search, next to the data feed name. - Symbols listed on the Indian exchanges. Using these symbols for simulation trading violates the [NSE Data Sharing & Usage Policy](https://nsearchives.nseindia.com/web/sites/default/files/inline-files/NSE_DataUsageandSharingPolicy.pdf): "8.4. The Market Data shall not be provided to individuals/entities engaging in gaming, virtual trading, an activity of similar nature or any other activity which shall be prohibited by the applicable laws or regulator or any government/judicial authority as the case maybe." [](https://nsearchives.nseindia.com/web/sites/default/files/inline-files/NSE_DataUsageandSharingPolicy.pdf) [](https://nsearchives.nseindia.com/web/sites/default/files/inline-files/NSE_DataUsageandSharingPolicy.pdf) **! Note:** This restriction applies to Paper Trading only. It does not affect integration with Indian brokers that support live trading on these symbols.
