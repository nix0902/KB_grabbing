- [ Help Center ](/) - / - / Trading - / [ I've got issues/questions about OKX ](/support/folders/43000585946-i-ve-got-issues-questions-about-okx/) - / [ I can’t use instant trading with OKX ](/support/solutions/43000683200-i-can-t-use-instant-trading-with-okx/) # I can’t use instant trading with OKX To properly use the instant trading mode with OKX, please follow these steps: Enable Instant orders placement - On the chart page, click the cogwheel icon in the top right corner to open the settings menu. - Select the Trading tab. - Enable the Instant orders placement option. - Click OK to save changes and exit the settings menu. Configure Margin mode in OKX - Log in to OKX, and open the Order Ticket . - From the order ticket, choose Margin mode (required) . - Select either cross or isolated mode. That's it! You can now place orders using the chart layout with the selected margin mode from the order ticket. Previous Previous What should I do before I start trading through OKX on TradingView?
