- [ Help Center ](/) - / - / [ General questions (settings, data, and more) ](/support/folders/43000547584-general-questions-settings-data-and-more/) - / [ I want to use brackets on the chart ](/support/solutions/43000659802-i-want-to-use-brackets-on-the-chart/) # I want to use brackets on the chart You can pull TP (Take Profit bracket) or SL (Stop Loss bracket) on the chart, to set them to the desired price level. There is no need to open a ticket-order to place brackets. This feature is available with all brokers that support brackets on TradingView.
