- [ Help Center ](/) - / - / [ What instruments can I trade through Interactive Brokers? ](/support/solutions/43000669303-what-instruments-can-i-trade-through-interactive-brokers/) # What instruments can I trade through Interactive Brokers? To find all tradable symbols for Interactive Brokers, connect to them via the Trading Panel, then open the Symbol Search and, finally, make sure that the Interactive Brokers checkbox is active (this option is only available once you’ve logged into your brokerage account). Then, if you are looking for a specific symbol, simply use the search bar. Previous Previous Why do I get disconnected from the Interactive Brokers platform when I connect to them on TradingView? **Описание:** This is a **TradingView Symbol Search interface** for finding gold futures contracts. Here’s a detailed breakdown:  


### **Header & Navigation**  
- **Title**: “Symbol Search” (top-left).  
- **Close Button**: “X” (top-right) to exit the search.  


### **Search Bar**  
- **Input Field**: Contains the query “GOLD” (with a magnifying glass icon).  
- **Math Operators**: Buttons for `÷`, `-`, `+`, `×`, `^`, and `1/` (for mathematical calculations).  


### **Category Tabs**  
Tabs to filter symbols by asset class (selected tab is highlighted in blue):  
- `All`, `Stocks`, `Futures` (active), `Forex`, `Crypto`, `Indices`, `Bonds`, `Economy`.  
- **Broker Filter**: Checkbox for “Interactive Brokers” (checked, limiting results to this broker).  


### **Results Table**  
Columns: `SYMBOL` (left), `DESCRIPTION` (middle), and a right-aligned source column.  

#### **Symbol Group (GC)**  
- **Expand/Collapse**: “G” icon (left) to show/hide sub-symbols.  
- **Main Symbol**: `GC` (parent symbol) with description “GOLD FUTURES” (blue text).  
- **Source**: “futures COMEX” + US flag (right).  


#### **Futures Contracts (Sub-Symbols)**  
List of gold futures with expiration months (all from COMEX, US flag):  
- `GCK2022`: GOLD FUTURES (MAY 2022)  
- `GCM2022`: GOLD FUTURES (JUN 2022)  
- `GCN2022`: GOLD FUTURES (JUL 2022)  
- `GCQ2022`: GOLD FUTURES (AUG 2022)  
- `GCV2022`: GOLD FUTURES (OCT 2022)  
- `GCZ2022`: GOLD FUTURES (DEC 2022)  
- `GCG2023`: GOLD FUTURES (FEB 2023)  
- `GCJ2023`: GOLD FUTURES (APR 2023)  
- `GCM2023`: GOLD FUTURES (JUN 2023)  
- `GCQ2023`: GOLD FUTURES (AUG 2023)  


### **Source Filter**  
- “All sources” (with globe icon + dropdown arrow) to filter by data provider.  


### **Footer**  
- Text: “Simply start typing while on the chart to pull up this search box” (instructional).  


### **Purpose**  
This interface lets users search for and select gold futures contracts (e.g., `GC`, `GCK2022`) to view charts, trade, or analyze them in TradingView. The “Futures” tab and “Interactive Brokers” filter ensure results match the desired asset class and broker.
