Order presets are a tool that allows traders to speed up their trades, quickly switch between trading strategies, and easily place pending orders with preset parameters via hotkeys or Buy/Sell buttons on the chart. Order presets are a feature that will help you save time and simplify the implementation of your trading strategies. With order presets, you can save and instantly apply order settings, speeding up the trading process and always being ready for market changes.
