# Chart trading on TradingView: key features and advantages 
 Chart trading is a TradingView feature that consists of a new order type — projected orders — and the functionality for managing orders and positions directly on [Supercharts](https://www.tradingview.com/chart/). This provides a more intuitive and visual approach to trading compared to using only the order panel.

#### Chart trading components 

The functionality consists of:

- **Projected orders:** A special type of orders available on Supercharts. They allow you to view and set the parameters of a future order (price, order type, quantity, brackets) directly on the chart and then send the order to the broker
- **Order management:** The ability to modify the parameters of a placed order or an open position directly on the chart without locking the interface with an order ticket

#### Projected orders 

When preparing to open a position, you'll see:

- Order placement (confirmation) button
- Bracket addition buttons
- Quantity selection through the calculator
- Order type selection
- Project order cancellation

#### Order management 

If you want to change the parameters of an existing order or position, you'll see:

- Discard changes
- Confirm changes
- Bracket addition buttons
- Quantity change through the calculator
- Close order or position

#### Key rules and logic for both states 

**Freedom:** Editing on the chart doesn’t block the interface. You can move different parts of the order — the main order and brackets — in several steps before submission.

**Unified context with order ticket:** Changes to the projected order are immediately reflected in the order ticket and vice versa.

**Uniqueness:** At any given time, only one projected order can exist.

Trading modes:

- **With confirmation:** Editing or projecting an order on the chart triggers the appearance of the “Confirm” and “Discard” buttons. Changes are applied only after confirmation
- **Without confirmation:** Changes are applied instantly after completing an action — drag, cancel, or close

**Changing quantity:** You can change volume only for the parent order through the calculator directly on the chart.

**Validation:** Order price, quantity, and bracket price for projected orders are synchronized with the order panel. Errors and warnings are displayed on the chart.

Canceling changes:

- You can cancel changes only back to the original state before editing began; there’s no step back
- Cancellation occurs via the “Discard” button
- Deselecting the order doesn’t cancel the editing or projecting mode

Interaction
Most actions are performed directly on the charts, with changes automatically reflected across multiple layouts.

#### Entry point 

A projected order appears on the chart:

- In trading mode with confirmation, through the “+” menu or chart context menu
- When creating an order through the order panel — the order appears on the chart in any trading mode

An order (or position) becomes manageable on the chart:

- When attempting to change its parameters — price, quantity, or bracket — in trading mode with confirmation
- When editing is initiated through the context menu on an order or position from the chart in any trading mode
- When editing is initiated through the account manager

Multi-layout mode:

- The projected order is displayed on all charts of the same symbol in multi-layout mode
- On the active chart, the projected order is displayed with confirm and/or cancel buttons
- On inactive charts, these buttons appear when hovering over the order
- Changes across all charts are synchronized
