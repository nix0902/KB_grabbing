- [ Help Center ](/) - / - / [ How to start trading through Genial Investimentos on TradingView ](/support/solutions/43000771094-how-to-start-trading-through-genial-investimentos-on-tradingview/) # How to start trading through Genial Investimentos on TradingView Complete your registration Before trying to connect, make sure you have fully completed the registration process following the broker’s [instructions](https://suporte.genialinvestimentos.com.br/hc/pt-br/articles/42344307663124-Como-contratar-a-plataforma-Trading-View-na-Genial) and received the email with your login credentials for the integration. Allocate funds for the integration Don’t forget to allocate funds for the integration in the Trader Cockpit on the broker’s platform or app. For detailed instructions, please check [this](https://suporte.genialinvestimentos.com.br/hc/pt-br/sections/360008750152-Trader-Cockpit) section. How to reset password If you don’t remember your password, you can reset it using this [link](https://login.flexscan.com.br/tv).
