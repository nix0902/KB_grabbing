- [ Help Center ](/) - / - / [ General questions (settings, data, and more) ](/support/folders/43000547584-general-questions-settings-data-and-more/) - / [ Chart trading - order projection ](/support/solutions/43000736178-chart-trading-order-projection/) # Chart trading - order projection An order projection is a visual representation of a placed order that exists only on the chart and is not sent to the broker before placement. You can project an order that has not yet been placed directly from the chart. The following actions are available to you: - Selecting the price of the future order by dragging it on the chart - Selecting the quantity by clicking on the quantity and entering it through the calculator - Selecting the order type through the drop-down menu by clicking on the order name - Adding and editing future brackets by dragging them on the chart - Placing an order from the chart by clicking on the buy/sell button or by pressing Enter - Cancelling by clicking on the cross or by pressing ESC All changes to the future order on the chart are synchronized with the order ticket and vice versa. Pending orders are available for projecting on the chart - Limit, Stop, Stop Limit. Market order type is not supported for chart projection. Only one order can be projected at a time, creating another project order deletes the previous one if it has not been placed. Previous Previous What does the Featured label mean?
