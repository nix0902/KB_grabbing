- [ Help Center ](/) - / - / [ Selecting data type in broker settings ](/support/solutions/43000725196-selecting-data-type-in-broker-settings/) # Selecting data type in broker settings To improve trading experience with PhillipCapital Turkiye at TradingView you are provided with an opportunity to choose to receive real-time BIST_MIXED data at no extra cost. You can always turn real-time BIST_MIXED data on/off in the broker's menu.TradingView provides free real-time BIST exchange data for this broker integration’s users, however, it comes without volume. Turning off the use of real-time BIST_MIXED data will cause you to receive BIST data with a 15 minute delay, which includes volume. Please note, you will not be affected in any way if you already have a purchased subscription to the paid BIST data. Previous Previous Why can’t I cancel Take Profit / Stop Loss?
