Before trading cryptocurrencies, log into Broker’s Client Portal and request trading permissions. To do so, please take the steps outlined below. Click the User menu (head and shoulders icon in the top right corner) &gt; Settings &gt; Trading &gt; Trading Permissions. The Trading Permissions panel displays your current trading permissions. Select Add/Edit under Cryptocurrencies. If applicable, review the Paxos Customer Agreement, type your name in the box that reads Sign Here, then click Continue. Read and acknowledge the legal documents and disclosures by clicking each link. After reading through each document, type your name in the box that reads Sign Here, then click Continue. **Описание:** This TradingView interface displays a **grid of asset class tiles** for configuring trading instruments, organized in 3 rows (5, 5, 4 tiles) with consistent UI elements:  

### Core UI Elements per Tile:  
- **Icon**: Visual representation of the asset class (e.g., line graph for Stocks, piggy bank for Mutual Funds).  
- **Label**: Asset class name (e.g., “Stocks,” “Cryptocurrencies”).  
- **Info Button (ⓘ)**: Top-right corner; provides details about the asset class.  
- **Action Button**: Bottom of each tile, with two types:  
  - *Add/Edit*: For active/instantly tradable classes (e.g., Stocks, Bonds, Cryptocurrencies).  
  - *Request*: For restricted/conditional classes (e.g., NextShares, Complex Products, Event Contracts).  


### Tile Breakdown (Row-by-Row):  
1. **Top Row (5 Tiles)**:  
   - *Stocks*: Line graph icon; “Add/Edit” button.  
   - *Bonds*: Document icon; “Add/Edit” button.  
   - *Warrants*: Crossed lines icon; “Add/Edit” button.  
   - *Options*: Line graph icon; shows “Options level 1” (current access) + “Update Options Level” (upgrade button) + “Add/Edit” button.  
   - *Futures*: Wheat/barrel icon; “Add/Edit” button.  

2. **Middle Row (5 Tiles)**:  
   - *Futures Options*: Document + line graph icon; “Add/Edit” button.  
   - *Single Stock Futures*: Chart stack icon; “Add/Edit” button.  
   - *Mutual Funds*: Piggy bank icon; “Add/Edit” button.  
   - *Currency/Forex*: Currency symbols icon; “Add/Edit” button.  
   - *Metals*: Gold bars icon; “Add/Edit” button.  

3. **Bottom Row (4 Tiles)**:  
   - *NextShares*: “NextShares” label + arrow icon; “Request” button (restricted access).  
   - *Complex or Leveraged Exchange Traded Products*: Hourglass icon; “Request” button (restricted).  
   - *Cryptocurrencies*: Bitcoin icon; shows “Basic Crypto” (current access) + “Enroll in Crypto Plus” (upgrade button) + **“Add/Edit” button (highlighted in red)**.  
   - *Event Contracts*: Line graph icon; “Request” button (restricted).  


### Purpose:  
This screen lets users **manage trading instruments** by adding/editing active classes (via “Add/Edit”) or requesting access to restricted ones (via “Request”). The “Update Options Level” and “Enroll in Crypto Plus” buttons offer upgrades for enhanced functionality. Each tile’s icon/label clarifies the asset class, while the info button provides additional context.
