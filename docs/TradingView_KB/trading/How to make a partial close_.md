Partial close can be implemented only when the “Instant order placement” mode is disabled. To turn it off, you need to open Settings on the upper toolbar → Trading: After clicking “X” to close a position, check “partial close” in the confirmation pop-up and select the number of units: Kindly note that not all broker accounts support partial close. If the feature is not available, this option will not appear on the confirmation pop-up. Previous Previous Where can I see the spread?
