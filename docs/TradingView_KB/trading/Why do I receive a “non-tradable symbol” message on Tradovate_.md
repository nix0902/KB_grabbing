If you would like to trade futures on Tradovate via TradingView, you need to have a subscription for the real-time data on the **broker platform**. In case data package is purchased on TradingView, the orders can not be placed. However, we support data verification that can be enabled via Live Tradovate accounts, so you will not have to pay twice.
