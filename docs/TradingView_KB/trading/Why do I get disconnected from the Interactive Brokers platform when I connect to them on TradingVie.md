Interactive Brokers only allows for one active connection. Attempting to connect to a broker with an already active session will discontinue the initial connection. If you want to be logged in to TWS (or another platform) and connected to IBKR on TradingView with the same account at the same time, you can create an additional user for your IBKR account.
