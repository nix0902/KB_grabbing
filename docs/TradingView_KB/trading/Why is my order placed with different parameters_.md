- [ Help Center ](/) - / - / [ Why is my order placed with different parameters? ](/support/solutions/43000482781-why-is-my-order-placed-with-different-parameters/) # Why is my order placed with different parameters? Always pay attention to the main field in each section of the order ticket. The borders of those fields are highlighted. You set the value in this field manually. This value is fixed and does not change automatically. The remaining fields in each section are recalculated when the main field changes.
