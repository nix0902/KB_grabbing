# Filter by market

**URL:** https://www.tradingview.com/support/solutions/43000732562-filter-by-market/

---

- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / News - / [ News Flow ](/support/folders/43000594706-news-flow/) - / [ Filter by market ](/support/solutions/43000732562-filter-by-market/) # Filter by market The result of filtration by market is news related to selected market/markets. The drop-down contains the list of markets where news is presented. You can select one or few markets to filter news. If you set the stocks market, you can use an additional filter Stocks country . You can select one or few countries in this filter. The result of filtration by stocks country is news about stocks of selected countries. Additional filter is affected only on the stock market. It’s not applied to other markets. The drop down of stocks countries contains a list of popular countries. You can select one country in the drop down. If the stocks country is not set, the result of filtration is news about stocks of the entire world. Click to More countries… button opens dialog to search by all countries. You can select one or a few countries. Previous Previous Filter by symbol Next Next Filter by corporate news Launch Supercharts
