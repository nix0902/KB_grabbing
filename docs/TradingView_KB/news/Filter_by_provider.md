# Filter by provider 
 The result of filtration by the provider is news from selected news provider/providers.

The drop-down contains a list of popular providers. You can select one or a few filter values.

You can see the list of all providers after clicking the *Show more…* button.

This image shows a **provider selection modal** from the TradingView mobile app, designed to let users choose which news/data providers to enable for their trading experience. Here’s a detailed breakdown of the UI elements:  


### 1. Header Section  
- **Title**: “Provider” (bold, black text) at the top-left, indicating the modal’s purpose.  
- **Close Button**: A black “×” icon at the top-right, used to dismiss the modal without saving changes.  


### 2. Search Bar  
- A search field with a magnifying glass icon (left) and placeholder text “Search” (gray). This allows users to filter the list of providers by name.  


### 3. Provider List (Checkboxes)  
A vertical list of news/data providers, each with a checkbox (left) and provider name (right). Checkboxes are either **checked (blue with a white checkmark)** or **unchecked (empty gray square)**.  

- **Checked Providers**:  
  - *Reuters* (blue checkmark)  
  - *TradingView* (blue checkmark)  

- **Unchecked Providers** (gray empty checkboxes):  
  - MarketWatch  
  - Dow Jones Newswires  
  - Trading Economics  
  - MT Newswires  
  - Mace News  
  - MarketBeat  
  - Barchart  
  - Cointelegraph  
  - Beincrypto  
  - (Partially visible: Another provider, likely “...” or a truncated name)  


### 4. Action Buttons (Bottom)  
Three buttons at the bottom of the modal:  

- **“Select all”**:  
  - Icon: Two overlapping checkboxes (one with a checkmark, one without).  
  - Purpose: Toggle to select *all* providers (or deselect all, if already selected).  

- **“Cancel”**:  
  - Text: “Cancel” (black, outlined button).  
  - Purpose: Dismiss the modal *without* applying any changes (reverts to the previous provider selection).  

- **“Apply”**:  
  - Text: “Apply” (white text on a blue background, filled button).  
  - Purpose: Save the current checkbox selections and close the modal (applies the chosen providers).  


### Overall Purpose  
This modal lets users customize which news/data sources they want to see in TradingView (e.g., for market news, data, or analysis). Users check/uncheck providers, then confirm with “Apply” or cancel to exit. The “Select all” button streamlines bulk selection, and the search bar helps find specific providers quickly.
