# Filter by symbol

**URL:** https://www.tradingview.com/support/solutions/43000732561-filter-by-symbol/

---

- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / News - / [ News Flow ](/support/folders/43000594706-news-flow/) - / [ Filter by symbol ](/support/solutions/43000732561-filter-by-symbol/) # Filter by symbol The result of filtration by symbol is news related to selected symbol/symbols. The drop-down contains a list of symbols. You can select one symbol in the drop-down. Also you can reset selected values. Click on the More symbols… button opens the search symbol dialog. In the search dialog, you can select one or a few symbols. Symbol filters can’t be used together with a watchlist filter. If the watchlist filter is selected, the filter by symbol is disabled. You need to reset the watchlist filter to use the symbol filter. Previous Previous News Flow's filters overview Next Next Filter by market Launch Supercharts
