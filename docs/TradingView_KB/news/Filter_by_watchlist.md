# Filter by watchlist 
 The result of filtration by watchlist is news related to symbols from the selected watchlist.

The drop-down contains a list of user’s watchlists. You can select one watchlist in the drop-down. Also you can reset the selected value. You can’t select a few watchlists in the filter.

Filter by symbol and filter by watchlist can’t be used at the same time. If the watchlist filter is selected, the filter by symbol is disabled.
