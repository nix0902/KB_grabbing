# News Flow's filters overview 
 The filter panel is located above the list of the news. The following filters are available:

*Watchlist* - news related to symbols from the watchlist;

*Symbol* - the symbol mentioned in the news;

*Market* - the market mentioned in the news;

*Corporate news* - news by different types of corporate activity;

*Region* - region related to the news;

*Provider* - news provider.

If you set the stocks market, you can see an additional filter, *Stocks country*.

You can use one or a few filters to search for news and set one or a few values for the filter.

If filters are not set, the result of filtration is all available news in the user’s language.

You can reset values of any filter by clicking the Reset button.
