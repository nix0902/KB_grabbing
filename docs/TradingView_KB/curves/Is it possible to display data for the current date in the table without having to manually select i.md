Yes, it is possible. In addition to selecting a specific date in the calendar, the function of displaying data relative to the current day has been added. You can select one of the following preset options: - Current date - Yesterday's date - A week ago - A month ago - A year ago This simplifies working with the table and allows you to get up-to-date information faster.
