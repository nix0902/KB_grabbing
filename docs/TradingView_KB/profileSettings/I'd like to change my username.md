- [ Help Center ](/) - / - / [ I want to change my account information ](/support/folders/43000548578-i-want-to-change-my-account-information/) - / [ I'd like to change my username ](/support/solutions/43000482515-i-d-like-to-change-my-username/) # I'd like to change my username Every TradingView user is eligible to change a username twice. We don't allow multiple name changes to avoid any kind of social manipulation or confusion. To change your username, please go to Settings and billing - [Public profile](https://www.tradingview.com/settings/#public-profile), enter your new username and save the changes. Please be aware of the following when changing your username: - Don't choose a name that is occupied by another user - Don't choose a name that includes "admin," "moderator" or "tradingview" - Don't choose a name that violates our [House Rules](https://www.tradingview.com/house-rules/) - Use only English letters, numbers, underscores and hyphens If you’re unable to change your username, it means you’ve already reached your limit.
