First, login to TradingView by either using a backup code or from the device where you have an active session, disable two-factor authentication, and then re-enable it using the authenticator app or SMS verification from your new phone. You can read more about how to set up two-factor authentication [here](https://tradingview.freshdesk.com/en/support/solutions/articles/43000572460-how-to-configure-two-factor-authentication-to-my-tradingview-account).
