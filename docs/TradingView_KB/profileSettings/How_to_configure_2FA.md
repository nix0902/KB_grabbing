# How to configure 2FA 
 Two-factor authentication is an extra step to protect your account from hackers. When you enable two-factor authentication, after entering your password you'll also need to generate and input a special one-time code each time you log in.

**CONTENTS:**

- [Authentication via mobile app](#mobile)
- [SMS authentication](#SMS)
- [How to disable 2FA](#Disable)

There are two ways to receive a one time code for two-factor authentication, which you can use with your TradingView account:

- Generate a code using an authentication generator app
- Receive a code via SMS — a message will be sent to the phone number linked to your account

TradingView recommends using authenticator apps, as they are more secure and reliable than text messages. SMS messages may not reach certain countries and can be delayed.

In addition to one-time codes, you'll also receive a list of backup codes, which will be automatically generated after enabling two-factor authentication.
 **! Important: **Save your backup codes and store them in a secure place. They are the only way to regain access to your account if you lose your login credentials or cannot use your two-factor authentication (2FA) device. 
To activate two-factor authentication for your TradingView account, go to the "Privacy and Security" tab in your profile → [Account Settings](https://www.tradingview.com/settings/#account-settings).

#### 

#### Authentication via mobile app

To start using this option, click "Enable" in the authentication app field.

If you logged in through social media (e.g., your Google account) you need to set a password first:

- Click "Reset"
- Go to your email and follow the instruction to set a password
- Return to the TradingView page and click "Enable" again

You will see a message informing you that you'll receive a letter containing a code. You will need to manually type or paste this code into the dialog window.

Next, you need to scan a QR code using a third-party app.

We recommend using one of the following:

- Google Authenticator (available on [Google Play](https://play.google.com/store/apps/details?id=com.google.android.apps.authenticator2) and [App Store](https://apps.apple.com/us/app/google-authenticator/id388497605))
- [Microsoft Authenticator](https://www.microsoft.com/en-us/account/authenticator)
- [Duo Mobile](https://duo.com/product/multi-factor-authentication-mfa/duo-mobile-app)
- [Authy](https://authy.com/)

Then, paste the code generated by the app into the field provided.

If you have correctly entered the code from the authenticator app but the system isn't accepting it, please ensure that your devices are synced and showing the correct time.

In the next step, save the backup codes provided by the system.

TradingView recommends writing these codes down manually and keeping them in a safe place.

Do not rely on copying and pasting, and do not store them on your devices. Only after saving the codes should you click "I saved the codes."

If something happens to your backup codes, you can always generate new ones.
 **! Important: ** TradingView cannot recover your account if you lose access to your 2FA codes. You are solely responsible for keeping your authentication methods and recovery codes safe. By enabling 2FA, you acknowledge and accept that TradingView is not liable for any loss of access due to lost or inaccessible 2FA credentials. ** ** 
#### SMS authentication

If you choose to receive an SMS verification message for two-factor authentication, you will receive a text message for two-factor authentication with a verification code sent to the phone number linked to your account. The message you receive will be from TradingView.

** ** 
 ** ** 

If a phone number is not linked to your account, you will be prompted to enter one. To confirm that the number belongs to you, you will need to enter a code that will be sent via SMS or WhatsApp.

After successfully enabling two-factor authentication, you'll receive a list of one-time backup codes to use in case you don't have access to your smartphone. Please save these codes in a safe place so you don't lose access to your account.

#### How to disable 2FA

To disable two-factor authentication, go to the same "Account settings" tab and select "Disable" next to the two-factor method you're using.

If you change your mobile device, you'll need to re-configure two-factor authentication for your TradingView account:

- Log in to TradingView using either a backup code or a device where you have an active session
- Disable two-factor authentication
- Re-enable it using the authenticator app or SMS verification on your new device

If you want to change the phone number used for 2FA:

- Go to the "[Account settings](https://www.tradingview.com/settings/#account-settings)" tab in your profile settings
- Click "Disable" next to "Phone verification"
- Click "Enable" and enter a new phone number for 2FA
