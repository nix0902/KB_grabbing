# My phone number is in use inside another account 
 A phone number cannot be transferred because of technical and security reasons. 

However, if you have access to this account, you can log into it and replace the number with another one to make the necessary number free and verify it in the current account. 

Or, you can [delete the account](https://www.tradingview.com/support/solutions/43000550534-i-want-to-delete-my-account/) and after the deactivation process, the number will become free to use.
