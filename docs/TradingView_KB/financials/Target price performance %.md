# Target price performance %

**URL:** https://www.tradingview.com/support/solutions/43000734805-target-price-performance/

---

- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Financials - / [ Statistics ](/support/folders/43000572993-statistics/) - / [ Target price performance % ](/support/solutions/43000734805-target-price-performance/) # Target price performance % The Target price performance is a metric that measures the expected percentage change in a security's price over the next year, based on analysts' forecasts. It represents the difference between the [target price](https://www.tradingview.com/?solution=43000734804) and the current closing price. Target price performance % = (Target price / Closing price - 1) * 100 - Target price is the forecasted value of the security in 1 year, calculated as the arithmetic mean of the estimates predicted by the consensus of FactSet analysts. - Closing price is the current market price of the security. The formula calculates the percentage difference between the target price and the closing price. If the result is positive, it indicates that analysts expect the security's price to increase, while a negative result suggests that analysts expect the price to decrease.
