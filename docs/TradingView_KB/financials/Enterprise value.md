Enterprise Value is a metric that is used to assess the total value of a company by factoring in debt and cash. It is a less popular alternative to market capitalization, but it can give a more clear picture of a company’s true value. Formula: Price close * Common shares dil. + [Pref. stock](https://www.tradingview.com/?solution=43000563879) + [Total debt](https://www.tradingview.com/?solution=43000553622) + [Accumulated minority interest](https://www.tradingview.com/?solution=43000563884) - [Cash & Short-term investments](https://www.tradingview.com/?solution=43000563702) What does Enterprise value mean? It is believed that Enterprise value can be used to calculate how much a company will cost if it is bought out by another company.
