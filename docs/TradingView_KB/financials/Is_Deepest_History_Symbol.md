# Is Deepest History Symbol 
 When this option is enabled, the screener will keep only one trading symbol per bond — the one with the deepest and longest available price history across all exchanges.

Many bonds are listed and traded on multiple markets under different tickers. However, these tickers all represent the same underlying instrument. By selecting *Is Deepest History Symbol*, the system automatically identifies and retains only the ticker that has the the earliest available trading date.

This helps you:

- Avoid duplicate listings of the same bond across multiple exchanges
- Work with the most comprehensive historical dataset for analysis and backtesting
- Ensure cleaner, more consistent screening results
