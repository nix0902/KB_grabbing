- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Financials - / [ Statistics ](/support/folders/43000572993-statistics/) - / [ KZ index ](/support/solutions/43000597844-kz-index/) # KZ index What is the Kaplan-Zingales (KZ) index? The KZ index is a relative measure of a company’s dependence on external financing. Formula: KZ Index = -1.001909 x Cash flows / K + 0.2826389 x Q + 3.139193 x Debt / Total capital + -39.3678 x Dividends / K + -1.314759 x Cash / K Where: - Cash Flows = (Income before extraordinary items t + [Total depreciation and amortization](https://www.tradingview.com/?solution=43000563892) t ) - K = [Property, Plant, and Equipment](https://www.tradingview.com/?solution=43000563657) t-1 - Q = ([Market capitalization](https://www.tradingview.com/?solution=43000597015) t + [Total shareholder's equity](https://www.tradingview.com/?solution=43000557442) t - Book value of common equity t - [Deferred tax assets](https://www.tradingview.com/?solution=43000563683)) / Total shareholder's equity t - Debt = [Total long term debt](https://www.tradingview.com/?solution=43000553621) t + [Notes payable](https://www.tradingview.com/?solution=43000563600) t + [Current portion of long term debt](https://www.tradingview.com/?solution=43000563557) - Dividends = [Total cash dividends paid](https://www.tradingview.com/?solution=43000564183) t (common and preferred) - Cash = [Cash and short-term Investments](https://www.tradingview.com/?solution=43000563702) t Note that the model is a probabilistic model, so it will not perfectly predict which companies are financially constrained and which are not. In any case, it should help raise warning flags. What does the KZ index mean? The higher the KZ index, the more difficult the financial situation will be for a company when financial conditions worsen, as they may have difficulty financing current activities.
