# Bad debt / Doubtful accounts 

#### What is Bad debt / Doubtful accounts?
 Bad debt / Doubtful accounts represents the accumulated provision set aside for claims due within one year that are doubtful, past due, or unlikely to be collected.

It excludes Provision for doubtful long-term receivables.
