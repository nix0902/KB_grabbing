- [ Help Center ](/) - / - / Financials - / [ Common questions about financial data ](/support/folders/43000562459-common-questions-about-financial-data/) - / [ Earnings report date ](/support/solutions/43000761837-earnings-report-date/) # Earnings report date The Report date in the Earnings card on the chart shows the expected or confirmed date when a company will publish its earnings report. Exact vs. approximate dates Report dates can be: - Exact — received directly from the company or official sources. - Approximate — calculated by our data provider based on their methodology. An approximate date is marked with the “≈” symbol. How approximate dates are calculated - If a company has a historical reporting pattern, the date is estimated based on previous report dates. - If no clear pattern exists, the provider sets the date to Wednesday, one month after the end of the reporting period. Updating an approximate date If the approximate date has passed and no earnings report has been published: - The provider moves the date forward by one day. - This continues daily until: the actual report is released, or - an official confirmed date becomes available.
