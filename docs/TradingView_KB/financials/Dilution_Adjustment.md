# Dilution Adjustment 

#### What is Dilution Adjustment?
 Dilution Adjustment represents the maximum possible income reduction in case of all convertible securities, such as preferred stocks, convertible debt instruments, stock options and warrants, when used to calculate the [Net Income](https://www.tradingview.com/?solution=43000553617).
