# Dividends 
 Dividends contain the next fields:

*Date* - provides the Ex-Dividend date, which is the last date for shareholders to collect the most recent quarterly dividend.

*Gross* - provides the gross amount of Dividends paid. For countries with double taxation, it represents the dividend that shareholders received. After that, the shareholder is expected to pay the income tax on dividends. 

*Net* - provides the net amount of the Dividend. Available for countries with dividend imputation systems (example - Australia) and represents the franked dividend amount that shareholders received.
