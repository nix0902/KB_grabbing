- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Financials - / [ Bonds ](/support/folders/43000594677-bonds/) - / [ Coupon next reset date ](/support/solutions/43000729609-coupon-next-reset-date/) # Coupon next reset date The next date of coupon reset is the upcoming date on which the coupon rate on a variable rate bond or adjustable rate security is scheduled to be adjusted or reset. Variable rate bonds have coupon rates that are not fixed but instead fluctuate based on changes in a specified benchmark, such as a reference interest rate like LIBOR or the Treasury yield. The next date of coupon reset marks the point at which the bond's coupon rate will be recalculated according to the terms outlined in the bond's prospectus or offering documents. For investors holding variable-rate bonds, the next date of coupon reset is significant as it determines the new coupon rate that will apply for the upcoming period. Understanding when the coupon rate will reset allows to anticipate potential changes in their interest income and adjust their investment strategies accordingly.
