# Other Common Equity 

#### What is Other Common Equity?
 Other Common Equity represents [Common Equity](https://www.tradingview.com/?solution=43000563866) besides [Retained Earnings](https://www.tradingview.com/?solution=43000563867) and [Paid in Capital](https://www.tradingview.com/?solution=43000563871).

It includes:

- Unrealized Gain/Loss Marketable Securities;
- Cumulative Translation;
- Revaluation Reserves;
- Other Appropriated Reserves;
- Unappropriated Reserves;
- ESOP Debt Guarantee.
