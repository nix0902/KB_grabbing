# EBITDA per employee 
 EBITDA per employee is a financial metric that measures the company's [EBITDA](https://www.tradingview.com/?solution=43000553610) generated [per employee](https://www.tradingview.com/?solution=43000597080) on an annual basis. It provides insight into how efficiently a company is utilizing its workforce to generate earnings.

EBITDA per employee = EBITDA / Number of employees

This metric can be used to compare companies within the same industry or track a company's performance over time. A higher EBITDA per employee indicates that the company is generating more earnings per employee, which can be a sign of operational efficiency.
