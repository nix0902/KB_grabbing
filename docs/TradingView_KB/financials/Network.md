# Network 
 A blockchain network is a distributed system of nodes that exchange information and participate in the operation of the blockchain, which is a subtype of a chronologically organized transaction database. Public blockchains use various consensus mechanisms, such as Proof-of-Work, Proof-of-Stake, or Proof-of-History, to confirm transactions. This ensures decentralization, transparency, and reliability of the blockchain network.
