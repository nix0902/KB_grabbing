It is calculated as [Return on equity](https://www.tradingview.com/?solution=43000597021) divided by [Price to book ratio](https://www.tradingview.com/?solution=43000597016). It should be remembered that for the quarterly value of ROE, the LTM value of Net income is taken in the calculations.
