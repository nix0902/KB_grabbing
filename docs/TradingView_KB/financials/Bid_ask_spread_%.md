# Bid ask spread % 
 The Bid ask spread % measures the liquidity of a bond by showing the relative difference between its bid and ask prices. It is calculated using the formula:

Bid ask spread % = (Ask - Bid) / ((Ask + Bid) / 2) * 100

A smaller spread indicates higher liquidity, meaning the bond can be traded with minimal price impact.
