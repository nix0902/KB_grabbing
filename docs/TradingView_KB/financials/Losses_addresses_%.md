# Losses addresses % 

#### What is “Losses addresses %”?
 The percentage of addresses with realized losses out of all addresses currently holding this crypto.

#### Why is “Losses addresses %” important?

The losses' indicator provides total holders' realized losses for a particular crypto-asset. In contrast to the “in/out/at money” indicator, which covers unrealized positions.
