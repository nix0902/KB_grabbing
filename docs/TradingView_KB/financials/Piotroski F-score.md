- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Financials - / [ Statistics ](/support/folders/43000572993-statistics/) - / [ Piotroski F-score ](/support/solutions/43000597734-piotroski-f-score/) # Piotroski F-score What is the Piotroski F-Score? The Piotroski score is a discrete score between 0-9 that reflects nine criteria used to determine the strength of a firm's financial position. It was originally developed to give investors an idea as to which companies are the strongest when grouped with value stocks. This is done to help highlight the best and worst performers before adding these stocks to a portfolio. The financial strength of the companies is determined by applying these nine accounting-based restrictions and awarding one point for each restriction that a stock passes: A. Profitability - [Net Income](https://www.tradingview.com/?solution=43000553617): Give 1 point if Net income in the current period is positive; otherwise give 0 points. - [Operating cash flow](https://www.tradingview.com/?solution=43000553627): Give 1 point if Cash flow from operations in the current period is positive; otherwise give 0 points. - [Return on assets](https://www.tradingview.com/?solution=43000597054): Give 1 point if the current period's Return on assets (ROA) is higher when compared to the ROA of the previous period; otherwise give 0 points. - Quality of earnings: Give 1 point if the Cash flow from operations exceeds Net income before Extraordinary items; otherwise give 0 points. B. Leverage, Liquidity, and Sources of Funds - Decrease in Leverage: Give 1 point if there is a lower ratio of [Long term debt](https://www.tradingview.com/?solution=43000553621) in the current period compared to the value in the previous period; otherwise give 0 points. - Increase in Liquidity: Give 1 point if the current period's [Current ratio](https://www.tradingview.com/?solution=43000597051) is higher when compared to the Current ratio of the previous period; otherwise give 0 points. - Absence of Dilution: Give 1 point if the firm did not issue new shares / equity in the preceding period; otherwise give 0 points. C. Operating efficiency - Give 1 point if the current period's [Gross margin](https://www.tradingview.com/?solution=43000597811) is higher compared to the Gross margin of the previous period; otherwise give 0 points. - Asset turnover: Give 1 point if there is a higher [Asset turnover](https://www.tradingview.com/?solution=43000597022) ratio period on period (as a measure of productivity); otherwise give 0 points. What does Piotroski F-score mean? If a company has a score of 8 or 9, it is considered a good value. If the score adds up to between 0-2 points, the stock is considered weak.
