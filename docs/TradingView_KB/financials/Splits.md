# Splits 
 Provides information about events such as stock splits or rights issues with the ex-date of this event.

If there was a stock split it will provide how many new shares (numerator) the stockholder will have after the split takes place for every share held prior to the split (denominator).

For events like rights issue it returns the fraction that is calculated by the formula:

[(new term + old term) * previous day’s closing price]/[(new term * issue price) + (old term * previous day's closing price)]
