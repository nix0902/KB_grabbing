# AUM performance % 

#### What is AUM Performance?
 AUM performance indicates the percentage change in Assets Under Management (AUM) over a set period, showing a fund's asset growth or reduction.

#### How is AUM Performance Calculated?

The AUM performance is calculated using the following formula:

AUM performance % = (AUM at the end of period - AUM at the start of period) / AUM at the start of period * 100

The periods over which the return is calculated can be:

1M (1 month)

3M (3 months)

YTD (Year-To-Date)

1Y (1 year)

3Y (3 years)

5Y (5 years)
