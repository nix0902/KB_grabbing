# Deferred Income, Current 

#### What is Deferred Income, Current?
 Deferred Income, Current represents revenues received in a prior fiscal period for goods and/or services yet to be completed, delivered or provided. Revenues will be recognized as income within the current operating cycle and are to be carried as a liability until then.
