# Can I access real-time futures and options data from NSE and BSE? 
 Yes, futures and options from both NSE and BSE are available to all users. However, whether the data is real-time or delayed depends on your subscription type and account status.

Let’s break it all down by exchange.

#### BSE data 

- By default, BSE futures and options data is delayed for all users
- To access real-time data, you need to [purchase the corresponding data package](https://www.tradingview.com/data-coverage/)
- If you have already purchased the package, real-time BSE futures and options data will be available automatically

#### NSE data 

Here, the access depends on your user status.

For non-professional users, real-time NSE futures and options data is available by default.

For professional users:

- By default, NSE futures and options data is delayed
- To receive real-time data, you must sign the required exchange agreement
- If you have already signed this agreement in the past, you do not need to sign it again
