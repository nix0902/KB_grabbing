# I can’t find a particular DEX pool from the supported exchange on TradingView 
 Decentralized exchange (DEX) is a crypto exchange where transactions are executed on-chain directly between crypto traders.

You can find a dex pool on TradingView using the symbol search:

- Open the *Crypto* tab and choose the type of exchange – *DEX*;
- Or, open the *Crypto* tab and use the *Sources* list to find an exchange;
- Look pools up by token contract address (all available pools with this token will appear) or pool contract address in the symbol search;

To support the most popular and liquid pools on TradingView, we filter some of them out based on the following criteria:

- Pools’ liquidity and trading volume. We set liquidity and trading volume limits for each blockchain network and exchange. Any pool crossing the threshold will be added to our platform. Thresholds vary from dex to dex, depending on the trading activity of the exchange, and can be different each day. On the other hand, pools that lost liquidity and/or trading volume will be deleted from the platform regularly;
- Pools’ uniqueness. We only add most liquid pools with identical token composition. Once a new, more liquid pool appears, it will displace a similar, less liquid pool.
