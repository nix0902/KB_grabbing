# What is economic data? 
 Economic data are data points that describe the state of the country's economy as a whole (GDP, Inflation Rate), the state of a particular industry (Steel Production), or a system (ICU Beds).

At the moment, 300 metrics are available for 200 countries.

A complete list of available data can be found [here](https://www.tradingview.com/support/folders/43000581956-list-of-available-economic-indicators/).
