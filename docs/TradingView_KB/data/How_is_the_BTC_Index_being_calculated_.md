# How is the BTC Index being calculated? 
 The [BTC Index](https://www.tradingview.com/symbols/BTCUSD/?exchange=INDEX) is a [spread chart](https://www.tradingview.com/support/solutions/43000502298-spread-charts/) that tracks the average BTC/USD exchange rate since Bitcoin was born.

The most important value of this index is the deep data history, which goes all the way back to July 2010. This deep historical data is based on data of the former exchange Mt. Gox and several other exchanges of that time.

The more recent data is based on data from 4 key cryptocurrency exchanges: Bitstamp, Coinbase, Bitfinex and Kraken. The exact formula for this recent data is (BITSTAMP:BTCUSD + COINBASE:BTCUSD + BITFINEX:BTCUSD + KRAKEN:BTCUSD)/4.
