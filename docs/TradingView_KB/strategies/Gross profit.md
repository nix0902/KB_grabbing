- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Strategies - / [ Strategy report metrics ](/support/folders/43000599093-strategy-report-metrics/) - / [ Gross profit ](/support/solutions/43000681686-gross-profit/) # Gross profit The total realized profit generated by all winning trades within the strategy's selected period. This metric represents the absolute upside of the strategy before accounting for any losses. **Exclusions**: This value does not include profits from currently open positions and is not offset by any losing trades. **Calculation**: The cumulative sum of all positive entries in the Net P&L column (found in the List of Trades tab). **Commission handling**: If a commission value is set in the strategy settings, it is automatically deducted from each winning trade before being added to this total.
