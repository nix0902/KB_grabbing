To export strategy data, click on the “Download” button in the Strategy Tester. Information from each tab is exported separately: if you click on the "Download" button from the "List of Trades" tab, you will only export the trade information. Exporting strategy data from the "Performance Summary" tab will only export the metrics. Previous Previous What are strategies, backtesting and forward testing? **Описание:** This TradingView image displays the **Strategy Tester** tab (selected, gray background) within a trading platform, focused on an **RSI Strategy report** (expandable header with a red arrow).  

### Top Navigation & Header  
- **Tabs**: *Editor* (unselected), *Strategy Tester* (selected), *Trading Panel* (unselected) for switching between tools.  
- **Report Header**: “RSI Strategy report” with a caret (^) to expand/collapse the report.  


### Left Menu (Dropdown)  
A menu is open, offering strategy-related actions:  
- **Bar magnifier**: Toggle (gray, off) + info icon (?) for zooming chart bars.  
- **Add alert...**: Bell icon + “A” shortcut for setting price/condition alerts.  
- **Settings...**: Gear icon + “P” shortcut for strategy configuration.  
- **Download data as XLSX**: Download icon + red border (highlighted) to export report data to Excel.  
- **Open strategy...**: Folder icon + “O” shortcut to load a saved strategy.  


### Right Section (Report Content)  
- **Tabs**: *Analysis*, *Risk/performance ratios*, *List of trades* (for navigating report sections).  
- **Metric**: “Max equity drawdown” with an info icon (i) for definition, showing “174,450.00 MYR 17.29%” (peak-to-trough equity loss).  
- **Chart**: A line graph (red line with dots) tracking equity over time (x-axis: 2002–2015) with colored bars (red/pink/green) representing trade performance (e.g., wins/losses).  


### Purpose  
The interface is for backtesting an RSI trading strategy, with tools to analyze performance, set alerts, adjust settings, and export data. The highlighted “Download data as XLSX” option emphasizes data portability.
