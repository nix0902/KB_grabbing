---

- [ Help Center ](/) - / - / [ I'd like to know more about values in the Strategy Tester Report ](/support/folders/43000587044-i-d-like-to-know-more-about-values-in-the-strategy-tester-report/) - / [ Overview tab ](/support/solutions/43000681733-overview-tab/) # Overview tab This tab provides key performance metrics for the strategy. At the top of the tab are the metrics of the same name from the Performance Summary tab. Clicking anywhere in the Overview chart will bring your main chart to the close of the corresponding trade. The checkmarks in the bottom left corner allow you to specify the data that is displayed on the Overview chart. Using the buttons at the bottom right, you can choose between the Absolute and the Percentage format for the values displayed on the Overview chart. **Описание:** This TradingView image displays a **Performance Summary** tab of a backtesting or trading strategy analysis, with the following key elements:  


### 1. Top Navigation Tabs  
- **Tabs**: *Equity*, *Performance Summary* (active), *List of Trades*, *Properties* (for switching between views).  


### 2. Performance Metrics (Top Row)  
A series of key trading metrics, each with an info icon (ℹ️) for details:  
- **Net Profit**: 86.53 USD (0.01% gain, green text).  
- **Total Closed Trades**: 380 (total trades executed).  
- **Percent Profitable**: 37.89% (green text, % of profitable trades).  
- **Profit Factor**: 1.584 (green text, ratio of gross profit to gross loss).  
- **Max Drawdown**: 44.50 USD (0% of initial capital, red text, peak-to-trough loss).  
- **Avg Trade**: 0.23 USD (2.14% gain, green text, average profit per trade).  
- **Avg # Bars in Trades**: 29 (average duration of trades in “bars”/time units).  


### 3. Main Chart  
- **Chart Type**: Equity curve (green line) with a shaded area (light green) representing growth.  
- **Y-Axis (Left)**: Equity value (e.g., 1,000,000.00 to 1,000,090.00 USD).  
- **Y-Axis (Right)**: Drawdown (negative values, e.g., -44.00 to 0.00, showing peak-to-trough losses).  
- **X-Axis**: Trade number or time (1 to 373).  
- **Overlay**: Purple vertical bars (right side) represent drawdown magnitude (taller bars = larger drawdown).  


### 4. Bottom Controls  
- **Checkboxes**:  
  - *Equity* (checked): Shows the equity curve.  
  - *Drawdown* (checked): Shows drawdown bars.  
  - *Buy & hold equity* (unchecked): Would compare strategy performance to a “buy and hold” baseline.  
- **Toggle Buttons**: *Absolute* (active, shows raw USD values) vs. *Percentage* (shows % changes).  


### Purpose  
This view analyzes a trading strategy’s performance, highlighting profitability, risk (drawdown), and trade characteristics. The equity curve visualizes growth, while metrics quantify success (profit factor, % profitable) and risk (max drawdown).
