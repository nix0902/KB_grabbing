# Gross loss 
 The total realized loss generated by all losing trades within the strategy's selected period. This metric represents the absolute downside and is a critical indicator of risk management and strategy efficiency.

**Exclusions**: This value does not include losses from currently open positions. It only accounts for trades that have been fully closed and settled.

**Calculation**: The cumulative sum of all negative entries in the Net P&L column (found in the List of Trades tab).

**Commission handling**: If a commission value is set in the strategy settings, it is added to the magnitude of the loss. 

- *Strategy insight*: Reducing Gross loss is often more effective for increasing Net P&L than simply chasing higher Gross profit. Monitoring this metric helps identify if the strategy's risk-to-reward ratio is staying within acceptable limits.
