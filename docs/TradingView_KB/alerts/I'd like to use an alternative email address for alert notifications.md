- [ Help Center ](/) - / - / [ Alerts notifications ](/support/folders/43000547712-alerts-notifications/) - / [ I'd like to use an alternative email address for alert notificati… ](/support/solutions/43000474394-i-d-like-to-use-an-alternative-email-address-for-alert-notifications/) # I'd like to use an alternative email address for alert notifications You can use an alternative email if you'd like to receive your alerts notifications to another email address. Simply check the Send plain text box in the alert settings dialog and [specify an email address](https://www.tradingview.com/chart/?solution=43000474398) that you'd like to use. Note that some providers (e.g., T-Mobile and Verizon) offer this service for personal use only and might limit the number of messages you can receive. Please make sure to specify terms and conditions from the provider beforehand.
