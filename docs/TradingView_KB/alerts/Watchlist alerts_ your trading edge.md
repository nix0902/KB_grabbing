This feature allows you to use the same operators available for other types of alerts and saves your time while studying various symbols — from cryptocurrency to economy metrics — on [Supercharts](https://www.tradingview.com/chart/). Every watchlist alert functions independently for each symbol in the list. Such alerts trigger individually whenever the condition is met. The alert will stop only when it's been triggered for all symbols, until then, it remains active. It'll also trigger if you add a symbol after creating the alert. If you remove a symbol from the watchlist, the alert will no longer apply to it. This ensures that your alerts always reflect the most current set of symbols in your watchlist. **! Important:** Watchlist alerts are calculated only for the chart with the Regular Trading Session display selected, and trigger correspondingly. You can switch to RTH session as follows: To create one, click on the "Add alert on the list…" button in the watchlists dropdown. You can also create it directly from the "Symbols" dropdown in the "Create alert" dialog if you access it through any other method when configuring alerts.
