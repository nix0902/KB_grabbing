- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Alerts - / [ Alerts settings ](/support/folders/43000547663-alerts-settings/) - / [ I cannot set alerts with Log Scale enabled ](/support/solutions/43000690942-i-cannot-set-alerts-with-log-scale-enabled/) # I cannot set alerts with Log Scale enabled Due to the current implementation of the logarithmic scale and certain technical factors, alerts based on this scale may not accurately reflect the exact position of a drawing, leading to unexpected trigger behaviors. To avoid this issue, we recommend switching from the logarithmic scale to the linear scale. This way, alerts will be based on more precise calculations, ensuring more accurate trigger conditions.
