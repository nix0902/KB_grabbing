A [real-time data](https://www.tradingview.com/chart/?solution=43000471705) symbol looks as follows (no red icon is displayed). The prefix of the exchange is only displayed in the alerts manager section. Kindly note that the prefix is not displayed in the alert dialog. Only the symbol name is indicated.
