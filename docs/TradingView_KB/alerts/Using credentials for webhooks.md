- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Alerts - / [ Webhooks usage ](/support/folders/43000560150-webhooks-usage/) - / [ Using credentials for webhooks ](/support/solutions/43000722015-using-credentials-for-webhooks/) # Using credentials for webhooks Our alerts are **not designed for automated trading**. Sending personal information, including passwords and logins, via webhooks poses significant security risks. When configuring webhooks, ensure that you **do not include** sensitive information such as login credentials or passwords in the webhook URL and webhook message. Transmitting sensitive data through webhooks can expose it to unauthorized parties and lead to security vulnerabilities. Always use secure, authenticated endpoints and encrypt sensitive data to protect your information. Previous Previous Webhook authentication Next Next Webhook resubmission Launch Supercharts
