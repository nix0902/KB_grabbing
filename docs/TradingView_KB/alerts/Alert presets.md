- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Alerts - / [ Alerts settings ](/support/folders/43000547663-alerts-settings/) - / [ Alert presets ](/support/solutions/43000774194-alert-presets/) # Alert presets What are alert presets Alert presets allow you to save your alert configurations as templates and quickly apply them to different symbols or conditions. Instead of manually setting up the same alert parameters every time, you can create a preset once and reuse it with a single click. The feature is useful for traders who: - Want to maintain templates in one place - Need a quick setup with specific notification settings - Use similar conditions across multiple symbols **Note: You can save up to 5 alert presets.** What Gets Saved in a Preset When you save an alert preset, the following settings are stored: Condition Settings For each condition in your alert: - **Series** (if using indicators or strategies) - **Series plot **(if applicable) - **Operator** (crossing, crossing up, crossing down, greater than, etc.) - Bands including : Moving averages with their plots - Indicators with their plots - **Note:** **Value fields and drawing tools are NOT saved in presets** - **Resolution/Timeframe** (if specified) Alert Behavior - **Trigger frequency** - **Expiration** - saved as the time difference between the minimum possible date and your selected date, or marked as open-ended Customization - **Alert Name** (only if different from the default) - **Message template** (only if different from the default) Notifications - All notification settings from the **Notifications tab**, including: Notify in app - Show toast notification - Send email - Play sound - Send plain text - **Note:** **Webhook URLs are NOT saved in presets** How to create a preset Configure your alert with the desired settings: - Apply the condition (price, indicator, strategy) - Configure your message - Configure notification settings (mobile push, email, sound) - Click on the preset button → Save alert preset... Enter a descriptive name for your preset - Select which settings to save using the checkboxes (if none is chosen, all checkboxes are checked by default) - Click "Save" How to use alert presets - Open the create alert dialog - Click on the preset dropdown menu - Select your saved preset from the list - The applied preset will be highlighted in black - Create the alert Reset Changes To disable the applied preset and return to the default settings (as they were when you opened the alert dialog), click **"Reset changes"**. Delete a Preset To remove a preset from your library: Click the **bin icon** next to the preset in the dropdown menu - The preset will be permanently deleted **Important:** You cannot edit an already created preset. To modify settings, you'll need to create a new preset with the updated parameters. **Описание:** This TradingView interface displays an **alert configuration menu** with the following elements:  

### Top Section (Action Buttons)  
- **“Save alert preset…”**: A button with a cloud icon, used to save the current alert configuration as a reusable preset.  
- **“Reset changes”**: A button with a circular arrow icon, which reverts all unsaved modifications to the alert settings.  

### Middle Section (Preset List)  
- **“ALERT PRESETS”**: A header labeling the list of saved alert configurations.  

### Bottom Section (Preset Item)  
- A dark gray rectangular card titled **“Just an alert”**, containing the text **“Condition, Trigger, Expiration, Notifications”**—this is a saved preset that bundles alert criteria (e.g., price conditions, trigger logic, expiration dates, and notification preferences) for quick reuse.  


The interface is part of TradingView’s alert management system, enabling users to save, reset, or apply predefined alert settings for technical analysis or trading notifications.
