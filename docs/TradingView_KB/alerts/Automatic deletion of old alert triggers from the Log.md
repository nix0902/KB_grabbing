- [ Help Center ](/) - / - / Alerts - / [ Alerts settings ](/support/folders/43000547663-alerts-settings/) - / [ Automatic deletion of old alert triggers from the Log ](/support/solutions/43000766116-automatic-deletion-of-old-alert-triggers-from-the-log/) # Automatic deletion of old alert triggers from the Log The server automatically deletes old entries to keep things running smoothly. Cleanup runs every day at 21:00 UTC. Alert triggers are deleted if they meet at least one of the following conditions: - The trigger is older than 30 days - There are more than 100 trigger events for the same alert (only the most recent 100 are kept) If you need to keep old alert triggers, you can download log as CSV file in advance.
