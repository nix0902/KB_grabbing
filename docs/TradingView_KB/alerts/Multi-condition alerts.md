# Multi-condition alerts

- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Alerts - / [ Alerts settings ](/support/folders/43000547663-alerts-settings/) - / [ Multi-condition alerts ](/support/solutions/43000761492-multi-condition-alerts/) # Multi-condition alerts Create alerts that check multiple conditions at once — smarter, more precise signals without writing a single line of code. With this feature, you can combine up to five different conditions — including price, drawings, and indicators — into a single powerful alert. The alert will only trigger when all conditions are met simultaneously. Available with the Plus plan and above, multi-condition alerts count as just one technical alert toward your limit. To create one: - Open the [alert creation](https://www.tradingview.com/support/solutions/43000595315-how-to-set-up-alerts/) dialog - Click "Add condition" - Configure it, then click "Add" - All your conditions now appear as buttons with previews that you can click to edit You can mix conditions with different timeframes. In this case, the alert will fire once all conditions across the chosen intervals are true, with the frequency based on the smallest timeframe used. Things to keep in mind: - All conditions must use the same symbol. - Multi-condition alerts don't support strategies and scripts with alert() function. - Multi-condition alerts don’t support scripts written in Pine version earlier than 4 that include alertcondition() function.
