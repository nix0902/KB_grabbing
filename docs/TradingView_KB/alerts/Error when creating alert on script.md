# Error when creating alert on script

---

- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Alerts - / [ Alerts settings ](/support/folders/43000547663-alerts-settings/) - / [ Error when creating alert on script ](/support/solutions/43000754635-error-when-creating-alert-on-script/) # Error when creating alert on script If an error occurs when creating a script alert, there are two possible reasons: 1. You’re trying to create an alert on another user’s script, which has not been published by the author. In this case, you are also unable to see the source code for this script. 2. If you know that that script belongs to you and not to another user, it was likely deleted and is no longer available. In this case, you won’t be able to see the source code for this script either. Please, check if the script still appears in your personal indicators list. If not, try to reapply it to add an alert once again. If the above method did not work and the error is still being reproduced, please contact our support team.
