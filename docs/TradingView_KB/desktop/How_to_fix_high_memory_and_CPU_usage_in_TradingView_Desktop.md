# How to fix high memory and CPU usage in TradingView Desktop 
 This guide provides common solutions to enhance the performance of the TradingView Desktop app. Follow the suggested solutions below if you experience slow response, low productivity, high system load, freezes, lags, or crashes. By following these steps, you can optimize the performance of the TradingView Desktop app and ensure a smoother, more efficient experience.

#### Ensure your operating system and drivers are up to date 

Keeping your operating system and drivers up to date is crucial for compatibility and overall performance, directly affecting the TradingView Desktop app.

#### Keep TradingView Desktop updated 

Updating TradingView Desktop to the latest version ensures you have the most recent performance improvements, bug fixes, and updates. Here’s how to check for updates:

- Open the app menu by clicking the button in the top right corner (three horizontal dots) > “Settings” > “About” tab to check for updates.
- If an update is available, the app will install it quickly. You should then relaunch the app to switch to the new version.

#### Close unnecessary tabs 

TradingView processes vast amounts of data and renders real-time charts and visual elements, which can be resource-intensive. To reduce the load and improve performance, close tabs that are not essential to your current work.

#### Clear app cache 

Clearing cached files can enhance performance and resolve issues caused by outdated or corrupted files. Follow these steps to clear the cache:

- Open the app menu by clicking the button in the top right corner (three horizontal dots) > “Settings” > “Service” > “Clear cache”.
- Relaunch the app.

#### Toggle off hardware acceleration 

Hardware acceleration allows the app to offload tasks to your GPU for better performance. However, it can sometimes cause visual issues. To toggle off hardware acceleration:

- Open the app menu by clicking the button in the top right corner (three horizontal dots) > “Settings” > “Service”.
- Check the "Disable hardware acceleration" checkbox.
- Relaunch the app.
- If the issue persists, uncheck the "Disable hardware acceleration" checkbox to enable hardware acceleration again, and then relaunch the app.

#### Reset the app 

Resetting TradingView Desktop to its default settings can resolve performance issues by restoring the original configuration and eliminating problematic files. Note that the app state will be lost, but your saved layouts, changes, and data are stored remotely. To reset the app:

- Open the app menu by clicking the button in the top right corner (three horizontal dots) > “Settings” > “Service” > “Back to defaults” button.
- Relaunch the app.

#### Adjust resolution settings on macOS 

On macOS, using scaled resolutions can impact performance because the system renders everything at a higher resolution before scaling it down. To improve performance:

- Open the app menu by clicking the button in the top right corner (three horizontal dots) > “System Preferences” > “Displays”.
- Change the resolution to any labeled as "default" or "low resolution".
