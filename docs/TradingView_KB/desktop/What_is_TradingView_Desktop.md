# What is TradingView Desktop 
 [TradingView Desktop](https://www.tradingview.com/desktop/) is a program you can download and install on your computer to access a dedicated space for financial analysis with all the capabilities of our platform. It is available on Windows, macOS, and Linux.

Additionally, the desktop app provides extra features that are not available via the web version, so go ahead and check what it has to offer to make your analysis more competitive and advanced.

#### TradingView Desktop features 

The desktop version provides you with the same functionality as the TradingView web platform. But its app-based nature comes with unique advantages:

- Stay focused without distractions — your trading tabs won’t get lost among news sites or social media. Focus on what matters most to achieve better results
- Automatically restores the ticker, interval, and active watchlist in every tab upon app launch
- Link tickers across chart tabs — assign the same color tag to different tabs, change the ticker in one, and all others will display the same symbol

- Connect screeners to your chart
- Synchronize crosshair across tabs and windows

- Customize tab titles for better organization

- Supports system-native notifications
- Automatically follows the system theme and dynamically switches between light and dark modes based on the OS preferences with the Auto appearance mode in macOS and Windows

#### TradingView Desktop in a nutshell 

Concentration is key in trading and investing. Our desktop app delivers the same in-depth analysis as the web version — plus exclusive features.

Use it to maximize your performance with powerful tools: multi-monitor support, synchronized tabs, customizable layouts, and other.

To stay informed about updates, check the [Desktop release notes](https://www.tradingview.com/support/solutions/43000673888-tradingview-desktop-releases-and-release-notes/) and stay tuned to what's happening in trading and investment.
