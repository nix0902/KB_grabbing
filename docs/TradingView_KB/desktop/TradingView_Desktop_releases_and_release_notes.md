# TradingView Desktop releases and release notes 

#### January 20, 2026. Meet the customizable New Tab in version 3.0.0
 This update is available for Windows and Linux only.
 Features and improvements 
- New Tab — now customizable. Choose which products appear on your New Tab
- Creating a new layout from the Layouts section on the New Tab now starts with a naming dialog
- Updated the starting splash screen
 Bug fixes 
- Prevented multiple tabs from closing when repeatedly clicking the close button
- Fixed an issue where tabs were missing from the app window on launch
- Resolved an Ubuntu-specific issue where downloads behaved incorrectly when selecting a save location

#### 

#### October 20, 2025. Fix for performance issue on macOS Tahoe in version 2.14.0

With this release, we stopped supporting macOS Big Sur (version 11). On this operating system, the last available app version — 2.13.0 — will continue to work but will no longer receive updates. The minimum supported macOS version is now Monterey (version 12).

This update is available for Mac and Linux only.
 Features and improvements 
- Applied minor refinements to the main app menu
- Updated Electron to version 38
 Bug fixes 
- Addressed a performance issue on macOS Tahoe

#### June 6th, 2024. Integrated link handling in version 2.7.8

#### 
 Features and improvements 
- Enabled integrated link handling for all supported locales on Windows (previously available for English locale only). Links to TradingView charts and ideas now bypass the default browser, opening directly within the desktop app for Windows. This change ensures faster access and a more seamless workflow.  Bug fixes 
- Fixed an issue where “out of memory” errors led to app crashes. Now, such errors will only crash the affected tab, leaving the rest of the application unaffected.
