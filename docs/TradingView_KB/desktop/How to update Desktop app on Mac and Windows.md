If you’ve faced that and your app is not up to date, please take the steps described below. For Mac - Go to [tradingview.com/desktop](//tradingview.com/desktop/) and download the app installer; - Run the installer, move the app icon to the **Applications **folder, and choose to replace the existing version; - You’re all set - enjoy! For Windows - Open [tradingview.com/desktop](https://www.tradingview.com/desktop/) in your browser and download the app for Windows (a file named TradingView.msix); - Run the app installer file and click **Update**; ** ** - You’re all set.
