# System requirements for TradingView Desktop app 
 [TradingView Desktop](https://www.tradingview.com/desktop/) supports all major desktop operating systems. Below are the system requirements for each operating system.

#### Windows system requirements 

- **OS versions****: **Windows 10 (version 22H2) or Windows 11 (version 22H2 or newer).
- **App Installer****:** The App Installer app must be installed on your system to install TradingView Desktop. This app ships as part of Windows, so you probably already have it. [Learn more.](https://learn.microsoft.com/en-us/windows/msix/app-installer/install-update-app-installer)
- **Compatible processo****r: **64-bit Intel or AMD processor. ARM is not supported.

#### macOS system requirements 

- **OS versions****:** macOS Monterey (version 12) or newer, including the latest macOS Tahoe (version 26). 
- **Compatible processor****: **Intel processor or Apple Silicon (M1, M2, etc.).

#### Linux system requirements 

- **Package manager: **TradingView Desktop on Linux is distributed as snap and .deb packages. You can choose the option that suits you best: You can use the .deb package on any Debian-based Linux with no additional requirements.
- The snap package requires the Snap package manager to be installed.
**Note:** The Snap build is currently unstable on native Wayland. Use an X11 session or force XWayland** **for TradingView.
 - **OS versions: **While the following are the officially tested distributions, the app is known to work well on other Linux distributions that support .deb or snap packages.
 Ubuntu 20.04 LTS (Focal Fossa)
- Ubuntu 22.04 LTS (Jammy Jellyfish)
- Ubuntu 24.04 LTS (Noble Numbat)
 - **Compatible processor:** 64-bit Intel or AMD processor. ARM is not supported.

#### Network requirements 

- **IPv6 is not currently supported****.** Users on networks that prioritize IPv6 may experience connectivity issues.
