- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Desktop - / [ Features and configuration ](/support/folders/43000574972-features-and-configuration/) - / [ How to edit tab title ](/support/solutions/43000689717-how-to-edit-tab-title/) # How to edit tab title We found out that some of our users would’ve preferred to see a layout name on the tab title instead of a ticker symbol, because the former can be more informative. It makes perfect sense for a setup where you have a different layout for each asset type you trade. In this case, you’d like to see your tabs with titles like Crypto, US Stocks, FX, etc. Well, it’s up to you what your tabs look like — you can adjust it in the Settings. And once you’d like to see the default tab title, you can use the Reset to default option. Take a look at the video below to learn more about customizing the tab title.
