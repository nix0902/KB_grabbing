- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Portfolio - / [ Overview ](/support/folders/43000598042-overview/) - / [ Overview: Portfolio distribution ](/support/solutions/43000756102-overview-portfolio-distribution/) # Overview: Portfolio distribution This section is intended for visual analysis of the asset distribution within the portfolio by the selected category. It consists of a pie chart and a table with detailed information. Functionality: At the top of the block, there is a category selector that determines how assets are grouped: - Assets — each asset is displayed as a separate category. - Asset types — assets are grouped by type (for example: Stock, Crypto, Forex, etc.). - Sectors — assets are grouped by economic sectors. - Currency — assets are grouped by the currency in which they are traded. Table: This section also includes a table containing detailed information for each item in the current distribution. The table consists of the following columns: - Asset type — the name of the category according to the selected distribution type (for example, ticker, asset type, sector, or currency). - Portfolio Value — the current market value of the asset in the portfolio, expressed in the portfolio currency. - Allocation — the asset’s share in the portfolio, expressed as a percentage. - Unrealized Gain — unrealized profit or loss on the position. - Unrealized Gain % — unrealized profit or loss as a percentage of the invested amount.
