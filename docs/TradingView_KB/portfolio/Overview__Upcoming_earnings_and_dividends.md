# Overview: Upcoming earnings and dividends 

#### 

#### Upcoming earnings 
 Displays companies that are expected to report earnings in the near future. The list is divided into two blocks:

- Today & Yesterday — companies that have already reported today or yesterday.
- Next 14 Days — companies scheduled to report within the next two weeks.

The table includes the following columns:

- Symbol — ticker and company name.
- Date — earnings report date.
- Time — estimated or actual report time.
- Estimate EPS — estimated earnings per share.
- Actual EPS — actual earnings per share (if already published).
- Surprise — difference between actual and estimated earnings, in %.
- Market cap — company’s market capitalization.

#### Upcoming dividends 

Displays a list of stocks with upcoming dividend events and allows the user to filter them in two ways: by ex-date or pay date. When selecting one of these filters, a table is formed that includes the following columns:

- Symbol — ticker and company name.
- Amount — dividend amount per share.
- Ex-date — the cutoff date by which the stock must be held to receive the dividend.
- Pay date — the actual dividend payment date.
- Dividend yield — current annualized dividend yield, expressed as a percentage.

By default, filtering is set to ex-date with a 14-day horizon. This allows the investor to quickly identify which stocks need to be purchased before a specific date to receive dividends.
