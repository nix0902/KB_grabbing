# Why is Portfolio value less than the total Value of all my assets? 
 This is due to the nature of the Portfolio value calculation, which includes:

- Free cash
- the current market value of open positions.

More on this in: [Portfolio Summary: Portfolio Value and Cash](https://www.tradingview.com/support/solutions/43000756016/)

At the same time, Value for each asset consists of the sum of Invested and Unrealized gain. More on this in: Value – [Holdings page](https://www.tradingview.com/support/solutions/43000756132/)

Therefore, Portfolio value will differ from the sum of Value across all positions by the amount of Free cash, which already includes realized profit. See also: [Portfolio Summary: Total and Realized gain](https://www.tradingview.com/support/solutions/43000756023/)
