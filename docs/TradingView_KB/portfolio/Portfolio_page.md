# Portfolio page 

 A general page that displays cards of all created portfolios, as well as cards for creating new ones. The page includes two types of cards:

- cards of existing portfolios

- cards for creating new portfolios

#### Portfolio card 

 Each card displays the main current information about the portfolio:

- portfolio name

- date of the last structure update

- [Portfolio value

](https://www.tradingview.com/support/solutions/43000756016/)
- [Total gain %

](https://www.tradingview.com/support/solutions/43000756023/)
- [Annualized yield

](https://www.tradingview.com/support/solutions/43000756024/)
- number of unique instruments with an active position

Portfolio management is done via the additional menu (three-dot icon):

- delete portfolio
- edit portfolio settings

Clicking on a card opens the internal page of the selected portfolio.

#### Cards for creating a new portfolio 

- Create new by hand — [How to create a portfolio manually](https://www.tradingview.com/support/solutions/43000756008/)
- Add from Watchlist — [How to create a portfolio from a watchlist](https://www.tradingview.com/support/solutions/43000756009/)
- Import CSV file — [How to create a portfolio via transaction import](https://www.tradingview.com/support/solutions/43000756010/)

When selecting one of the cards, the corresponding creation dialog will open. Details are provided in the articles for each method.
