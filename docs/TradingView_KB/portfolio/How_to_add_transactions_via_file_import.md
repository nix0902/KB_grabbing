# How to add transactions via file import 
 To add transactions to a portfolio using a CSV file:

- Go to the Transactions tab.
- Click the Import button in the upper right corner of the table. A pop-up window with import settings will open.
- Or click on the dropdown menu next to the Add transaction button in the upper right corner and select the Import transactions option.

#### Import settings 

- File upload: Drag the CSV file into the window or click the Browse button to select it from your device. Maximum file size — 25 MB.
- Replace full transaction history: If this option is enabled, the entire current transaction history will be replaced with the ones specified in the uploaded file. By default, import works in merge mode, where: Only unique transactions are added.
- If all trade parameters (Symbol, Side, Qty, Fill Price, Commission, Closing Time) fully match an existing one, it will not be added again.
 - Example file: For convenience, an example CSV file with transactions in the required format is available. The download link is directly in the import window — Download example.

#### After upload 

- Make sure the correct import option is selected.
- Click the Import button — transactions will be added to the portfolio for supported instruments.

Note: If there are format errors, a message will appear indicating the line where the error occurred. In this case, transactions will not be added.

#### File format 

The CSV file must contain the following fields: Symbol, Side, Qty, Fill Price, Commission, Closing Time. More on the format: [How to create a portfolio via transaction import](https://www.tradingview.com/support/solutions/43000756010/).
