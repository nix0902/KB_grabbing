**Описание:** This TradingView portfolio performance chart displays a **blue line graph** with a light blue shaded area (representing portfolio value over time) against a white background.  

### Key UI Elements:  
- **X-axis**: Dates (spanning from late June to late September, with labeled months: *Jul, Aug, Sep* and numerical dates).  
- **Y-axis**: Portfolio value (USD) ranging from -4,000 to 12,000.  
- **Legend**: A blue box labeled *“Portfolio”* with the current value *“2,883.83 USD”* (blue text).  
- **Timeframe Buttons**: Horizontal row of buttons for different periods:  
  - *1 month*: 4,935.16 USD (green text)  
  - *3 months*: 6,355.97 USD (green text, highlighted/selected)  
  - *6 months*: 31,707.18 USD (green text)  
  - *Year to date*: 20,132.40 USD (green text)  
  - *1 year*: 50,842.23 USD (green text)  
  - *All time*: 3,098.40 USD (green text)  

### Chart Purpose:  
The line graph visualizes portfolio performance over the selected timeframe (3 months, in this case), showing fluctuations in value. The shaded area emphasizes the trend, while the buttons allow users to compare performance across different time horizons. The legend and current value provide real-time portfolio context.
