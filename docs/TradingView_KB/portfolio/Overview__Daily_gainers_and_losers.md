# Overview: Daily gainers and losers 
 Displays the assets with the highest increase and the highest decrease in value over the last trading day.

#### Interface structure: 

- The Daily Gainers block displays assets with positive dynamics (growth).
- The Daily Losers block displays assets with negative dynamics (decline).

Each row represents a separate asset, sorted by the amount of change in descending order (the largest gain or loss shown first).

Each block displays no more than 5 assets with the highest values of growth or decline, respectively, regardless of the total number of assets in the portfolio.

#### The data is presented as follows: 

- Asset icon
- Ticker
- Proportional change bar
- Change value in percentage (on the right)

On hover, key metrics for the asset are displayed:

- Change % — relative change in profit for the day from yesterday’s Value (Daily gain %)
- Change — change in position profit over the last trading day (Daily gain)
- Total value — current position value in the portfolio currency (Value)

It is important to understand that the data is shown for the last day with active trading — this is necessary for cases when trading for the instrument has not yet opened today, so you can still see the latest available changes.

Note: The data source and calculation method are identical to the [Holdings page](https://www.tradingview.com/support/solutions/43000756132/) section.
