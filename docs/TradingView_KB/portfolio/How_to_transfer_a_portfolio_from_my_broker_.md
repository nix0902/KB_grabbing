# How to transfer a portfolio from my broker? 
 To transfer a portfolio from your broker into the system, follow these steps:

- Export the full transaction history

Log in to your broker’s client area and download a file with the complete history of your operations. The file is usually available in CSV or Excel format. You will need a CSV file.

- Format the file for import

Edit the exported file to match the import format requirements. For more details on the file structure and requirements, see the article [How to create a portfolio via transaction import](https://www.tradingview.com/support/solutions/43000756010/).

- Import the data

You have two options for uploading the data:

- Create a new portfolio based on the prepared file — select the option to create via transaction import.
- Add data to an existing portfolio through the Transactions → Import.

For more details, see the guide [How to add transactions via file import](https://www.tradingview.com/support/solutions/43000756014/).
