At the moment, only a limited range of assets is available. For more details, see the section: [Why can’t all asset types be added?](https://www.tradingview.com/support/solutions/43000756140/) Because of this, there is currently no option to directly add unsupported instruments. If you have suggestions for adding specific asset types, please contact support — this will help us better account for your priorities when planning future updates.
