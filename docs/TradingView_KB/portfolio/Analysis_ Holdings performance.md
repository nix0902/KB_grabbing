- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Portfolio - / [ Analysis ](/support/folders/43000598045-analysis/) - / [ Analysis: Holdings performance ](/support/solutions/43000756111-analysis-holdings-performance/) # Analysis: Holdings performance Displays performance by portfolio assets in the form of an interactive bar chart. Allows for a quick assessment of which assets showed the best and worst dynamics. Display modes: - All – all portfolio assets are displayed. - Gainers – only assets with positive returns for the entire lifetime of the portfolio. - Losers – only assets with negative returns for the entire lifetime of the portfolio. - **Dividends – only assets that have generated dividend income over the entire lifetime of the portfolio.** The bar chart is sorted in descending order of return (based on the selected mode). Each row represents a single asset. Display controls: - Display sold – toggle that enables/disables the display of closed (sold) assets. Enabled by default. - Value – allows switching the display of values between: Absolute – absolute profit/loss values (in portfolio currency). - Percentage – relative values (as a percentage of the invested amount). On hover over an asset, key metrics are shown: - Total gain % – relative profit on the position compared to the invested amount. - Total gain – total profit on the position in the portfolio currency. - Total value – current value of the position in the portfolio currency (Value). Note: All calculations and data are synchronized with the [Holdings page](https://www.tradingview.com/support/solutions/43000756132/) table.
