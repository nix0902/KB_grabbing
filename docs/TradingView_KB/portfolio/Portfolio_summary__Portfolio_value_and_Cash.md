# Portfolio summary: Portfolio value and Cash 

#### 

#### Portfolio value 
 Reflects the current total value of the user's portfolio in the portfolio currency, taking into account free cash and the current value of open positions.Portfolio value represents the answer to the investor's key question: How much money will I get if I sell all assets right now?

#### Cash 

Reflects the amount of free cash available in the portfolio at the moment.

Calculation: Cash = Portfolio value − ∑current positions value Note: Cash includes Realized gain

#### Handling cash shortage: 

If a Buy transaction leads to a negative cash position, the system will automatically detect this situation and suggest compensating the negative balance via a deposit transaction.This is necessary to ensure calculation integrity and to reflect the correct portfolio balance.
