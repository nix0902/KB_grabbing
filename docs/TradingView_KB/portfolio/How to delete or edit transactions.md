- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Portfolio - / [ Transaction Management ](/support/folders/43000598046-transaction-management/) - / [ How to delete or edit transactions ](/support/solutions/43000756015-how-to-delete-or-edit-transactions/) # How to delete or edit transactions You can edit or delete both individual transactions and all transactions for a specific instrument. Editing and deleting individual transactions - Go to the Transactions tab. - Select the desired transaction type — Trades, **Cash,** or **Dividends**. - Find the desired row and click the context menu icon at the end of the row. - Select: Edit transaction — an editing dialog will open. After making changes, click Save. - Delete — a deletion confirmation window will open. Confirm by clicking Delete. Deleting all transactions for an instrument - Go to the Holdings tab. - Find the desired instrument and click the context menu (three dots at the end of the row). - Select Delete. - Confirm deletion of all transactions for the instrument in the pop-up window by clicking Delete. Note: To delete transactions for already closed instruments, enable their display by activating the Display sold option.
