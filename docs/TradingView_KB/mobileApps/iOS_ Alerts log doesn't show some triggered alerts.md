- [ Help Center ](/) - / - / Mobile apps - / [ Push notifications ](/support/folders/43000547714-push-notifications/) - / [ iOS: Alerts log doesn't show some triggered alerts ](/support/solutions/43000506690-ios-alerts-log-doesn-t-show-some-triggered-alerts/) # iOS: Alerts log doesn't show some triggered alerts Alerts log in your iOS app contains alerts that were received as push notifications only. Note that in order to see alerts in the log, the following criteria have to be met: Notify on App option has to be enabled and a Push notifications need to be delivered. Please be advised that alerts are not going to be shown in the log otherwise.
