# iOS: I'm not receiving notifications about triggered alerts 
 Please complete the following steps if you are unable to receive notifications on your iOS app. 

- Make sure that you created an alert in the correct way (details can be found [here](https://www.tradingview.com/?solution=43000474590)).
- Make sure that you are logged in using your correct credentials.
- Make sure that push notifications are enabled for the TradingView app.
- Try to re-login into the app
