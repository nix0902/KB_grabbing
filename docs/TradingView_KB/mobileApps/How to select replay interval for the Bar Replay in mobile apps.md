- [ Help Center ](/) - / - / [ Other questions ](/support/folders/43000558391-other-questions/) - / [ How to select replay interval for the Bar Replay in mobile apps ](/support/solutions/43000747376-how-to-select-replay-interval-for-the-bar-replay-in-mobile-apps/) # How to select replay interval for the Bar Replay in mobile apps In the Bar Replay mode, you can choose different intervals for replay. The replay interval is the amount of time that corresponds to one data update in the Bar Replay mode. If it is set to 5 minutes, then for each tick in the replay mode, data will be added that corresponds to 5 minutes in real-time. The current interval is displayed on the replay panel to the right of the replay speed. Tapping on it will open a menu for selecting available intervals for this chart. The minimum replay interval for second charts is 1 second, and for intraday and daily intervals, it is 1 minute. Weekly and monthly charts can only be played in one-day intervals.
