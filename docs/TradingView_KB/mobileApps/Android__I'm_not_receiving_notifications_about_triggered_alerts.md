# Android: I'm not receiving notifications about triggered alerts 
 Please complete the following steps if you are unable to receive notifications on your Android app. 

- Make sure that you created an alert in the correct way (details can be found [here](https://www.tradingview.com/?solution=43000474595)).
- Make sure that you are logged in using your correct credentials.
- Make sure that push notifications are enabled for the TradingView app.
- Make sure that the TradingView app is whitelisted in the Autorun manager (it must be installed on your device).
- Try to re-login into the app.

If the issue still remains, please try to solve the issue for your particular device. You can find the instructions [](https://tradingview.freshdesk.com/a/solutions/articles/43000512097?lang=ru-RU)[here](https://www.tradingview.com/support/solutions/43000512097-troubleshooting-push-notifications-on-android-devices/).
