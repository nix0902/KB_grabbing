# How to enable/disable Buy/Sell Buttons in the app? 
 The display of the Buy/Sell buttons depends on the *one-tap trading* setting. Trading settings can be accessed via the three-dot menu in the Order Panel or on the floating Buy/Sell panel. *If One-tap trading is disabled:*
- Buy/Sell buttons are hidden to keep the chart uncluttered.
- You can place trades using the Trade button (up/down arrow icon), which opens the Order Ticket.
*If One-tap trading is enabled:*
- The Trade button (up/down arrow icon) shows or hides the floating Buy/Sell panel.
- Tapping Buy or Sell places an order immediately, without opening the Order Ticket.
- Order presets created in the Order Ticket are available via the three-dot menu in the panel.
