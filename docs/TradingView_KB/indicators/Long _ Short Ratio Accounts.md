- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Indicators - / [ Built-in Indicators ](/support/folders/43000587405-built-in-indicators/) - / [ Long / Short Ratio Accounts ](/support/solutions/43000762399-long-short-ratio-accounts/) # Long / Short Ratio Accounts A trader sentiment metric that shows the proportion of accounts holding long versus short positions for a specific trading pair or symbol. This metric reflects the number of accounts with a net position — not the trade volume or position size. If an account holds a net long position, it is counted as one long account, regardless of how large the position is; the same applies to accounts with a net short position. The ratio is calculated as Long/Short Ratio = Number of Long Accounts / Number of Short Accounts. On the chart, the data is available as the "Long / Short Ratio Accounts" indicator in the "Financials" tab. "Long / Short Ratio Accounts" is available for crypto derivatives of the following exchanges: BYBIT, BINANCE. Previous Previous Liquidations Next Next Long Short Accounts % Launch Supercharts
