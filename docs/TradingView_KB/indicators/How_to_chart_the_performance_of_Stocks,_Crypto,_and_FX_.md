# How to chart the performance of Stocks, Crypto, and FX? 
 All users, regardless of subscription, have the option to compare multiple symbols on one screen.

To activate this feature, click on the Compare button located on the top control panel. Then, in the window that appears, select the symbol you need.

Now several symbols are available for analysis at the same time.

This image shows a **TradingView chart interface** displaying the **Bitcoin (BTC) / U.S. Dollar (USD)** trading pair on the Bitstamp exchange, with a 30-minute (30m) candlestick timeframe. Here’s a detailed breakdown of all elements:


### **Top Navigation Bar**
- **Left Side:**
  - **BTCUSD**: The trading pair (Bitcoin vs. US Dollar).
  - **+**: Button to add a new symbol/asset.
  - **30m**: Timeframe selector (30-minute candles).
  - **Chart Type Icons**: Line, candlestick, bar, etc. (currently candlestick).
  - **Indicators**: Dropdown to add technical indicators (e.g., RSI, MACD).
  - **Grid/Chart Layout**: Icon to toggle chart grid or layout.
  - **Alert**: Set price alerts.
  - **Replay**: Replay historical price action.
  - **Undo/Redo**: Navigation arrows for chart actions.

- **Right Side:**
  - **TradingView**: Brand logo/dropdown.
  - **M/M/S**: Market data (M) / My Charts (M) / Saved (S) toggles.
  - **Refresh/Settings/Fullscreen/Camera/Publish**: Refresh data, open settings, enter fullscreen, take a screenshot, or publish the chart.


### **Symbol & Market Info (Below Navigation)**
- **BTC/USD · 30 · Bitstamp**: Confirms the symbol (BTC/USD), timeframe (30m), and exchange (Bitstamp).
- **Price Data**:  
  `O 113,084 H 113,123 L 112,906 C 113,066 -17 (-0.02%) +1,041 (+0.93%)`  
  - `O`: Open price (113,084)  
  - `H`: High price (113,123)  
  - `L`: Low price (112,906)  
  - `C`: Close price (113,066)  
  - `-17 (-0.02%)`: Change from previous close (red = down).  
  - `+1,041 (+0.93%)`: Change from a prior reference (e.g., daily open, green = up).  
- **ETHUSD (Secondary Symbol)**:  
  `ETHUSD · Bitstamp -6.56%`: Ethereum (ETH) price, showing a 6.56% drop (red = down).


### **Main Chart Area**
- **Candlestick Chart**: Red/green candles represent price action (red = close < open; green = close > open). The chart shows Bitcoin’s price movement over ~2 days (x-axis: time, y-axis: price).
- **Percentage Scale (Right Y-Axis)**: Tracks price changes as a percentage (e.g., -8.50% to +0.50%), useful for comparing volatility.
- **Trend Line (Purple)**: A moving average or custom trend line overlaid on the chart, showing the overall price trend.


### **Left Sidebar (Tools)**
Icons for charting tools:  
- **Trend Line**: Draw trend lines.  
- **Fibonacci**: Add Fibonacci retracement levels.  
- **Patterns**: Recognize chart patterns.  
- **Text/Label**: Add annotations.  
- **Zoom**: Magnify/zoom out.  
- **Magnet**: Snap to grid.  
- **Lock**: Lock chart elements.  
- **Visibility**: Toggle element visibility.  
- **Globe**: Access global market data.  
- **Trash**: Delete elements.  
- **Star**: Save chart.


### **Bottom Bar**
- **Timeframe Selector**: `1D 5D 1M 3M 6M YTD 1Y 5Y All` (switch between daily, weekly, monthly, etc., timeframes).  
- **Calendar Icon**: Open date/time selector.  
- **Pine Editor/Trading Panel**: Tabs to access Pine Script (TradingView’s programming language) or trading order panel.  
- **UTC Time**: `11:46:46 UTC` (current time).


### **Right Sidebar (Additional Tools)**
Icons for:  
- **Notes**: Add notes to the chart.  
- **History**: View chart history.  
- **Alerts**: Manage price alerts.  
- **Chat**: Community chat.  
- **Indicators**: Add indicators.  
- **Drawing Tools**: More drawing options.  
- **Calendar**: Event calendar.

- **Help**: Access support.


### **Key Takeaways**
- The chart focuses on **Bitcoin’s 30-minute price action** on Bitstamp, with a secondary view of Ethereum’s performance.  
- UI elements enable **technical analysis** (indicators, drawing tools), **timeframe adjustment**, and **trading actions** (alerts, publishing).
