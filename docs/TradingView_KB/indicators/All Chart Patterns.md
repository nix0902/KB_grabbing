Possible values: All, Awaiting and Reached, Only Awaiting, Last Awaiting. If Last Awaiting is selected, only one pattern with the Awaiting status will be displayed, the first point of which is located to the right of all the others. - Price Targets - Defines which price targets will be drawn depending status of the pattern. Possible values: All, Only Awaiting, Awaiting and Reached, None. - In Progress – Search for emerging patterns. All other parameters of the patterns are taken by default from the corresponding individual indicators.
