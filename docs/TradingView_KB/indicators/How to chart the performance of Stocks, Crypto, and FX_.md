All users, regardless of subscription, have the option to compare multiple symbols on one screen. To activate this feature, click on the Compare button located on the top control panel. Then, in the window that appears, select the symbol you need. Now several symbols are available for analysis at the same time. ### **Top Toolbar**  
- **Symbol & Timeframe**: “BTC/USD” (current instrument) with a “+” (add symbol), “30m” (timeframe, 30-minute candles).  
- **Tools**: Icons for chart type (candlestick), indicators (dropdown), layout (grid), time (UTC), alerts, replay, navigation (back/forward), and TradingView menu.  
- **Actions**: “Publish” (share chart), camera (screenshot), and other utility icons.  


### **Chart Header**  
- **Instrument Details**: “Bitcoin / U.S. Dollar · 30 · Bitstamp” (instrument, timeframe, exchange).  
- **Price Data**: “O 113,084 H 113,123 L 112,906 C 113,066” (open, high, low, close prices) + “-17 (-0.02%)” (daily change) + “+1,041 (+0.93%)” (additional metric, e.g., volume or another asset).  
- **ETHUSD Reference**: “ETHUSD · Bitstamp -6.56%” (Ethereum’s 24h change, for context).  


### **Main Chart Area**  
- **Candlestick Chart**: Red/green candles show price action (red = close < open; green = close > open) over time.  
- **Secondary Indicator (Bottom)**: A pink line (likely RSI, MACD, or another oscillator) tracks momentum/relative strength, with a percentage scale (right y-axis: -8.5% to 0.5%).  


### **Left Sidebar (Tools)**  
Icons for:  
- Drawing tools (trendlines, shapes),  
- Indicators,  
- Timeframes,  
- Alerts,  
- Chat,  
- Chart types,  
- Calendar,  
- Settings, etc.  


### **Right Sidebar (Metrics)**  
- **Percentage Scale**: Tracks the secondary indicator’s values (e.g., -8.5% to 0.5%).  
- **Icons**: For notes, history, alerts, chat, and other features.  


### **Bottom Toolbar**  
- **Time Axis**: Timestamps (e.g., “22”, “03:00”, “06:00”) show candle periods.  
- **Timeframe Selector**: “1D 5D 1M 3M 6M YTD 1Y 5Y All” (switch between timeframes).  
- **UTC Time**: “11:46:46 UTC” (current time).  
- **Tabs**: “Pine Editor” (code indicators) and “Trading Panel” (order execution).  


### **Purpose**  
This chart is used for **technical analysis** of Bitcoin’s price, combining candlestick patterns (price action) with a secondary indicator (momentum/oscillator) to identify trends, reversals, or overbought/oversold conditions. The ETHUSD reference provides context for altcoin performance.
