- [ Help Center ](/) - / - / [ My drawings do not get synchronized across charts or layouts ](/support/solutions/43000629998-my-drawings-do-not-get-synchronized-across-charts-or-layouts/) # My drawings do not get synchronized across charts or layouts The drawings can only be synced when you have the Drawings Synchronization enabled. To enable synchronization of drawing objects, open the multichart mode in the Select Layout menu. Now turn on the synchronization of drawing objects using the button on the left panel. Drawing tools can be synchronized between layout charts or globally, depending on the selected mode. If synchronization is disabled, drawing objects will be saved only on a specific chart of a particular layout. Synchronization by layout synchronizes the drawings on the current symbol across all charts in the current layout. Disabling synchronization on a drawing object removes it from all other charts. Global synchronization. Drawing objects will be synchronized across all charts and layouts, which means they will appear even in completely new layouts. Switching the global synchronization mode on a drawing object will remove it from all other layouts. You can also synchronize each specific drawing object by selecting the “Sync in layout” or “Sync globally” option in its context menu. Objects are globally synchronized between different windows/tabs/devices in real-time only when the layouts autosave is enabled. If autosave is disabled, objects are synchronized once only after manual saving.
