- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Chart - / [ I want to export/download chart data ](/support/folders/43000561757-i-want-to-export-download-chart-data/) - / [ How to export chart data ](/support/solutions/43000537255-how-to-export-chart-data/) # How to export chart data You can save chart data, including tickers and indicators, into a CSV file. Then you can import it into spreadsheet software for further analysis. To download your [Supercharts](https://www.tradingview.com/chart/) data, choose "Download chart data…" from the dropdown menu on the upper toolbar. Next, select the chart and click "Download". The data you download becomes available as soon as chart data is loaded. The data you export is what your chart shows you. If you need more data, scroll the chart left or drag the x-axis and then download a CSV.
