- [ Help Center ](/) - / - / [ My indicators are not synchronized to all charts of my layout ](/support/solutions/43000762823-my-indicators-are-not-synchronized-to-all-charts-of-my-layout/) # My indicators are not synchronized to all charts of my layout Indicators are not automatically synchronized between the charts of one layout. Still, you can apply a specific indicator to all charts in a layout by clicking on Add this indicator to entire layout from the indicator's context menu. To add several indicators from one chart in a multi-chart layout to all, you can also right-click on the chart and select Apply these indicators to entire layout .
