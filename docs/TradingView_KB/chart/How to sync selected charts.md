# How to sync selected charts

**URL:** https://www.tradingview.com/support/solutions/43000761094-how-to-sync-selected-charts/

---

- [ Help Center ](/) - / - / Chart - / [ How to work in the multi-chart mode ](/support/folders/43000578567-how-to-work-in-the-multi-chart-mode/) - / [ How to sync selected charts ](/support/solutions/43000761094-how-to-sync-selected-charts/) # How to sync selected charts You can synchronize symbol and interval between charts individually by adding them to different groups. To do this, please follow these steps: 1. Apply the multiple charts mode (Select layout button on the top bar). 2. Enable Symbol sync, Interval sync, or both. 3. Add the necessary charts to the group by marking them with an emoji in the Status line or Series context menu: click the Chart Syncing button and select an emoji from the list. Charts marked with the same emoji will be synchronized by the parameters selected in step 2. 4. To remove a chart from a group, simply uncheck the emoji.
