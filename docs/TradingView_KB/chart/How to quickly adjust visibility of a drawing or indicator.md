- [ Help Center ](/) - / - / [ How to quickly adjust visibility of a drawing or indicator ](/support/solutions/43000686263-how-to-quickly-adjust-visibility-of-a-drawing-or-indicator/) # How to quickly adjust visibility of a drawing or indicator To quickly adjust the visibility of a drawing or indicator at intervals: - Open the context menu of the drawing or indicator by right-clicking on the selected element. - Go to the menu item Intervals visibility. - Select one of the quick interval visibility options. The selected option will set the appropriate settings in the Visibility tab. If necessary, you can make additional adjustments through the object's settings dialog box. This setting is also available by clicking on the three dots icon in the floating drawing toolbar and indicator legend. Previous Previous How can I enable backadjustment for continuous futures?
