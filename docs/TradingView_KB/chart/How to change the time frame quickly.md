You can add an interval to your favorites by clicking on the star icon next to it and quickly change the time frame by selecting it from the top panel. Alternatively, it is possible to change the interval by using your keyboard: just start typing the number. For example, if you want to set a 125-minute interval, type "125" on your keyboard and press the Enter key.
