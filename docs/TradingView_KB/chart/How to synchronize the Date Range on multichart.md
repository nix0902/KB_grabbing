It’s a way more convenient and far easier way to analyze the price movements of an instrument (or several instruments) at different intervals. To use this feature, just do the following: - Open multiple charts in one window using the Select Layout menu - Enable the “Date range” option there in the “Sync in Layout” section Now, scrolling and changing the visible date range of one of the charts will be synchronized with all other charts in this window.
