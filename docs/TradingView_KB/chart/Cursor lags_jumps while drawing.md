# Cursor lags/jumps while drawing

**URL:** https://www.tradingview.com/support/solutions/43000472716-cursor-lags-jumps-while-drawing/

---

- [ Help Center ](/) - / - / [ Cursor is being moved or displayed incorrectly ](/support/folders/43000547271-cursor-is-being-moved-or-displayed-incorrectly/) - / [ Cursor lags/jumps while drawing ](/support/solutions/43000472716-cursor-lags-jumps-while-drawing/) # Cursor lags/jumps while drawing If you are noticing an unusual behavior while drawing (i.e. you can’t set an anchor point to a specific area), then you might be working with Magnet Mode enabled. Magnet Mode makes all the movable drawings attach to OHLC values, hence the difficulties with moving the cursor. To disable Magnet Mode, press the magnet icon on the left-hand side menu (the icon should turn white when disabled). Please make sure that Magnet Mode is disabled on your chart: Magnet Mode may also be enabled or disabled by holding Ctrl/Command key while drawing.
