- [ Help Center ](/) - / - / Chart - / [ Bar Replay ](/support/folders/43000547807-bar-replay/) - / [ Bar Replay doesn't work, there is no Replay toolbar ](/support/solutions/43000475470-bar-replay-doesn-t-work-there-is-no-replay-toolbar/) # Bar Replay doesn't work, there is no Replay toolbar We currently do not support the replay of: - spreads (formula of symbols), - non-traditional chart types ([Renko](https://www.tradingview.com/?solution=43000502284), [Kagi](https://www.tradingview.com/?solution=43000502272), [PnF](https://www.tradingview.com/?solution=43000502276), [Range](https://www.tradingview.com/?solution=43000474007), [Line Break](https://www.tradingview.com/?solution=43000502273), [Volume footprint](https://www.tradingview.com/support/solutions/43000726164-volume-footprint-chart/), [Time Price Opportunity](https://www.tradingview.com/?solution=43000725590)), - converted to non-default currency charts, - [tick-based charts](https://www.tradingview.com/?solution=43000709225). If you click on the Replay button and the replay panel did not appear, most likely, one of the above chart types or symbols were selected. Previous Previous How do I turn Bar Replay on?
