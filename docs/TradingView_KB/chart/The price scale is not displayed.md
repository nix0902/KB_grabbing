- [ Help Center ](/) - / - / Chart - / [ How to adjust the scales on a chart? ](/support/folders/43000549291-how-to-adjust-the-scales-on-a-chart/) - / [ The price scale is not displayed ](/support/solutions/43000480249-the-price-scale-is-not-displayed/) # The price scale is not displayed Visibility of the scales is automatically adjusted. The scale is always displayed if something is attached to it. The price scale is hidden if nothing is attached to it. Thus, if you want to turn on the scale, attach an indicator or a symbol to it.
