- [ Help Center ](/) - / - / [ The color of bars or candlesticks is incorrect ](/support/solutions/43000473886-the-color-of-bars-or-candlesticks-is-incorrect/) # The color of bars or candlesticks is incorrect You may have applied an indicator that paints the bars on the chart. Try hiding indicators on the chart to see if that's the case. If some bars are automatically painted using an opposite color, it might be possible that the Color Bars Based on Previous Close option is enabled on your chart. It can be turned off in the chart settings. Previous Previous Candles have disappeared from the chart Next Next The chart looks way too compressed Launch Supercharts
