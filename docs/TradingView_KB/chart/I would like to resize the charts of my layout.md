- [ Help Center ](/) - / - / [ How to work in the multi-chart mode ](/support/folders/43000578567-how-to-work-in-the-multi-chart-mode/) - / [ I would like to resize the charts of my layout ](/support/solutions/43000629993-i-would-like-to-resize-the-charts-of-my-layout/) # I would like to resize the charts of my layout To resize a chart, just drag one of its sides. Double-clicking on any of the chart borders returns it to its default state. You can also move several chart borders at the same time. To do this, hold down the Shift key while dragging. To change the size of a chart on a mobile device, select the desired border with a single tap and then move it. A double tap on any border returns the charts to their default position.
