# Learn to use column charts 
 Column charts are unique and differ from other chart types as these use a completely different approach to display values on your [Supercharts](https://www.tradingview.com/chart/). They can be used for both technical and fundamental analysis, and for analyzing various financial data.

#### What are column charts 

A quick look at these can help you better grasp their most common use cases. This chart type uses vertical columns stemming from the bottom scale.

At their core, columns are mostly used for financial data to see how the exact values have changed over a significant period of time.

As you can see, they are available for day-based time intervals only.

#### Column charts vs area charts 

The reason we compare these two is because both appear similar. But, by default, columns are colored depending on the previous column price, while area is a line-type chart, which doesn't color price values.

**Column charts**

- Provide clear picture of precise chart values
- Easy to see changes in relationship to previous values due to colored nature of columns
- Well-suited for economic data

**Area charts**

- Better for trend analysis
- More intuitive visual interpretation
- Better suited for technical data

#### Enabling the chart type 

Once you are on the Supercharts, open the chart type menu on the upper toolbar, and select "Columns."

#### Settings 

The only configuration available for this chart type is changing how columns are colored — depending on the current or previous bar. To note, we also call them "bars," as "bars" are a broad name for all the charting figures.

To change this chart type's color style, find the gear "Settings" button on the upper toolbar, and go to the "Symbol" tab.

#### Column charts in a nutshell 

You can use column charts to display economic indicators such as GDP, interest rates, or inflation rates. They will appear as a consecutive array of green and red columns, helping you easily spot macroeconomic trends and shifts.

Depending on your trading style, you can also apply this chart type to tradable assets and analyze them from a broad perspective.
