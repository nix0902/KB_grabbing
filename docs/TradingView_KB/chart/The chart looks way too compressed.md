- [ Help Center ](/) - / - / [ The chart looks way too compressed ](/support/solutions/43000473898-the-chart-looks-way-too-compressed/) # The chart looks way too compressed It's likely that you have used an indicator, which has levels that are located far from the symbol series. The chart becomes compressed since automatic chart scaling forces it to fit all indicator levels in the visible area. The issue can resolved by hiding the indicator or adjusting it so that levels are closer to the values of the symbol. Here is an example with the Pivot Points Standard indicator.
