On your attempt to remove a locked drawing, you’ll get a pop-up dialog with deletion options. If you check “Don’t show again”, the chosen option will be applied as default next time. To change the behavior, open Remove options drop-down menu on the left panel, and toggle Always remove locked drawings setting: Previous Previous How to enable/disable floating data tooltip?
