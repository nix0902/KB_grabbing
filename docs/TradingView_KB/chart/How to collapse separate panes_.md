- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Chart - / [ Hotkeys and tips ](/support/folders/43000561752-hotkeys-and-tips/) - / [ How to collapse separate panes? ](/support/solutions/43000665813-how-to-collapse-separate-panes/) # How to collapse separate panes? To minimize a separate pane with an indicator, an additional symbol, or a main series, simply click the “Collapse Pane” button in the right corner (make sure that the visibility of the pane buttons is enabled in the Chart Settings, on the Canvas tab). Restore it to its original size by pressing the button again. In addition, you can minimize/restore the panel by double-clicking anywhere in the pane with the Ctrl / ⌘ key pressed.
