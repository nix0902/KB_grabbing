- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Chart - / [ Hotkeys and tips ](/support/folders/43000561752-hotkeys-and-tips/) - / [ How to use the Compare tool ](/support/solutions/43000543053-how-to-use-the-compare-tool/) # How to use the Compare tool With the "Compare" tool you can examine two or more symbols simultaneously. This allows you to benchmark one asset against another, spot correlations, or assess the market's health. For example, you can compare the SPY, DIA, and QQQ. The result is one single chart containing easy-to-read, relevant information spanning the entire market. Another way to use this tool is to compare two companies within the same sector. For example, two automobile companies — General Motors and Ford. By doing so, you can narrow your research and gain a better understanding of the two major players within a single sector. **! Note:** When you add a symbol to a chart, the vertical axis will change from price to percentage. The percentage shown is the price movement for whatever period is currently visible on the chart. This is absolutely necessary. If you switch it back to price, the chart will collapse unless the price values of the symbols are very close to each other. Below is an example of a Google and Apple comparison chart when switched back to price. Their prices are about $500 apart. As a result, when you switch to price, both data series flatten out and the chart loses its effectiveness. To add a comparison to your current data series/chart, simply click on the "Compare or Add symbol button" on the toolbar along the top of the chart. Several popular symbols are already listed, which you can select to add to the chart. However, you can add any symbol for your analysis. **Описание:** This TradingView interface displays a **1-day (D) candlestick chart** for **Apple Inc. (AAPL)** on the NASDAQ exchange. Here’s a detailed breakdown of elements:  


### 1. **Top Navigation Bar**  
- **Symbol/Exchange**: “AAPL” (Apple Inc.) with a NASDAQ logo, confirming the stock and exchange.  
- **Timeframe Selector**: “D” (Day) indicates the chart’s time period (1-day candles).  
- **Buttons (Left to Right)**:  
  - **+ (Add Symbol)**: Blue-outlined button (highlighted by an arrow) to add new symbols to the chart.  
  - **Indicators**: Opens a menu to add technical indicators (e.g., moving averages, RSI).  
  - **Chart Type**: Toggles between chart styles (e.g., candlestick, line, Heikin-Ashi).  
  - **Alert**: Sets price/volume alerts for the symbol.  
  - **Replay**: Replays historical price action for backtesting.  


### 2. **Price & Volume Summary**  
- **Title**: “Apple Inc · 1D · NASDAQ · TradingView” confirms the symbol, timeframe, and platform.  
- **Price Boxes**:  
  - Red box: “153.00” (likely a reference price, e.g., prior close or target).  
  - Blue box: “154.62” (current price or another reference).  
- **Key Metrics**:  
  - “O 158.93 H 159.28 L 154.50 C 154.73” = Open, High, Low, Close prices for the current day.  
  - “-3.79 (-2.4%)” = Daily price change (negative, indicating a decline).  
- **Volume**: “Vol 96.943M” = Trading volume for the period.  


### 3. **Chart Area**  
- **Candlestick Chart**: Displays price action with green (up) and red (down) candles. Each candle shows open, high, low, and close prices for the day.  


### 4. **Left Sidebar (Tools)**  
Icons for charting tools (e.g., trendlines, Fibonacci retracements, text labels, zoom, and other analysis features) are visible, allowing users to annotate or modify the chart.  


### Purpose  
This interface is used for **technical analysis** of AAPL’s stock, enabling traders to track price trends, add indicators, set alerts, and use drawing tools to identify patterns or make trading decisions.
