- [ Help Center ](/) - / - / [ I got a 'Your chart layout was changed' message ](/support/solutions/43000484320-i-got-a-your-chart-layout-was-changed-message/) # I got a 'Your chart layout was changed' message This message means that the chart layout you are working with has been saved elsewhere. You can continue working with the layout in your current window, but if you add any changes and save it, then all modifications made on a different device/in a different browser or tab will be lost. We display this message to make sure that you don't lose your work/analysis.
