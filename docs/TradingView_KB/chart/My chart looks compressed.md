# My chart looks compressed

**URL:** https://www.tradingview.com/support/solutions/43000762934-my-chart-looks-compressed/

---

- [ Help Center ](/) - / - / [ Chart and/or indicators are being displayed incorrectly ](/support/folders/43000547290-chart-and-or-indicators-are-being-displayed-incorrectly/) - / [ My chart looks compressed ](/support/solutions/43000762934-my-chart-looks-compressed/) # My chart looks compressed If you use an indicator that has levels that are located far from the symbol series, the candles become compressed since automatic scaling forces the chart to fit all indicator levels in the visible area. To resolve this, you may enable the "Scale price chart only" option in the price scale right-click context menu.
