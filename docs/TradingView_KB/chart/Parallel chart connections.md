- [ Help Center ](/) - / - / [ Why I'm getting a message/warning/notice on my chart page? ](/support/folders/43000550182-why-i-m-getting-a-message-warning-notice-on-my-chart-page/) - / [ Parallel chart connections ](/support/solutions/43000694474-parallel-chart-connections/) # Parallel chart connections Your [Supercharts](https://www.tradingview.com/chart/) can support up to 200 simultaneous chart connections in the web version of TradingView and in the desktop or mobile app. **! Note:** Each tab in the web version and apps is counted as one connection. Opening one tab in your browser and one in an app counts as two connections. Once the limit is reached, the oldest connection (the earliest opened tab) will get closed. For the oldest closed connection, you'll get the following pop-up message. If you'd like to reconnect and save drawing tools or other changes in your chart layout, click "Restore connection" and then save the layout. Restoring the current connection when the maximum connections limit has already been exceeded will terminate the connection in another tab. To optimize your connections, you can use the [multi-chart layouts](https://www.tradingview.com/support/solutions/43000629990-leveraging-multi-chart-layouts-in-your-analysis/) feature.
