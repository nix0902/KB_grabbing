**Описание:** This is a **TradingView notification modal** prompting the user to manage chart drawings for performance optimization. Here’s a breakdown of its elements:  

### 1. Header & Close Button  
- **Title**: *“It’s time to clean up!”* (bold, top-left) – Alerts the user to review chart graphics.  
- **Close Icon** (top-right, “×”) – Dismisses the modal without taking action.  


### 2. Explanatory Text  
- **Message**: *“You have a lot of graphic objects on the chart. Some of them might not be needed anymore. Remove drawings that you don’t plan to use to improve performance.”* (centered, regular font) – Explains the purpose: reducing unnecessary chart objects to boost performance.  


### 3. Action Buttons  
- **“Not Now”** (left, gray outline button) – Dismisses the modal, deferring the cleanup task.  
- **“Open Manage Drawings”** (right, blue solid button) – Opens the “Manage Drawings” tool to review and delete unused chart objects (e.g., trendlines, shapes, indicators).  


### Purpose  
The modal targets users with excessive chart graphics, offering a quick way to optimize performance by cleaning up unused drawings. The “Not Now” button provides an opt-out, while “Open Manage Drawings” initiates the cleanup workflow.
