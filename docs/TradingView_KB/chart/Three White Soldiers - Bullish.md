- [ Help Center ](/) - / - / Chart - / [ Learn more about candlestick patterns ](/support/folders/43000570503-learn-more-about-candlestick-patterns/) - / [ Three White Soldiers - Bullish ](/support/solutions/43000583793-three-white-soldiers-bullish/) # Three White Soldiers - Bullish This bullish reversal pattern is made up of three long-bodied, green candles in immediate succession. Each one opens within the body before it and the close is near to the daily high. The bearish version of this pattern is the[ Three Black Crows pattern](https://www.tradingview.com/chart/?solution=43000583792).
