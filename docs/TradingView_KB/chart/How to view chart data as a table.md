Each table’s row corresponds to a specific point in time, while columns display values for symbols and indicators. This can make your analysis more flexible and precise, allowing you to compare values, track changes, and spot patterns without extra clicks. To switch to table view, right-click anywhere on your chart background and select “Table view” from the context menu. Here are some key points you should keep in mind when working with the table: - Values in the table match exactly how they appear on the chart. - Data is sorted by time, with the latest information always at the top. - The table refreshes automatically, ensuring you never miss critical data. - The data for hidden indicators is not shown in the table. Only active data is displayed. - You can change symbols in table view from the right-hand watchlist panel or by typing a symbol from your keyboard.
