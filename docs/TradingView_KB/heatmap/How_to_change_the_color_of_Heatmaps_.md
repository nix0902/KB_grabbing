# How to change the color of Heatmaps? 
 There are three color schemes available in the Heatmap for viewing maps:

- Classic

- Color-blind

- Monochrome

The last two modes are suitable for people with color perception peculiarities.

To change the color scheme, you must click on the cogwheel button.
