- [ Help Center ](/) - / - / Heatmap - / [ How to work with Heatmap ](/support/folders/43000591054-how-to-work-with-heatmap/) - / [ How to change the color of Heatmaps? ](/support/solutions/43000707158-how-to-change-the-color-of-heatmaps/) # How to change the color of Heatmaps? There are three color schemes available in the Heatmap for viewing maps: - Classic - Color-blind - Monochrome The last two modes are suitable for people with color perception peculiarities. To change the color scheme, you must click on the cogwheel button. In the settings dropdown, select the desired mode: Previous Previous How to set up the display of the Heatmap?
