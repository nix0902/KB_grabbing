What is size by and color by? ](/support/solutions/43000707144-how-to-set-up-a-heatmap-what-is-size-by-and-color-by/) # How to set up a Heatmap? What is size by and color by? The following parameters are available for work on market heatmaps: Size - the parameter affects the sorting of the entire map and the size of the symbol cell in proportion to the selected parameter. The color parameter affects the color of the map. By this parameter, you can filter the values using the map legend. It is necessary to select the dimension of the thermal multiplier of the legend.
