- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Watchlist - / [ How to manage text notes ](/support/folders/43000584933-how-to-manage-text-notes/) - / [ How to create a text note ](/support/solutions/43000667881-how-to-create-a-text-note/) # How to create a text note There are several ways to create a text note: - using hotkey **Alt/Option + N** ; - opening the context menu from the **More** button on the data series or by right-clicking on the chart; - using the context menu of the list of quotes or the screener; - using the **Add note** button in the details widget or by clicking on an existing note. To add a new note, simply enter your text in the window and click the **Add note** button. Previous Previous How to customize the Details section?
