- [ Help Center ](/) - / - / Watchlist - / [ How to manage my lists ](/support/folders/43000549989-how-to-manage-my-lists/) - / [ How to import or export a watchlist ](/support/solutions/43000487233-how-to-import-or-export-a-watchlist/) # How to import or export a watchlist To export your watchlist, first, open it and then find the "Advanced view" button. In the dialog box, find the table with quotes and click the "Export list" button. To import a watchlist, click on the name of the watchlist in the right toolbar and select the "Import list…" function from the menu. **! Note:** The input file must be in the .txt format and symbols should have the exchange prefix and comma separated.
