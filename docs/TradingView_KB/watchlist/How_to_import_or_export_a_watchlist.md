# How to import or export a watchlist 
 To export your watchlist, first, open it and then find the "Advanced view" button.

In the dialog box, find the table with quotes and click the "Export list" button.

To import a watchlist, click on the name of the watchlist in the right toolbar and select the "Import list…" function from the menu.

**! Note:** The input file must be in the .txt format and symbols should have the exchange prefix and comma separated.

### 1. Top Navigation Bar  
- **Left**:  
  - Lightbulb icon + “Layout A” (dropdown): Switches between predefined chart layouts.  
  - Lightbulb icon: Toggles “Idea” (trading ideas) visibility.  
- **Middle**:  
  - Lightning bolt: Toggles “Alerts” (price/indicator alerts).  
  - Hexagon: Toggles “Pine Editor” (for creating custom indicators).  
  - Bracket icon: Toggles “Chart Layout” (resize/move chart panels).  
  - Camera icon: Toggles “Chart Snapshots” (save chart images).  
- **Right**:  
  - “Publish” button: Publishes custom scripts/ideas to the TradingView community.  


### 2. Watchlist Section (Main Area)  
- **Header**:  
  - Star icon + “Watchlist” (dropdown arrow): Opens the list management menu (shown in the dropdown).  
  - “+” icon: Adds a new item to the watchlist.  
  - Grid icon: Toggles “List View” (switches between list/grid display).  
  - “⋯” (three dots): More options for the watchlist.  
- **Dropdown Menu (Open)**:  
  - **“Share list” toggle**: Enables/disables sharing the watchlist (blue border = active).  
  - “Copy link”: Copies the watchlist’s shareable URL.  
  - “Add alert on the list…”: Sets price/indicator alerts for the entire list.  
  - “Make a copy…”: Duplicates the watchlist.  
  - “Rename”: Renames the watchlist.  
  - “Add section”: Organizes the watchlist into sections (e.g., “Stocks,” “Crypto”).  
  - “Clear list”: Removes all items from the watchlist.  
  - “Create new list…”: Makes a new watchlist.  
  - **“Import list…” (highlighted in red)**: Imports a watchlist from a file/link.  
  - **“RECENTLY USED”**: Shows recently accessed lists:  
    - “Watchlist” (star icon, dark background = active).  
    - “Favorites” (cat icon).  
    - “My list” (rocket icon).  
    - “US Exchanges, Gap Gainers” (no icon).  
  - “Open list…”: Opens a saved watchlist (with “W” shortcut hint).  


### 3. Watchlist Data (Right of Dropdown)  
A table of assets with:  
- **Columns**: (Implied) Asset name, “Chg%” (percentage change), “Ext” (extended data, e.g., exchange).  
- **Rows**: Assets with percentage changes (e.g., `0.07%`, `1.80%`, `1.98%`, `0.49%`, `1.28%`, `0.66%`, `1.21%`, `-0.14%`, `0.84%`). Green = positive change, red = negative.  


### 4. Bottom Navigation  
- **Left**: Apple logo + “AAPL”: Shows the currently selected asset (Apple stock) in the watchlist.  
- **Middle**:  
  - Grid icon: Toggles “List View” for the watchlist.  
  - Pencil icon: Edits the watchlist (add/remove assets).  
  - “⋯” (three dots): More options for the selected asset.  


### 5. Right Sidebar (Icons)  
Vertical icons for quick access:  
- Clock: “Time” (timeline/period selector).  
- Layers: “Chart Type” (candlestick/line/area).  
- Chat bubble: “Chat” (trading community).  
- Target: “Drawing Tools” (trendlines, Fibonacci).  
- Calendar: “Events” (earnings/dividend calendars).  
- Grid: “Indicators” (add technical indicators).  
- Wi-Fi: “Data Feed” (switch data sources).  
- Question mark: “Help” (TradingView support).  


### Purpose of the Interface  
This screen is for **managing watchlists** (tracking assets) and customizing the TradingView layout. The dropdown menu lets users share, copy, rename, import, or organize watchlists, while the sidebar provides quick access to charts, alerts, and community features. The watchlist displays real-time percentage changes for tracked assets (e.g., stocks, crypto).
