# I want to change the width of my watchlist columns 
 You can customize the width of any column of your watchlist as follows: move the mouse cursor over the name of the required column, find the right border, hold down the left mouse button, and drag the border to the desired column width.

Please note that all column settings are stored in the local storage of the browser, so you don’t have to configure them every time you switch from mobile to desktop.
