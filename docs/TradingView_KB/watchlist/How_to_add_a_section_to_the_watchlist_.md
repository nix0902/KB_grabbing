# How to add a section to the watchlist? 

- Open the required list
- Select the symbol above which you want to create a section
- Right-click on it to open the menu
- Select the Add section menu item
 You can learn more about sections in our [blog post](https://www.tradingview.com/blog/en/new-watchlist-features-21389/).
