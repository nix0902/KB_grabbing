- [ Help Center ](/) - / - / Screener - / [ Features and tools ](/support/folders/43000593100-features-and-tools/) - / [ Currency conversion in the Stock Screener ](/support/solutions/43000639101-currency-conversion-in-the-stock-screener/) # Currency conversion in the Stock Screener Stock Screener and ETF Screener displays some data in currency format. For example price data (such as the Last Price of a symbol) is always presented in the currency of a given company, which may not correspond to the local market currency. Please note that this data cannot yet be converted. Financial data (Market Capitalization, Basic EPS, etc.) is listed in the market currency, but can have up to nine conversion options. - Default currency: the currency of the local market in which the company is located. - USD, EUR, JPY, GBP, AUD, CAD, CHF, CNY: values can be converted to this currency on any market. For example, there are different ticker currencies for the UK market (this can be seen from the LAST column). 1.The Default conversion option in the Financials menu (GBP in the current example) converts the financial values into the currency of the local market (e.g. British pound sterling for the UK market.) 2. The next conversion option recalculates the financial values into USD currency. Note that the Market Cap column is recalculated, while the LAST column remains in the company’s currency. **Описание:** This TradingView **Market Screener** interface displays a filtered list of UK-listed stocks with detailed financial data. Here’s a breakdown of key elements:  


### **Top Navigation & Filters**  
- **Header**: “Market Screener” (dropdown) + “Save” (cloud icon, to save the screener) + gear icon (settings) + expand/collapse arrow.  
- **Filter Buttons** (left to right):  
  - *Market*: GB (UK) flag + dropdown.  
  - *Index*: Dropdown (e.g., FTSE 100).  
  - *Price*: Dropdown (price range filter).  
  - *Change %*: Dropdown (percentage change filter).  
  - *Market cap*: Dropdown (market capitalization filter).  
  - *P/E*: Dropdown (price-to-earnings ratio filter).  
  - *Sector*: Dropdown (industry sector filter).  
  - *Analyst Rating*: Dropdown (analyst recommendations).  
  - *Perf %*: Dropdown (performance filter).  
  - *Revenue growth*: Dropdown (revenue growth filter).  
  - *PEG*: Dropdown (price/earnings-to-growth ratio).  
  - *ROE*: Dropdown (return on equity filter).  
- **Additional Filters**:  
  - *Primary Listing*: “Yes” (dropdown, filters for primary listings).  
  - *+*: Add new filter.  
  - *Reset all*: Clear all filters.  


### **Screen Settings (Dropdown Menu)**  
- *Auto refresh every min*: Toggle (enable/disable auto-refresh).  
- *Financials currency*: USD (dropdown, set currency for financial metrics).  


### **Tab Navigation**  
- Tabs: *Overview* (active), *Performance*, *Extended Hours*, *Valuation*, *Dividends*, *More* (dropdown).  
- Icons: Grid/list toggle (left), refresh (right), expand/collapse columns (far right).  


### **Data Table**  
- **Search/Count**: *Symbol* (search bar) + “1182” (total results).  
- **Columns** (sortable):  
  - *Symbol*: Ticker (e.g., AZN, SHEL).  
  - *Name*: Full company name (e.g., AstraZeneca, Shell).  
  - *Price*: Current price (e.g., 10394 GBX).  
  - *Change %*: Daily percentage change (green = +, red = -).  
  - *Volume*: Trading volume (e.g., 215.305K).  
  - *Rel Volume*: Relative volume vs. average.  
  - *Market cap*: Market capitalization (e.g., 202.805B USD).  
  - *P/E*: Price-to-earnings ratio.  
  - *EPS*: Earnings per share (e.g., 3.75 USD).  
  - *+*: Add/remove columns.  


### **Stock Listings (Sample)**  
Rows show UK-listed companies (e.g., AstraZeneca, Shell, HSBC) with:  
- Ticker, full name, price, % change, volume, relative volume, market cap, P/E, and EPS.  
- Color-coded % change (green = positive, red = negative).  


### **Purpose**  
This screen lets users filter UK stocks by multiple criteria (market, sector, P/E, etc.) to identify investment opportunities, with real-time data (price, volume, market cap) and financial metrics (P/E, EPS) for analysis.
