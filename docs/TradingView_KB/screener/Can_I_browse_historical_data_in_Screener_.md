# Can I browse historical data in Screener? 
 At the moment, there is no filter in the screener that would allow using historical data. 

The point is that the screener is designed to somewhat reflect on the latest data, not the historical one. The chart has the concept of elapsed time (time axis), but the screener does not have it now.
