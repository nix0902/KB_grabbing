# Base and Quote currency 
 [CEX Screneer](https://www.tradingview.com/cex-screener/) has a filter that allows you to filter assets by base and quote currency:

- **Base currency **is the currency of the asset that is being considered for purchase. It is located first in the currency pair.

- **Quote currency** is the currency in units of which the price of one unit of the base currency. In other words, this is the currency in which the transaction with the Base currency is considered.

Combining these two filters allows you to quickly find the necessary trading pairs. Select the BTC option in the Base Currency filter if you need all pairs where Bitcoin is bought. After selecting the option, the filtered list shows BTC quotes in currencies.

If you need a specific pair (for example, "BTCUSD"), set the Base Currency filter to BTC and, at the same time, the Quote сurrency to USD.
