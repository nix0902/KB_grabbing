# Sharia-compliant securities 
 Sharia-compliant (halal) securities are investment instruments (such as stocks or bonds) of companies or issuers whose activities comply with the rules and principles of the Islamic financial system. Key aspects of Islamic economic principles include the prohibition of interest income (riba), the prohibition of investments related to the production of alcohol, tobacco, weapons, gambling, and other activities and products prohibited in Islam, as well as the requirement of fairness and ethics in investments. Trading in such securities is available both on exchanges in Islamic countries and on major international platforms.

**Описание:**

This image shows a **TradingView Stock Screener** interface, designed for filtering and analyzing stocks based on customizable criteria. Here’s a detailed breakdown of all elements:  


### 1. **Header & Global Controls**  
- **“Stock Screener” Dropdown**: A dropdown menu (with a downward arrow) to switch between different screener types (e.g., stock, crypto, forex).  
- **“Save” Button**: A cloud icon + “Save” text, used to save the current screener configuration for future use.  
- **Theme/Settings Icons**: Two icons on the right (a hexagon for theme/settings, and an upward arrow for expanding/collapsing the interface).  


### 2. **Filter Tabs (Top Row)**  
A series of dropdown filters to refine stock searches:  
- **Market**: Dropdown (globe icon + “Entire world”) to filter by geographic region (e.g., US, Europe, Asia).  
- **Index**: Dropdown to filter by stock index (e.g., S&P 500, NASDAQ).  
- **Price**: Dropdown to filter by price range.  
- **Change %**: Dropdown to filter by percentage price change (e.g., daily, weekly).  
- **Market cap**: Dropdown to filter by market capitalization (e.g., large-cap, mid-cap).  
- **P/E**: Dropdown to filter by Price-to-Earnings ratio.  


### 3. **Additional Filters (Second Row)**  
More advanced filters:  
- **EPS dil growth**: Dropdown for Earnings Per Share (diluted) growth.  
- **Div yield %**: Dropdown for dividend yield percentage.  
- **Sector**: Dropdown to filter by industry sector (e.g., Technology, Healthcare).  
- **Analyst Rating**: Dropdown for analyst ratings (e.g., Buy, Hold, Sell).  
- **Perf %**: Dropdown for performance metrics (e.g., 1-year return).  
- **Revenue growth**: Dropdown for revenue growth rate.  


### 4. **Third Row Filters**  
- **PEG**: Dropdown for Price/Earnings-to-Growth ratio.  
- **ROE**: Dropdown for Return on Equity.  
- **Beta**: Dropdown for stock volatility (beta coefficient).  
- **“Sharia compliant Yes” Filter**: A highlighted filter (blue background) showing the current selection: *“Sharia compliant Yes”* (indicating only stocks compliant with Islamic finance rules are shown).  
  - A dropdown arrow (↑) allows expanding this filter.  
  - A “Reset” option (with an “X” icon) to clear this filter.  
- **“+” Button**: To add a new filter.  
- **“Reset all”**: To clear all applied filters.  


### 5. **View & Layout Controls**  
- **Grid/List Toggle**: Icons to switch between grid (table) and list views.  
- **“Overview” Dropdown**: A dropdown to change the data view (e.g., “Overview,” “Fundamentals,” “Technical”).  
- **Refresh/Sort Icons**: A circular refresh icon (to reload data) and a sort icon (to reorder columns).  


### 6. **Search & Results Table**  
- **“Symbol” Search Bar**: A search field (with a magnifying glass icon) to find specific stocks. The number “1806” indicates the total results matching the filters.  
- **Table Columns**: Headers for:  
  - Symbol (stock ticker, e.g., “CATL,” “SHOP”).  
  - Company Name (e.g., “CONTEMPORARY AM,” “SHOPIFY INC”).  
  - Price (e.g., “37.07 CNY,” “103.02 CAD”).  
  - Change % (percentage price change, colored red for losses, green for gains).  
  - Volume (trading volume, e.g., “34.799M”).  
  - Rel Volume (relative volume vs. average).  
  - Market cap (market capitalization, e.g., “113.323B USD”).  
  - P/E (Price-to-Earnings ratio, e.g., “18.37”).  
- **“+” Button**: To add more columns to the table.  


### 7. **Stock Results (Table Rows)**  
Each row shows a stock with:  
- **Ticker & Logo**: E.g., “CATL” (Contemporary Amperex Technology), “SHOP” (Shopify Inc).  
- **Price & Change %**: E.g., “37.07 CNY” (price) with “-0.30%” (loss, red).  
- **Volume & Rel Volume**: E.g., “35.628M” (volume) and “0.88” (relative volume).  
- **Market Cap**: E.g., “98.489B USD” (China Shenhua Energy).  
- **P/E**: E.g., “792.46” (Shopify Inc, indicating high valuation).  


### 8. **“Sharia Compliant” Dropdown Menu (Open)**  
A dropdown menu is open under the “Sharia compliant” filter, showing:  
- **“Reset”**: Option to clear the filter (with an “X” icon).  
- **“SHARIA COMPLIANT” Header**: Label for the filter category.  
- **“Yes” (Selected)**: Highlighted in blue, confirming the filter is set to “Yes” (only Sharia-compliant stocks are displayed).  
- **“Remove”**: Option to delete the “Sharia compliant” filter entirely (with a trash icon).  


### Purpose of the Interface  
This screen is used to **screen stocks** by applying multiple filters (e.g., Sharia compliance, market cap, P/E) to narrow down results. Users can save filters, reset them, or add new ones to find stocks matching their investment criteria (e.g., ethical investing, value stocks, high-growth stocks). The table displays real-time or delayed stock data (price, volume, market cap) for the filtered results.
