If the added columns or filters include pre-market columns, pre-market trades will be considered. Otherwise, only trades made during the regular session will be counted. To disable this filter, click the Reset button: **Trading Session for Crypto assets ** For crypto assets, the trading session lasts 24 hours. The "Last trade time" filter works the same way : - **Current trading day **– shows cryptocurrencies that have been traded since the beginning of the current trading session. - **Before current trading day** – displays cryptocurrencies that have not been traded since the beginning of the current trading session. **Описание:** This is a **TradingView Stock Screener** interface for filtering and analyzing stocks. Here’s a detailed breakdown:  


### **Top Navigation & Controls**  
- **Header**: “Stock Screener” (dropdown menu) + “Save” (cloud icon, to save filters) + a settings icon (gear) + a collapse/expand arrow.  
- **Filter Buttons** (horizontal row, each with a dropdown arrow):  
  - *Market*: “US” (with US flag, to select region).  
  - *Watchlist, Index, Price, Change %, Market cap, P/E, EPS dil growth, Div yield %, Sector, Analyst Rating, Perf %, Revenue growth, PEG, ROE, Beta, Recent earnings date, Upcoming earnings date*.  


### **Active Filter & Add Filter**  
- **Active Filter**: “Last trade time” (dropdown menu is open, showing options: *Current trading day, Before current trading day* + a “Remove” trash icon).  
- **Add Filter**: “+” button (to add new filters).  


### **Tab Bar**  
- Tabs: *Overview* (selected, bold), *Price, Hours, Valuation, Dividends, Profitability, More* (dropdown for additional tabs).  
- A refresh icon (right) to reload data.  


### **Stock Table**  
- **Search/Filter**: “Symbol” field (with magnifying glass) + “13721” (current filter count).  
- **Columns** (with headers):  
  - *Symbol*: “AAPL” (Apple) with its logo.  
  - *Price*: “222.6” (current price).  
  - *Volume*: “4.6M” (trading volume).  
  - *Rel Volume*: “0.09” (relative volume).  
  - *Market cap*: “3.34T USD” (market capitalization, with a downward arrow indicating sort direction).  
  - *P/E*: “35.40” (price-to-earnings ratio).  
  - *EPS dil TTM*: “6.29 USD” (diluted earnings per share, trailing twelve months).  
  - *EPS dil growth TTM YoY*: “-2.13%” (year-over-year growth, red for negative).  
  - *Div*: “0.45” (dividend, partially visible).  
- A “+” button (right) to add more columns.  


### **Purpose**  
This interface lets users filter stocks by criteria (e.g., market, sector, P/E, earnings dates) and view key metrics (price, volume, market cap, EPS) to identify investment opportunities. The “Last trade time” filter refines results by when stocks last traded, while tabs organize data into categories (overview, valuation, profitability, etc.).
