- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Screener - / [ Features and tools ](/support/folders/43000593100-features-and-tools/) - / [ Chart view mode ](/support/solutions/43000724233-chart-view-mode/) # Chart view mode Now there are two ways of data display in Stock, ETF, CEX, DEX, and Crypto Сoins screeners: - Table view — data is presented in a table with column sets. - Chart view — data is presented in charts with price change on the selected interval and trading volume of the instrument. To switch between these two modes, click on Table view or Chart view in the top left corner of a screener: You can configure the display of charts. There are 2 setting modes available: - Auto - Manual You can read more information[ here](https://www.tradingview.com/support/solutions/43000728343-how-can-i-change-the-chart-view-grid/). You can click on Chart type button and select type after the Chart grid button. By default, Area chart type is selected, but you can also opt for other types: Bars, Candles and Line. You can also select a date range on charts: It’s also worth remembering that each range is associated with a specific timeframe in the Screener Chart view. There are the following range options: - 1 day — 5-minute timeframe - 5 days — 15-minute timeframe - 1 month — 1-hour timeframe - 3 months — 4-hour timeframe - 6 months — 1-day timeframe - YTD — 1-day timeframe - 1 year — 1-week timeframe - 5 years — 1-month timeframe - All data — 1-month timeframe - For example, if you select a 1-year range in the Screener Сhart view, each bar will equal one week.
