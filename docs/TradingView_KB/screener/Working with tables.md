To add a new column to the table, click on the button in the table header's right corner. You can also open the Add column dialog using the Shift + C shortcut. After selecting the required column, the column will immediately be added to the table. Adding the same indicator to the new Screener is possible but with a different set of parameters. For example, you can add a Change % column with a different set of timeframes: To sort by column, click on the column header and select the column sorting direction. To change the position in a column, you can use the move options: You can also move a column to any location by grabbing the column header and dragging it to a new position: To change the configuration of a column in a table, in the context menu, you need to select the Column settings action and set new column parameters: Select the Remove action in the context menu to remove a column in a table. **Описание:** This TradingView image displays a **stock screener table** with a context menu open, showing 8 listed securities and their key metrics. Here’s a detailed breakdown:  


### **Top UI Elements**  
- **Search/Filter Bar**: Left of the table, showing “Number ↑” (sorted by number of results) and “14248” (total results). A magnifying glass icon indicates search functionality.  
- **Column Headers**: Horizontal tabs for sorting/filtering: *Price, Change % (selected, highlighted), Volume, Rel Volume, Market cap, P/E, EPS dil TTM, EPS dil growth TTM YoY, Div yield % TTM, Sector*. A “+” icon on the far right allows adding new columns.  


### **Context Menu (Open on “Change %” Column)**  
A dropdown menu provides actions for the *Change %* column:  
- *PRICE CHANGE % 1 DAY* (header label for the column).  
- *Sort ascending* (↑= icon) / *Sort descending* (↓= icon): Toggle sort order of the “Change %” column.  
- *Move left/right*: Reorder column position.  
- *Move to the start/end*: Jump column to first/last position.  
- *Column settings…*: Customize column visibility/format.  
- *Remove*: Delete the “Change %” column.  


### **Table Rows (Securities)**  
Each row represents a stock with:  
- **Ticker/Logo**: Leftmost (e.g., “A” for Agilent Technologies, with a blue logo).  
- **Company Name**: Full name (e.g., “Agilent Technologies, Inc.”) with a “D” (delayed data) indicator.  
- **Price**: Current price (e.g., “140.21 USD”).  
- **Change %**: Daily price change (green = positive, red = negative; e.g., “+0.45%” for Agilent).  
- **Volume/Rel Volume**: Trading volume and relative volume (e.g., “0.19” for Agilent).  
- **Market Cap**: Total market value (e.g., “41.024B USD” for Agilent).  
- **P/E**: Price-to-earnings ratio (e.g., “33.40” for Agilent).  
- **EPS dil TTM**: Diluted earnings per share (trailing twelve months; e.g., “4.20 USD” for Agilent).  
- **EPS dil growth TTM YoY**: Year-over-year EPS growth (e.g., “+0.45%” for Agilent).  
- **Div yield % TTM**: Trailing twelve months dividend yield (e.g., “0.64%” for Agilent).  
- **Sector**: Industry classification (e.g., “Health Technology” for Agilent).  


### **Example Rows**  
- **Agilent Technologies (A)**: Health Technology, +0.45% change, 41.024B market cap.  
- **Alcoa Corporation (AA)**: Non-Energy Minerals, -491.83% change (extreme drop), 5.782B market cap.  
- **Asia Broadband (AABB)**: Finance, 0.00% change, 50.86M market cap.  


### **Purpose**  
This screen is used to **screen stocks** by metrics (price, change, volume, sector, etc.). The context menu lets users customize column sorting, order, or visibility to analyze securities efficiently.
