**Описание:** This TradingView image shows the **Stock Screener 2.0** interface with a **Market data** dropdown menu open. Here’s a detailed breakdown:  


### **Top Navigation & Filters**  
- **Header**: “Stock Screener 2.0” with a dropdown arrow (for version/selection).  
- **Market/Region**: “Market US” (dropdown for region, e.g., US stocks).  
- **Filter Buttons** (left to right):  
  - *Index*: Dropdown (e.g., S&P 500, Dow Jones).  
  - *Price*: Dropdown (filter by price range).  
  - *Change %*: Dropdown (filter by price change percentage).  
  - *Market cap*: Dropdown (filter by market capitalization).  
  - *P/E*: Dropdown (filter by price-to-earnings ratio).  
  - *EPS dil growth*: Dropdown (filter by diluted EPS growth).  
  - *Div yield %*: Dropdown (filter by dividend yield).  
  - *Sector*: Dropdown (filter by industry sector).  
  - *Analyst Rating*: Dropdown (filter by analyst ratings).  
  - *Perf %*: Dropdown (filter by performance).  
  - *Revenue growth*: Dropdown (filter by revenue growth).  
  - *PEG*: Dropdown (filter by PEG ratio).  
  - *ROE*: Dropdown (filter by return on equity).  
  - *Beta*: Dropdown (filter by beta).  
  - **+**: Button to add more filters.  


### **Tab Bar**  
Tabs for different data views: *Overview* (active), *Performance*, *Extended Hours*, *Valuation*, *Dividends*, *Profitability*, *More* (dropdown for additional tabs).  


### **Market Data Dropdown Menu**  
A dropdown titled *Market data* is open, showing **extended hours/pre-market metrics** (for after-hours trading data). Options include:  
- *Post-market Change %* (selected, highlighted).  
- *Post-market Change*  
- *Post-market Close*  
- *Post-market High*  
- *Post-market Low*  
- *Post-market Open*  
- *Post-market Volume*  
- *Pre-market Change %*  
- *Pre-market Change*  
- *Pre-market Change from Open %*  


### **Stock Table (Overview Tab)**  
A table lists stocks with columns:  
- **Symbol**: Ticker (e.g., “A”, “AA”).  
- **Company Name**: (e.g., “Agilent Technologies, Inc.”).  
- **Price**: Current price (e.g., “138.18 usd”).  
- **Change %**: Daily price change (e.g., “-1.15%”).  
- **Volume**: Trading volume (e.g., “2.121M”).  
- **Rel Vol**: Relative volume (not fully visible).  
- **Growth/Div Yield**: Additional metrics (e.g., “0.48%”, “0.65%”).  


### **UI Elements**  
- **Search Bar**: Top-left (magnifying glass icon) with “Symbol 14369” (total stocks screened).  
- **Settings/Expand Icons**: Top-right (gear for settings, arrow for full-screen).  
- **Column Headers**: Sortable (e.g., “Symbol ↑” indicates ascending sort by symbol).  


### **Purpose**  
This interface lets users **screen stocks** using filters (price, sector, P/E, etc.) and view real-time/extended-hours data. The dropdown menu customizes which market data metrics to display (e.g., post-market changes) for deeper analysis.
