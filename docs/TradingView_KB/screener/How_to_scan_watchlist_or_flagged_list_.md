# How to scan watchlist or flagged list? 

The Watchlist filter can be found in the upper toolbar. All colored and custom watchlists are available in the filter list:

Just select the necessary watchlist, and its instruments will appear in the table — but only if the assets correspond to the screener’s type. For instance, if your watchlist contains stocks and ETFs and you want to open it in the Stock screener, only the list’s stocks will appear in the table. If you select the same watchlist in the ETF screener, you’ll see only its ETFs.

Let’s consider another situation: the watchlist you want to use contains only cryptocurrencies, and you decide to open it in the Stock screener. In this case, you’ll see an empty table. 

You can save a screen with the applied Watchlist filter for quick access later.

If you create a new watchlist, it’ll be added to the filter menu.

Note, though: if you delete a watchlist, it’ll also disappear from the Watchlist menu.

This image shows the **TradingView Stock Screener interface** (a tool for filtering and analyzing stocks) with the following key elements:

### 2. **Stock Screener Header**  
- **“Stock Screener” Title**: Bold text with a downward arrow (dropdown) to expand/change the screener type.  


### 3. **Filter Tabs (Horizontal Row)**  
These tabs filter the stock list by criteria:  
- **Market**: “US” (with a U.S. flag icon) + downward arrow (filters by U.S. market).  
- **Watchlist**: “Red list” (with a red flag icon) + downward arrow (filters by a custom “red list” watchlist).

- **Price**: Dropdown (filters by price range).  
- **Sector**: Dropdown (filters by industry sector, e.g., Tech, Finance).  
- **Analyst Rating**: Dropdown (filters by analyst ratings).  
- **Perf %**: Dropdown (filters by performance percentage).  
- **Revenue growth**: Dropdown (filters by revenue growth metrics).  


### 4. **View Toggle & Tabs**  
- **Grid/List Toggle**: Two icons (grid for table view, wavy line for list view) to switch display formats.  
- **Tabs**: “Overview” (selected, bold), “Performance”, “Extended Hours”, and a truncated tab (e.g., “Valuation”) to switch between data views.  


### 5. **Stock Table Header**  
- **Search Icon + “Symbol”**: Search field for symbols (with “12” indicating 12 results).  
- **Columns**: “Price” (current price), “Change %” (percentage change), and a truncated column (e.g., “Volume” or “Market Cap”).  


### 6. **Stock List (Rows)**  
Each row displays a stock with:  
- **Red Flag Icon**: Indicates the stock is in the “Red list” watchlist.  
- **Company Logo**: Visual identifier (e.g., Tesla’s “T” logo, Mastercard’s circles).  
- **Ticker Symbol**: Short code (e.g., `TSLA`, `MA`).  
- **Company Name**: Full name (e.g., “Tesla, Inc.”) + icons:  
  - `D`: Indicates the stock is in the “Red list” watchlist.  
  - `DR`: Indicates a “Depository Receipt” (e.g., for foreign stocks).  
  - `P`: Indicates a “Preferred Stock”.  
- **Price**: Current price (e.g., `174.92 USD`).  
- **Change %**: Performance (red for negative, green for positive; e.g., `-0.42%` for Tesla, `+1.87%` for Oracle).

This screen lets users **filter, search, and analyze stocks** using criteria like market, watchlist, sector, price, and performance. The table displays key data (price, change %) for quick comparison, while filters help narrow results to match trading strategies.
