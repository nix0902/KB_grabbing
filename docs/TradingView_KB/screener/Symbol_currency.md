# Symbol currency 
 The **Symbol Currency** filter allows you to find stocks or ETFs based on the currency in which their market price is quoted and traded. Global companies often list on multiple exchanges in different local currencies. This filter helps manage currency exposure or focus on assets priced in a specific currency relevant to your portfolio or region.

To screen assets by their trading currency:

-Add the “Symbol Currency” filter to your screener setup.

In the added filter, select one or more currencies from the list. The screener will then only show assets priced in the selected currency/currencies.

The filter depends on the selected market. If you change the market, the options in the Symbol Currency filter will be adjusted according to the market criteria.

To disable this filter and see assets regardless of their trading currency, click the Reset button.

This image shows the **TradingView Stock Screener** interface, a tool for filtering and analyzing stocks based on various criteria. Here’s a detailed breakdown of the UI elements, their purposes, and what the interface displays:  


### 1. **Top Navigation & Header**  
- **“Stock Screener”**: The main title of the page, with a dropdown arrow (for additional screener options).  
- **“Save” (with cloud icon)**: Saves the current screener configuration (filters, columns, etc.) for later use.  
- **Settings/Theme Icon** (hexagon): Toggles app settings (e.g., theme, display preferences).  
- **Expand/Collapse Arrow**: Collapses or expands the top filter bar.  


### 2. **Filter Bar (Top Row of Buttons)**  
These are **dropdown filters** to narrow down stock results. Each has a dropdown arrow (▼) to reveal filter options:  
- **Market**: Filter by market (currently set to “US” with a US flag icon).  
- **Watchlist**: Filter by custom watchlists (e.g., “My Watchlist”).  
- **Index**: Filter by stock indices (e.g., S&P 500, Dow Jones).  
- **Price**: Filter by price range (e.g., “> $100”, “< $50”).  
- **Change %**: Filter by percentage price change (e.g., “+5% to +10%”).  
- **Market cap**: Filter by market capitalization (e.g., “Large-cap”, “Mid-cap”).  
- **P/E**: Filter by Price-to-Earnings ratio.  
- **EPS dil growth**: Filter by Earnings Per Share (diluted) growth.  
- **Div yield %**: Filter by dividend yield percentage.  
- **Sector**: Filter by industry sector (e.g., “Technology”, “Healthcare”).  
- **Analyst Rating**: Filter by analyst ratings (e.g., “Buy”, “Hold”).  
- **Perf %**: Filter by performance (e.g., 1-month, 1-year returns).  
- **Revenue growth**: Filter by revenue growth rate.  
- **PEG**: Filter by Price/Earnings-to-Growth ratio.  
- **ROE**: Filter by Return on Equity.  
- **Beta**: Filter by stock volatility (vs. the market).  
- **Recent earnings date**: Filter by recent earnings report dates.  
- **Upcoming earnings date**: Filter by upcoming earnings report dates.  
- **Symbol currency**: Currently open (shows a dropdown with “USD · US Dollar” selected, plus a “Remove” option to clear the filter).  
- **“+” Button**: Adds a new filter (e.g., custom metric, additional criteria).  


### 3. **Tab Bar (Below Filter Bar)**  
These tabs switch between different views of the screener:  
- **Grid/Chart Icon**: Toggles between table view (current) and chart view.  
- **“Overview”**: Current active tab (displays a summary of stocks).  
- **“Performance”**: Shows historical performance metrics (e.g., 1-day, 1-week returns).  
- **“Extended H”**: Likely “Extended Hours” (shows pre-market/post-market trading data).  
- **“More”**: Additional tabs (e.g., “Fundamentals”, “Technical”).  
- **Refresh Icon**: Reloads the screener data (updates stock prices, metrics).  


### 4. **Search & Column Headers**  
- **Search Bar (“Symbol”)**: Filters stocks by ticker symbol (e.g., “AAPL”, “MSFT”). The count “13875” shows the total number of stocks matching current filters.  
- **Column Headers**: Define the data displayed for each stock:  
  - **Symbol**: Ticker (e.g., AAPL, MSFT).  
  - **Price**: Current stock price (e.g., 203.19 USD).  
  - **Market cap**: Total market value (e.g., 3.05T USD = 3.05 trillion USD).  
  - **P/E**: Price-to-Earnings ratio (e.g., 32.30).

A popup menu for the “Symbol currency” filter:  
- **“SYMBOL CURRENCY” Header**: Label for the filter.

### Key Takeaways  
- The interface is used to **screen stocks** by applying filters (market, price, P/E, etc.) and view results in a table.
