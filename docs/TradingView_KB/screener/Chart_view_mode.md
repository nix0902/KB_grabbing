# Chart view mode 

Now there are two ways of data display in Stock, ETF, CEX, DEX, and Crypto Сoins screeners:

- Table view — data is presented in a table with column sets.
- Chart view — data is presented in charts with price change on the selected interval and trading volume of the instrument.

To switch between these two modes, click on Table view or Chart view in the top left corner of a screener:

You can configure the display of charts.

There are 2 setting modes available:

- Auto

- Manual

You can read more information[ here](https://www.tradingview.com/support/solutions/43000728343-how-can-i-change-the-chart-view-grid/).

You can click on Chart type button and select type after the Chart grid button. By default, Area chart type is selected, but you can also opt for other types: Bars, Candles and Line.

You can also select a date range on charts:

It’s also worth remembering that each range is associated with a specific timeframe in the Screener Chart view. There are the following range options:

- 1 day — 5-minute timeframe
- 5 days — 15-minute timeframe
- 1 month — 1-hour timeframe
- 3 months — 4-hour timeframe
- 6 months — 1-day timeframe
- YTD — 1-day timeframe
- 1 year — 1-week timeframe
- 5 years — 1-month timeframe
- All data — 1-month timeframe
- For example, if you select a 1-year range in the Screener Сhart view, each bar will equal one week.

With Chart view on in the Screener, you can apply all available filters and select the necessary market.

This image shows a **TradingView Stock Screener interface**, designed for filtering and analyzing stocks based on multiple criteria. Here’s a detailed breakdown of all elements:  


### 1. Header & Main Title  
- **“Stock Screener”**: The title of the tool, indicating its purpose (screening stocks).  
- **Dropdown arrow (next to “Stock Screener”)**: Allows switching between different screener types (e.g., “Stock Screener,” “Crypto Screener,” etc.).  


### 2. Top-Right Icons  
- **Hexagon icon**: Likely a “Settings” or “Preferences” button (for customizing the screener).  
- **Upward arrow icon**: A “Collapse/Expand” button (to minimize/maximize the screener panel).  


### 3. Filter Buttons (First Row)  
These are **dropdown filters** to narrow down stock results. Each has a downward arrow (▼) to reveal filter options:  
- **Market US**: Filters by U.S. markets (flag icon for “US” indicates the region).  
- **Watchlist**: Filters stocks in your watchlist.  
- **Index**: Filters by stock indices (e.g., S&P 500, Dow Jones).  
- **Price**: Filters by stock price (e.g., $0–$10, $100+).  
- **Change %**: Filters by percentage price change (e.g., “Up 5% today”).  
- **Market cap**: Filters by market capitalization (e.g., large-cap, mid-cap).  
- **P/E**: Filters by Price-to-Earnings ratio.  
- **EPS dil growth**: Filters by Diluted Earnings Per Share growth.  


### 4. Filter Buttons (Second Row)  
More dropdown filters for deeper analysis:  
- **Div yield %**: Filters by Dividend Yield percentage.  
- **Sector**: Filters by industry sector (e.g., Technology, Healthcare).  
- **Analyst Rating**: Filters by analyst ratings (e.g., “Buy,” “Hold”).  
- **Perf %**: Filters by performance (e.g., 1-month, 1-year return).  
- **Revenue growth**: Filters by revenue growth rate.  
- **PEG**: Filters by Price/Earnings-to-Growth ratio.  
- **ROE**: Filters by Return on Equity.  
- **Beta**: Filters by beta (volatility relative to the market).  


### 5. Filter Buttons (Third Row)  
Additional time-based and miscellaneous filters:  
- **Recent earnings date**: Filters by stocks with recent earnings reports.  
- **Upcoming earnings date**: Filters by stocks with upcoming earnings reports.  
- **“+” button**: Adds a *new custom filter* (e.g., custom metric, technical indicator).  


### 6. Tab Bar (Bottom Section)  
These tabs switch between different views of the selected stock:  
- **Chart view**: (Currently active, dark background) Displays the stock’s price chart.  
- **Overview**: Summary of the stock’s key metrics.  
- **Performance**: Historical performance data (e.g., returns over time).  
- **Extended Hours**: Pre-market/post-market trading data.  
- **Valuation**: Valuation metrics (e.g., P/E, P/B ratio).  
- **Dividends**: Dividend history and yield.  
- **Profitability**: Profitability ratios (e.g., net margin, ROA).  
- **More**: Additional tabs (e.g., “Technical,” “Fundamentals”) via a dropdown.  


### 7. Bottom-Right Icons  
- **Refresh icon**: Reloads the screener data (updates filters/results).  


### Purpose of the Interface  
This tool helps traders/investors **screen stocks** by applying multiple criteria (price, sector, performance, valuation, etc.) to find stocks matching their strategy. The tabs let users dive deeper into a stock’s chart, fundamentals, or performance after screening.

All elements work together to help traders/investors efficiently scan, analyze, and compare stocks.
