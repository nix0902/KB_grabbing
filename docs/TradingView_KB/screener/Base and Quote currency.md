- [ Help Center ](/) - / - / [ Base and Quote currency ](/support/solutions/43000752914-base-and-quote-currency/) # Base and Quote currency [CEX Screneer](https://www.tradingview.com/cex-screener/) has a filter that allows you to filter assets by base and quote currency: - **Base currency **is the currency of the asset that is being considered for purchase. It is located first in the currency pair. - **Quote currency** is the currency in units of which the price of one unit of the base currency. In other words, this is the currency in which the transaction with the Base currency is considered. Combining these two filters allows you to quickly find the necessary trading pairs. Select the BTC option in the Base Currency filter if you need all pairs where Bitcoin is bought. After selecting the option, the filtered list shows BTC quotes in currencies. If you need a specific pair (for example, "BTCUSD"), set the Base Currency filter to BTC and, at the same time, the Quote сurrency to USD. **Описание:** This TradingView **Screener** interface displays a cryptocurrency market data table with a **Base Currency dropdown menu** open, showing filtering options and a list of assets. Here’s a detailed breakdown:  


### 1. **Top Navigation & Filters**  
- **Header**: “Screener” (dropdown menu) + settings icon + expand/collapse arrow.  
- **Filter Buttons** (left to right):  
  - *Watchlist, Exchange, Symbol type, Base currency (open), Quote currency, Change %, Perf %, Volume, Vol Change %* (all dropdowns for filtering).  
  - **+** button: Add new symbols to the watchlist.  


### 2. **Base Currency Dropdown (Open)**  
- **Search Bar**: “Search” field to find currencies.  
- **Currency List** (with checkboxes):  
  - USD (US Dollar), EUR (Euro), BTC (Bitcoin), ETH (Ethereum), USDT (Tether), JPY (Yen), GBP (Pound Sterling), AUD (Australian Dollar), TRY (Turkish Lira), USDC (USDC), BRL (Brazilian Real), CNH (Offshore Yuan), PLI (Plugin Decentralized Oracle), CGPT (ChainGPT).  
- **Actions**: *Select all* (check all), *Remove* (delete selected).  


### 3. **Tab Navigation**  
- Tabs: *Overview* (active), *Performance, Tech* (inactive).  
- **View Toggle**: Grid/list icon + refresh icon (right).  


### 4. **Symbol Search & Count**  
- *Symbol* search bar (left) + “33392” (total symbols).  


### 5. **Data Table Columns**  
- **Symbol**: Ticker (e.g., `BTCUSDT.P`, `ETHUSDT.P`).  
- **Exchange**: Logo (e.g., Binance, Bybit, KuCoin).  
- **Volume 24h**: Trading volume (e.g., 18.4B, 17.4B).  
- **Vol Change % 24h**: Volume change (green = increase, e.g., +22.29%).  
- **Tech Rating**: Rating (e.g., “Neutral”, “Sell” with red down arrow).  
- **+** button: Add symbol to watchlist.  


### 6. **Table Rows (Examples)**  
- Rows show Bitcoin/Ethereum perpetual contracts (e.g., `BTCUSDT.P`, `ETHUSDT.P`) with exchange logos, 24h volume, volume change %, and tech ratings.  


### Purpose  
The Screener helps traders **filter, search, and analyze** cryptocurrency symbols by base currency, exchange, performance metrics (volume, change %), and technical ratings. The open dropdown lets users narrow results by selecting base currencies (e.g., USD, BTC) to focus on specific markets.
