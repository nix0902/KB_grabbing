**Описание:** This is a **TradingView Stock Screener 2.0** interface displaying a list of stocks filtered by user-defined criteria. Here’s a detailed breakdown:  


### **Top Navigation & Controls**  
- **Header**: “Stock Screener 2.0” (with a dropdown arrow for menu options).  
- **Icons**: Gear (settings) and upward arrow (expand/collapse) on the right.  
- **Left Sidebar**:  
  - *Save screen* (cloud icon) + *Export screen results* (download icon).  
  - *Create new screen…* (plus icon) + *Load screen…* (for saved filters).  
  - *Popular Screens* section: Pre-built filters (e.g., “Most capitalized,” “Highest net income,” “Healthy growth technology stocks”) with brief descriptions.  


### **Filter Bar (Top Middle)**  
Dropdown menus for filtering stocks by:  
- `Change %`, `Market cap`, `P/E`, `EPS dil growth`, `Div yield %`, `Sector`, `Analyst Rating`, `Perf %`, `Beta`.  
- A `+` button to add more filters.  


### **Tab Navigation (Below Filter Bar)**  
Tabs for detailed stock analysis:  
- `Valuation`, `Dividends`, `Profitability`, `Income Statement`, `Balance Sheet`, `Cash Flow`, `Technicals`.  


### **Stock Table (Main Content)**  
Columns (with data for each stock):  
- `Price`: Current stock price (e.g., 140.80 USD).  
- `Change %`: Daily price change (green = +, red = -; e.g., +0.72%).  
- `Volume`: Trading volume (e.g., 753.798K).  
- `Rel Volume`: Relative volume vs. average.  
- `Market cap`: Company market capitalization (e.g., 41.196B USD).  
- `P/E`: Price-to-earnings ratio.  
- `EPS dil TTM`: Diluted earnings per share (trailing 12 months).  
- `EPS dil growth TTM YoY`: EPS growth year-over-year.  
- `Div yield % TTM`: Dividend yield (trailing 12 months).  
- `Sector`: Industry classification (e.g., Health Technology, Finance).  


### **Stock Rows (Example Data)**  
Each row shows a stock with metrics (e.g., a “Health Technology” stock at 140.80 USD, +0.72% change, 41.196B market cap). Rows use color-coding:  
- Green: Positive `Change %` or `EPS dil growth`.  
- Red: Negative `Change %` or `EPS dil growth`.  


### **Purpose**  
The interface lets users **filter, analyze, and compare stocks** using fundamental/technical metrics, save custom screens, and export results. The table displays real-time data to identify investment opportunities.
