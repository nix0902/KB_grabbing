# Last trade time 
 There is the "Last trade time" filter is in the screener to filter active traded assets:

The "Last trade time" filter has two options:

- **Current trading day** option shows symbols that were traded in the current session of the selected market.
- **Before current trading day** option shows symbols that were not traded in the current session.

If the added columns or filters include pre-market columns, pre-market trades will be considered. Otherwise, only trades made during the regular session will be counted.

To disable this filter, click the Reset button:

**Trading Session for Crypto assets **

For crypto assets, the trading session lasts 24 hours.
The "Last trade time" filter works the same way :

- **Current trading day **– shows cryptocurrencies that have been traded since the beginning of the current trading session. 
- **Before current trading day** – displays cryptocurrencies that have not been traded since the beginning of the current trading session.
