If you want to find the condition of an indicator to the current symbol price (e.g. stocks that are traded above their SMA10), which is indicated as "Close" on the chart, use the "Price" value in the Screener. Please note that the "Price" value in the screener is the symbol's regular session value. If you enable extended hours on the chart, the "Price" of the screener and the "Close" of the chart will not match. Previous Previous Can I browse historical data in Screener? **Описание:** This TradingView image displays a **stock screener interface** with a focus on filtering and viewing stock data. Here’s a detailed breakdown:  


### **Top Filter Bar**  
A horizontal row of dropdown filters for refining stock searches:  
- **Market**: Dropdown (currently “US”) to select market region.  
- **Index**: Dropdown (e.g., S&P 500, Dow Jones) to filter by index.  
- **Price**: Dropdown with “SMA (10)” (Simple Moving Average, 10-day) to filter by price relative to this indicator.  
- **Change %**: Dropdown to filter by percentage price change.  
- **Market cap**: Dropdown to filter by market capitalization.  
- **P/E**: Dropdown to filter by Price-to-Earnings ratio.  
- **EPS dil growth**: Dropdown to filter by diluted earnings per share growth.  
- **Div yield %**: Dropdown to filter by dividend yield.  
- **Sector**: Dropdown to filter by industry sector.  
- **Analyst Rating**: Dropdown to filter by analyst ratings.  
- **Perf %**: Dropdown to filter by performance metrics.  
- **Revenue growth**: Dropdown to filter by revenue growth.  
- **PEG**: Dropdown to filter by Price/Earnings-to-Growth ratio.  
- **ROE**: Dropdown to filter by Return on Equity.  
- **Beta**: Dropdown to filter by volatility (beta).  
- **+**: Button to add a new filter.  


### **Tab Navigation**  
Below the filter bar, tabs for different data views:  
- **Overview** (active, highlighted), **Performance**, **Extended Hours**, **Valuation**, **Dividends**, **Profitability**, **Income Statement**, **Balance Sheet**, **Cash Flow**, **More** (dropdown for additional tabs).  
- **Toggle Columns**: Button (up/down arrows) to show/hide columns.  


### **Active Filter Pop-Up**  
A white pop-up box (center) defining the current filter:  
- **Price**: Label for the filter type.  
- **>**: Dropdown (e.g., “Above,” “Below”) to set the price comparison operator.  
- **SMA**: Dropdown (e.g., “SMA,” “EMA”) to select the moving average type.  
- **10**: Dropdown to set the moving average period (e.g., 10 days).  
- **1 day**: Dropdown to set the time frame (e.g., 1 day, 5 days).  
- **Blue checkmark**: Button to apply the filter.  
- **X**: Button to clear the filter.  


### **Stock Table**  
A scrollable table listing stocks with columns:  
- **Symbol**: Ticker (e.g., “A,” “AA”).  
- **Price**: Current stock price (e.g., 139.57 USD for Agilent Technologies).  
- **Change %**: Percentage price change (e.g., +5.47% for Alcoa Corporation).  
- **Volume**: Trading volume (e.g., 9.067M for Alcoa).  
- **Rel Volume**: Relative volume (e.g., 1.04 for Alcoa).  
- **Market cap**: Market capitalization (e.g., 6.027B USD for Alcoa).  
- **P/E**: Price-to-Earnings ratio (e.g., 11.76 for Asia Broadband).  
- **EPS dil TTM**: Diluted earnings per share (trailing twelve months, e.g., 4.20 USD for Agilent).  
- **EPS dil growth TTM YoY**: EPS growth year-over-year (e.g., +0.48% for Agilent).  
- **Div yield % TTM**: Dividend yield (e.g., 0.64% for Agilent).  
- **Sector**: Industry sector (e.g., “Health Technology” for Agilent).  


### **Search Bar**  
Left of the table:  
- **Q Symbol**: Search box (with “8625” entered) to find stocks by ticker.  
- **+**: Button to add a new stock to the watchlist.  


### **Stock Entries (Examples)**  
Rows display individual stocks:  
- **Agilent Technologies (A)**: Price 139.57 USD, +0.48% EPS growth, 0.64% dividend yield, Health Technology sector.  
- **Alcoa Corporation (AA)**: Price 33.77 USD, +5.47% change, 6.027B USD market cap, Non-Energy Minerals sector.  
- **Asia Broadband (AABB)**: Price 0.0200 USD, +1.01% change, 49.379M USD market cap, Finance sector.  


### Purpose  
The interface allows users to **screen stocks** using customizable filters (price, P/E, sector, etc.), view real-time/filtered data, and switch between fundamental/technical analysis tabs (Overview, Performance, Financial Statements). The pop-up filter refines results by price relative to a moving average, while the table displays key metrics for comparison.
