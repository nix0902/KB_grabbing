# What if I made more than 5 pull requests in a day? 
 Only 5 pull requests (PR) per day can be merged from your personal repository into the main one. If you create more than 5 PRs, the * Upload data* action will not run until the next day. You will be able to rerun the action 24 hours after the first PR made the day before.

To rerun the action:

- Go to *Actions* in your repository.
- Open the last run action.
- *Click Re-run all jobs*.
