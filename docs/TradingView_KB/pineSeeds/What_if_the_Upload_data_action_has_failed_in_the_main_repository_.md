# What if the Upload data action has failed in the main repository? 
 Check that you have not exceeded the [daily number of pull requests](https://github.com/tradingview-pine-seeds/docs/blob/main/faq.md#q-what-if-i-made-more-than-5-pull-requests-in-a-day). If not, contact us at pine.seeds@tradingview.com. Our support team is available on weekdays from 7 AM to 4 PM UTC.
