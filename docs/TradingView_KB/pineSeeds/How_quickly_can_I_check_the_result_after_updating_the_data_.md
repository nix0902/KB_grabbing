# How quickly can I check the result after updating the data? 
 Your EOD data is checked and uploaded to our storage several times a day. On the chart, you will see the values for the current day and earlier. If the data needs to be updated more frequently, you can connect the integration via[ REST API](https://www.tradingview.com/brokerage-integration/), but this option is only available for brokerage integration.
