# Can only trading data be integrated? 
 The TradingView platform is a handy tool for dealing with [trading data](https://github.com/tradingview-pine-seeds/docs/blob/main/data.md#data-format). Such data always has a difference between price values. But if your data series has a single value per day, use open = close = high = low, and volume = 0. It is suitable for economic indicators.
