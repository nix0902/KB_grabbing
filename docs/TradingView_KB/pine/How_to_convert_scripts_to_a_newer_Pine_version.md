# How to convert scripts to a newer Pine version 
 Scripts written in every Pine Script® version starting from v3 can be converted to the next version automatically using the converter available in the “Manage Script” menu:

In some cases, the code cannot be converted fully and requires manual changes. To learn how to manually fix these version-specific changes if necessary, consult the [Migration Guides overview](https://www.tradingview.com/pine-script-docs/migration-guides/overview/) in our User Manual.

**Описание:**

This image shows a **TradingView Pine Script Editor interface** with a dropdown menu open, displaying script management options. Here’s a detailed breakdown of the UI elements:  


### 1. Top Navigation Tabs  
- **“The Editor”**: The active tab (highlighted in gray), indicating the current view is the script editor.  
- **“Strategy Tester”**: A tab for testing trading strategies (not active).  
- **“Trading Panel”**: A tab for executing trades (not active).  


### 2. Script Header & Dropdown Menu  
- **“My Script”**: The name of the current script (with a wave icon, typical of Pine Script). A **dropdown arrow** (▲) next to it opens the menu.  
- **“Update on chart”**: A button with a refresh icon, used to apply the script to a chart (e.g., to visualize indicators/strategies).  


### 3. Dropdown Menu Options (from top to bottom)  
Each option has an icon and text, with a keyboard shortcut (e.g., `⌘ S` for “Save script”) where applicable:  

- **“Save script”** (cloud icon): Saves the current script to your TradingView account.  
- **“Make a copy…”** (duplicate icon): Creates a copy of the script (useful for experimenting without altering the original).  
- **“Rename…”** (pencil icon): Renames the script.  
- **“Version history…”** (clock icon): Views past versions of the script (for reverting changes).  
- **“Convert code to v6…”** (highlighted with a red border): Converts the script to Pine Script v6 (TradingView’s newer syntax, for enhanced features).  
- **“Create new”** (plus icon with arrow): Opens a submenu to create a new script (e.g., indicator, strategy).  


### 4. “Recently Used” Section  
- **“My Script”** (black button): A shortcut to quickly access the most recently used script.  


### 5. Code Editor (Background)  
- Visible code lines (e.g., `1 // @version=...`, `2 indicator...`, `3 plot(cl...`) show the Pine Script code being edited, with syntax highlighting (blue for keywords like `indicator`, `plot`).  


### Purpose of the Interface  
This screen is part of TradingView’s **Pine Script Editor**, used to write, edit, and manage custom trading indicators/strategies. The dropdown menu provides essential actions for script management (saving, copying, renaming, versioning, converting to v6, and creating new scripts), while the top tabs switch between editing, strategy testing, and trading.
