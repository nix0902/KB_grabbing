- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Drawings - / [ How to use various drawing tools ](/support/folders/43000547459-how-to-use-various-drawing-tools/) - / [ Long Position ](/support/solutions/43000517002-long-position/) # Long Position Application The Long Position tool allows the user to set an entry point and assume a long position from that point. Extending above and below that price level will be two boxes: green for the profit zone and red for the loss zone. Both zones can be manually adjusted by the user to change the risk/reward ratio. Inputs Account Size Defines the total account balance used for risk and position size calculations. Lot Size Determines the minimal amount of traded instruments. Risk Defines the amount of risk per trade. Entry Price Allows for precise placement of trade's entry point. Leverage Defines the leverage applied to the position. Leverage affects margin requirements. Profit Level Defines the target level at which the trade will be closed with profit. Stop Level Defines the level at which the trade will be closed to limit losses. Ticks Allows you to set the profit target or stop-loss as a distance in ticks from the entry price. The system automatically converts the specified number of ticks into a profit price or stop price based on the instrument's tick size. Price Allows you to manually specify the exact price level at which the profit target or stop-loss should be placed. QTY precision Controls the rounding and display precision of the quantity value. Style Lines Can change the color and thickness of the line separating profit/loss areas as well as the profit and loss targets. Stop Color Sets the color for the stop-loss region. Target Color Sets the color for the profit target region. Text Can select the text color and font size. Price labels Toggles the visibility of the drawing's price values on the price axis. Stats Controls which data is displayed on the Risk/Reward tool. Compact stats mode Shortens the position performance tags so they take up less space on the chart. Always show stats If this checkbox is not selected, Stats will appear beside only when the Long Position is selected by mouse click. Visibility In the Visibility properties dialog, you can toggle displaying of the Long Position on charts of different timeframes. For any timeframe you can select either to show or to hide it. **** ** ** Previous Previous Inside Pitchfork Next Next Parallel Channel Launch Supercharts

**Описание:** This TradingView interface displays a **4-hour (4h) candlestick chart** for **Apple Inc. (NASDAQ: AAPL)**. Here’s a detailed breakdown of the UI elements:  


### **Top Toolbar**  
- **Left**:  
  - *Apple Inc.* (ticker symbol) with a dropdown for symbol selection.  
  - *4h* (timeframe) with a dropdown to switch timeframes (e.g., 1m, 1h, 1D).  
  - *Indicators* (dropdown to add technical indicators).  
  - *Alert* (set price alerts).  
  - *Replay* (replay market data for backtesting).  
  - Navigation arrows (previous/next time periods).  

- **Right**:  
  - *Shares Layout* (customize chart layout).  
  - *Trade* (place orders) and *Publish* (share chart).  
  - Currency selector (*USD*).  


### **Chart Area**  
- **Candlestick Chart**: Red/green candles show price action (red = bearish, green = bullish).  
- **Price Scale (Right)**: Vertical axis with price levels (e.g., 243.93, 257.69).  
- **Time Scale (Bottom)**: Horizontal axis with dates (e.g., 29, 30, 2026, 5, 6…).  
- **Highlighted Zones**: Two colored rectangles (teal = support, pink = resistance) indicate key price levels.  


### **Left Sidebar (Tools & Indicators)**  
- **Tool Icons** (top to bottom):  
  - Drawing tools (line, trendline, Fibonacci, text, etc.).  
  - *Projection* menu (expanded):  
    - *Long Position/Short Position*: Analyze trade setups.  
    - *Forecast/Bars Pattern/Ghost Feed/Projection*: Predictive tools.  
  - *Volume-Based* menu:  
    - *Anchored VWAP/Fixed Range Volume Profile/Anchored Volume Profile*: Volume analysis.  
  - *Measurer* menu:  
    - *Price Range/Date Range/Date and Price Range*: Measure price/time ranges.  


### **Bottom Toolbar**  
- **Timeframe Buttons**: 1D, 5D, 1M, 3M, 6M, YTD, 1Y, 5Y, *All* (switch chart duration).  
- *Trading Panel* (expandable for order entry).  
- Timestamp: *12:36:51 UTC* (real-time data).  


### **Right Sidebar**  
- *Pre* (previous close: 258.60) and current price (257.69, green = up).  
- Icons for notes, alerts, chat, and additional tools (e.g., chart style, calendar, notifications).  


### **Key Purpose**  
This interface is used for **technical analysis**—traders analyze price trends, add indicators, and plan trades (long/short) using the expanded *Projection* menu. The candlestick chart visualizes price movements over 4-hour intervals, with support/resistance zones (teal/pink) highlighting key levels.
