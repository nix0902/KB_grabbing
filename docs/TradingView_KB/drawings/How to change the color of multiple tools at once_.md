- [ Help Center ](/) - / - / [ How to use various drawing tools ](/support/folders/43000547459-how-to-use-various-drawing-tools/) - / [ How to change the color of multiple tools at once? ](/support/solutions/43000677803-how-to-change-the-color-of-multiple-tools-at-once/) # How to change the color of multiple tools at once? To apply color settings to several or all drawing tools, press Ctrl / Cmd and click on the necessary tools, or select the zone in which they are located, then select the desired color for text, lines or fill then on the floating toolbar - and it will be applied immediately to all selected tools: Simultaneous color change also works for tool groups - just click on the required group in the Object Tree and select the color again on the floating toolbar: Previous Previous How to apply a template to several tools of the same type at once?
