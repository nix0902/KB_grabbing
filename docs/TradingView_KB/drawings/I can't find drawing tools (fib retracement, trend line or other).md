- [ Help Center ](/) - / - / [ I can't find drawing tools (fib retracement, trend line or other) ](/support/solutions/43000473103-i-can-t-find-drawing-tools-fib-retracement-trend-line-or-other/) # I can't find drawing tools (fib retracement, trend line or other) All available drawing tools are located in the panel on the left hand side. They are divided into subgroups based on instrument type. For example, the 3rd group contains the Gann and Fibonacci drawing tools. If the panel on the left is not displayed then you might have hidden it. Click the profile icon in the top left corner and set to display it again.
