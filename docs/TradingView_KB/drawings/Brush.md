- [ Help Center ](/) - / [ Knowledge base ](/knowledge-base/) - / Drawings - / [ How to use various drawing tools ](/support/folders/43000547459-how-to-use-various-drawing-tools/) - / [ Brush ](/support/solutions/43000516987-brush/) # Brush Application Brush is used as a way to draw freely or annotate on the chart. The paint brush is not bound to a specific shape like most other drawing tools. Style In the Style property dialog, it is possible to change the appearance of a drawing line: Line Sets the color and thickness of a drawing line. Background Toggles the background visibility, color and opacity level for a painting. Visibility In the Visibility properties dialog, you can switch a Brush drawing displaying on charts of different timeframes: Previous Previous Bars Pattern Next Next Callout Launch Supercharts

**Описание:** This TradingView chart displays **Apple Inc. (NASDAQ: AAPL)** on a **1-day (1D)** timeframe, with a candlestick chart showing price action from late October 2024 to early February 2025. Here’s a detailed breakdown of UI elements:  


### **Top Bar (Header)**  
- **Symbol & Timeframe**: “Apple Inc · 1D · NASDAQ” confirms the stock and chart interval.  
- **Price Data**: `O229.22 H234.51 L229.02 C234.07 +4.04 (+1.76%)` shows open, high, low, close, and daily change.  
- **Timeframes**: Buttons (1s, 1m, 5m, 15m, 30m, 1h, 4h, D, W, M) let users switch intervals.  
- **Tools & Actions**: Icons for drawing (pen), indicators, chart type (candle/line), zoom, and “Publish” (share chart).  


### **Left Sidebar (Tools)**  
Icons for:  
- Drawing (pen, line, shape),  
- Indicators (e.g., moving averages),  
- Chart type (candle/line),  
- Zoom (magnifying glass),  
- Timeframe selection (1D, 5D, 1M, etc.),  
- Pine Editor (for custom scripts),  
- Strategy Tester, Replay Trading, Trading Panel.  


### **Main Chart Area**  
- **Candlestick Chart**: Green candles = price increase; red = decrease. A blue “Brush” annotation (with “UP” text) highlights a bullish trend.  
- **Y-Axis (Price)**: Ranges from ~217.50 to 260.00 USD, showing price levels.  


### **Right Sidebar (Brush Tool Panel)**  
A pop-up for customizing the blue brush annotation:  
- **Tabs**: “Style” (active) and “Visibility”.  
- **Line Settings**: Color (cyan), style (solid/dashed/dotted).  
- **Background**: Checkbox (enabled, with a checkered pattern).  
- **Buttons**: “Cancel” (discard changes), “Ok” (apply), “Template” (save/load style presets).  


### **Bottom Bar**  
- **Time Scale**: X-axis with dates (Nov, Dec, 2025, Feb) and a timeline slider.  
- **Timeframe Buttons**: 1D, 5D, 1M, 3M, 6M, YTD, 1Y, 5Y, “All” (zoom out).  
- **Timestamp**: “12:04:54 UTC” (current time).  
- **Status**: “ADJ” (adjusted data, e.g., for splits/dividends).  


### **Additional UI**  
- **Top Right**: “TradingView” (logo), “Save” (save chart), currency selector (USD), and a menu icon.  
- **Bottom Right**: Expand/collapse panels, help, and settings.  


This layout enables traders to analyze price trends, customize annotations, and switch timeframes/indicators—all core to technical analysis on TradingView.
