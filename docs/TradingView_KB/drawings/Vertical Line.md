If you enjoy using shortcuts, simply hover the point where you want the vertical line to appear and press Alt + V : Style In the Style property dialog, it is possible to change the appearance of a vertical line: Line Can change the color and opacity of the vertical line as well as its thickness and the line style. Extend Extends the vertical line through the Indicator panes. Time Label Toggles the visibility of the time value of the vertical line on the horizontal axis. Text In the Text property dialog, it is possible to change the text format. When the Text checkbox is active, a text may be entered in the text box to be displayed beside the vertical line. You can change the text color and font size. You can also choose to bold and/or italicize the text. Text Alignment and Text Orientation drop-downs allow to set the text position along the vertical line. You can edit the text of a vertical line directly on the chart – you just need to click on the text field, and you can type. Coordinates In the Coordinates properties dialog, you can set precisely the position of the Vertical Line on the time scale by setting the bar number: Visibility In the Visibility properties dialog, you can switch displaying of a Vertical Line drawing on charts of different timeframes: Alert It is possible to set an alert to get notified if the series crosses the vertical line on a chart. Use the clock icon in floating drawing toolbar or in right click menu on the drawing itself: Then select a condition, periodicity and preferable method of notification and press Create: Previous Previous Triangle Pattern Next Next XABCD Pattern Launch Supercharts

- **Left**: *Indicators* (dropdown), *Alert* (set price alerts), *Replay* (backtest market movements), and navigation arrows (undo/redo).

### **Left Sidebar (Drawing Tools)**  
A menu of technical analysis tools, with **“Vertical Line”** (selected, highlighted in black) and other line/channel tools:  
- **Lines**: Trend Line, Ray, Info Line, Extended Line, Trend Angle, Horizontal Line, Horizontal Ray, Cross Line.  
- **Channels**: Parallel Channel, Regression Trend, Flat Top/Bottom, Disjoint Channel.  
- **Pitchforks**: Pitchfork, Schiff Pitchfork, Modified Schiff Pitchfork, Inside Pitchfork.  


### **Chart Area**  
- **Price Data**: Top of the chart shows real-time data: *O 11.69* (open), *H 11.78* (high), *L 11.62* (low), *C 11.74* (close), +0.08 (+0.69%) change.

- **Vertical Lines**: Two blue vertical lines (labeled *Wed 01 May ’24* and *Mon 01 Jul ’24*) mark a two-month sideways trading range, with a tooltip explaining: *“Vertical Lines used to highlight a two-month period of relatively sideways trading before an advance.”*  
- **Y-Axis (Right)**: Price scale (10.50–15.00 USD).  
- **X-Axis (Bottom)**: Time (Mar, Apr, May, Jun, Jul 2024).  


### **Bottom Toolbar**  
- *All* (filter data), calendar icon (date range), and timestamp (*04:09:47 UTC-4*).  


### **Right Sidebar**  
Icons for:  
- Chart notes, alerts, patterns, chat, and additional tools (e.g., Fibonacci, drawing, calendar, grid).  


### **Purpose**  
The chart analyzes price trends, using vertical lines to isolate a sideways trading period before a potential upward move. Drawing tools (left) enable technical analysis (e.g., trend lines, channels) to identify patterns. The interface supports real-time data, alerts, and sharing (via *Publish*).
