What this means is that the anchored text box will stay visible on the chart and doesn't scroll with a series, no matter how far into the past or future you scroll the chart: To add anchored text, place it on the chart and enter text for it. As with all text tools, you can edit the anchored text directly on the chart – you just need to click on the text field, and you can type. Text All options available for anchored text appearance may be set in the Text properties dialog: General Options Allows you to change the text color and font size. You can also choose to bold and/or italicize the text. Text Box Where the text is typed. Background Toggles the visibility, color and opacity of a background for the text box. Border Toggles the visibility of a border for the text box. Can also change the border color. Text Wrap Enables/Disables text wrap within the text box. Visibility In the Visibility properties dialog, you can switch displaying of anchored text for charts of different timeframes. **** ** ** Previous Previous ABCD Pattern Next Next Arc Launch Supercharts

**Описание:** This TradingView chart displays **Apple Inc. (NASDAQ: AAPL)** on a **1-day (1D)** timeframe, showing a candlestick chart with price action from July to early September. Here’s a detailed breakdown of the UI elements:  


### **Top Bar (Header)**  
- **Symbol & Timeframe**: “AAPL Apple Inc · 1D · NASDAQ” confirms the stock and chart period.  
- **Price Data**: “O239.30 H240.15 L238.74 C238.89 −0.80 (−0.33%)” shows open, high, low, close prices, and daily change.  
- **Timeframe Buttons**: “1s 1m 5m 15m 30m 1h 4h D W M” (seconds to monthly) for switching chart periods.  
- **Chart Type/Tools**: Icons for chart styles (e.g., candlestick, line), indicators, and drawing tools.  
- **Navigation/Save**: Arrows (back/forward), “TradingView Save” (save chart), and “Publish” (share).  


### **Left Sidebar (Tools)**  
Icons for core functions:  
- Zoom (magnifying glass), add indicators, drawing tools (lines, shapes), text, Fibonacci, and account/settings.  


### **Main Chart Area**  
- **Candlestick Chart**: Green/red candles represent price action (green = close > open; red = close < open).  
- **X-Axis (Time)**: Dates (Jul, Aug, Sep) mark the timeline.  
- **Y-Axis (Price)**: Values (197.50–245.00) show price levels.  


### **“Anchored Text” Popup (Text Annotation Tool)**  
A modal for adding/editing text on the chart:  
- **Title**: “Anchored Text” (editable via pencil icon).  
- **Tabs**: “Text” (active, for editing) and “Visibility” (control when text appears).  
- **Formatting**:  
  - Color picker (blue), font size (24), bold/italic buttons.  
  - Text box (labeled “Text”) for input.  
- **Styling Options**:  
  - “Background” (checked, with a pattern selector).  
  - “Border” (unchecked, with a color selector).  
  - “Text wrap” (unchecked).  
- **Buttons**: “Template” (save/load text styles), “Cancel” (discard), “Ok” (apply text).  


### **Bottom Bar**  
- **Timeframe Shortcuts**: “1D 5D 1M 3M 6M YTD 1Y 5Y All” for quick period changes.  
- **Timestamp**: “10:51:29 UTC-4” (current time).  
- **Tabs**: “Pine Editor” (code indicators), “Strategy Tester”, “Replay Trading”, “Trading Panel”.  


### **Right Sidebar**  
Icons for additional features:  
- Chart style, time settings, alerts, and a grid of tools (e.g., patterns, calendar).  


### **Purpose**  
This interface lets traders analyze AAPL’s price, add annotations (via the “Anchored Text” tool), switch timeframes, and access advanced tools (indicators, strategy testing) for technical analysis. The text popup customizes how annotations appear (color, background, font) to highlight key insights.
