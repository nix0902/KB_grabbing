**Описание:** This TradingView image displays a **candlestick chart** (green/red bars) with a **context menu** open on the right, showing options for a selected trading object. Here’s a detailed breakdown:  


### **Chart Elements**  
- **Candlestick Chart**: Green (bullish) and red (bearish) bars represent price action over time.  
- **Highlighted Zones**:  
  - *Green zone (top)*: Target area (price level for profit).  
  - *Red zone (bottom)*: Stop-loss area (price level to limit losses).  
- **Annotations**:  
  - *Green box*: “Target: 6,586 (7.13%) 6,586, Amount: 1137.84” (profit target details).  
  - *Red box*: “Stop: 11,945 (12.93%) 11,945, Amount: 750” (stop-loss details).  
  - *Teal box*: “Closed P&L: 6,586, Qty: 0.021, Risk/Reward Ratio: 0.55” (closed profit/loss, quantity, and risk/reward).  
- **Blue Circles**: Control points (handles) for resizing/moving the selected trading object (e.g., a trendline or order).  


### **Context Menu (Right Side)**  
A dropdown menu with options for managing the selected object:  
- **Template**: Apply a saved chart template.  
- **Visual order**: Adjust object layering (front/back).  
- **Visibility on intervals**: Show/hide the object on specific timeframes.  
- **Object Tree…**: View all chart objects in a hierarchy.  
- **Create limit order…**: Place a limit order based on the object’s levels.  
- **Clone**: Duplicate the object (shortcut: `⌘ Drag`).  
- **Copy**: Copy the object (shortcut: `⌘ C`).  
- **Sync Options**:  
  - *No sync* (checked): Object not synced across layouts.  
  - *Sync in layout*: Sync within the current layout.  
  - *Sync globally*: Sync across all layouts.  
- **Reverse**: Flip the object’s direction (e.g., reverse a trendline).  
- **Lock**: Prevent accidental edits.  
- **Hide**: Temporarily hide the object.  
- **Remove**: Delete the object (shortcut: `⌫`).  
- **Settings…**: Customize the object’s properties.  


### **Purpose**  
The chart visualizes a trading strategy (target, stop-loss, P&L) with interactive controls. The context menu allows users to modify, duplicate, or manage the selected trading object (e.g., a trendline, order, or indicator) for analysis or execution.
