For example, user can place Flag Marks at high/low price points of a trend: Style Flag mark has blue color by default and fixed size. In the Style property dialog, it is possible to change the color of a flag mark: Coordinates In the Coordinates properties dialog, you can set precisely the flag mark's position on the price scale (by setting the price) and the time axis (by setting the bar number): Visibility ** ** In the Visibility properties dialog, you can switch displaying of a flag mark drawing on charts of different timeframes: Previous Previous Fib Wedge Next Next Forecast Launch Supercharts
