The possibility to add text is available in all range tools: Date range, Price range, and Price and time range. To add text do the following: - Add the measurement object you need to the chart; - Open the settings for this object; - Go to the "Text" tab; - Enter text and click OK. The text entered in the settings will be displayed on the object on the chart. Previous Previous How to change the color of multiple tools at once?
