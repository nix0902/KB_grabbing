# I want to customize the visibility for several objects

- [ Help Center ](/) - / - / [ I want to customize the visibility for several objects ](/support/solutions/43000660419-i-want-to-customize-the-visibility-for-several-objects/) # I want to customize the visibility for several objects You can customize the visibility for several objects on specific intervals as follows: - Open the Object tree, place drawing objects into a group, and select “Settings” in the group’s drop-down menu. - Press Ctrl on Windows or Cmd on Mac, select several objects or groups in the Object tree with your mouse, open the drop-down menu and select “Settings”. - Press Ctrl on Windows or Cmd on Mac, select several objects right on the chart with your mouse, open the drop-down menu and select “Settings”. - Select several objects or groups and click the “Settings” button in the floating toolbar. - Select several objects on the chart or in the Object tree, open the drop-down menu and select “Visibility on intervals” to set the visibility of the objects in relation to the current interval.
