# How to Create API Keys - Binance Spot, Futures and Coin M
Create your API Keys on Binance and connect them with your Cornix account
Written by Hadar Cornix Updated over a week agoTable of contents
Creating Binance API - Step by StepStep 1 - Enter the API Management sectionStep 2 - Click the Create API buttonStep 3 - Choose API Key TypeStep 4 - Choose a name for Your API Step 5 - Security verificationStep 6 - API permissionsImportant Notes
Before you start, please make sure you complete your Binance account verification. Binance allows API key creation only for accounts that have completed their Basic and Intermediate Verification.
If you haven't completed both of your account's Basic and Intermediate verification procedures, kindly go back to Binance and complete it.
You can follow this link to learn how to verify your account.
Once your account is verified, you will be able to complete the steps.

Quick Connect:
If supported, we highly recommend using the ⚡Binance Quick Connect⚡ feature, for fast, easy and seamless process!
​
⚡Use Binance Quick Connect⚡


# Creating Binance API - Step by Step
Navigate to your Cornix Dashboard > Accounts > '+ Create'



Choose 'Binance' and then 'Connect with API Keys'

Now, navigate to your Binance account:
-
Click on the account menu in the upper right corner
-
Click 'API Management' and then 'Create API'
-
Choose the API keys type and enter a label
-
Go through the security verification
-
Enable 'Reading' permissions
-
Enable 'Spot & Margin' permissions.
*If you're creating a Binance Futures / Binance Coin-M accounts, please enable 'Futures' permissions as well.
-
Choose Restrict access to trusted IPs only
-
Add Cornix's IP addresses to the Binance list of trusted IPs
-
Copy & Paste the API Key and Secret key
## Step 1 - Enter the API Management section
Go to your account settings to get to the API Management section.


## Step 2 - Click the Create API button
Click on the Create API button. Depending on if you already have other APIs created, the input box and the button might appear in different parts of the screen.


## Step 3 - Choose API Key Type
You will have the option to choose Binance system generate your API keys - this will be the popular choice). If you know how to create your own API keys via software then this is an additional option you can choose.


## Step 4 - Choose a name for Your API
Set a name as seen in the picture and click 'Next'.


## Step 5 - Security verification
At this stage Binance will require to complete a few verification steps. It can be Email, phone number, authentication app, etc.
Once you complete the verifications, your API Key will be created.

## Step 6 - API permissions
Before sharing the API Keys to Cornix, you will need to make sure that the API restrictions match the image below. Specifically, make sure that the API permissions include:
-
Enable Reading
-
Enable Futures
-
Restricted access to trusted IPs only
​Note: You must add Cornix's IP addresses to the list of trusted IPs, to enhance the safety of your API keys and ensure they will not expire after a few months. Your unique list of IP addresses will be shown on the create account page.

If you would like to learn how to insert the API keys to Cornix, you can follow the links below:

## Important Notes
-
Binance Portfolio Margin accounts and Cross Margin accounts are not supported by Cornix, and therefore can't be connected.
