# How to Create an Account (Client)
How to link your exchange to Cornix through your web dashboard
Written by Hadar Cornix Updated over 8 months agoTable of contents
How to Create an Account Through Cornix Web DashboardQuick ConnectManual API CreationCreate an Account Through Cornix Telegram Bot
In order to trade automatically on your behalf, Cornix requires access to your exchange account.
This API based connection between Cornix to your exchange is called an account (Client).

Note: this guide assumes that you already have a basic understanding of what clients and API keys are. If not, please take the time to review guides about what is an account (client) and what is an API Key.

# How to Create an Account Through Cornix Web Dashboard
Navigate to your "Accounts" page on the left sidebar.
Click "+Create" / "Create Account" to start creating new connections.
​


Proceed to choosing your desired exchange from the list:


## Quick Connect
The "Quick Connect" option will create the exchange accounts automatically, creating encrypted and IP-restricted API keys.
✅ This is the most recommended way to connect your exchange!

Click here for specific guide to the exchange that you wish to Quick Connect


## Manual API Creation
If the quick connect option is not available for you, you can connect your exchange manually, by choosing "Connect with API Keys"

*Use this link to find the specific guide for the exchange that you wish to create APIs for

Proceed to create your API Keys according to the instructions on the left side of the screen (or the articles linked above).

The process of generating API keys may differ between exchanges and sub-exchanges (including differences between Spot and Futures).

Once you click 'Connect' , a pop-up will appear, mentioning the accounts were created successfully.

Now your accounts are ready to start trading
Visit this page to learn about Cornix Smart Trading Bots and how to set them

# Create an Account Through Cornix Telegram Bot
Creating an account through the Cornix Telegram Bot is only available for some legacy Cornix users. If you are one of those, and you wish to create an account (client) using the Cornix Telegram bot interface, please visit the following link for a step by step guide.
