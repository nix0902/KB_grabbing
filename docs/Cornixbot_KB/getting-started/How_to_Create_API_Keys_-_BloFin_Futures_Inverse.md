# How to Create API Keys - BloFin Futures & Inverse
Create your API Keys on BloFin and connect them with your Cornix account
Written by Hadar Cornix Updated over 3 months agoTable of contents
How to create your BloFin APIStep 1 - Go to the API sectionStep 2 - Click '+Create API Key'Step 3 - API Permissions and SettingsConnect your BloFin API to CornixStep 4 - Copy Your Keys to CornixStep 5 - Create Account in the Cornix Dashboard
### Notes
-
BloFin Exchange is currently supported for Futures and Inverse (USD) trading. Spot trading will be supported in the future.
-
Cornix supports connecting both main and sub accounts.

# How to create your BloFin API
## Step 1 - Go to the API section
Navigate to your BloFin Dashboard > Click your account > API


## Step 2 - Click '+Create API Key'
## Step 3 - API Permissions and Settings
-
Check "Connect to Third-Party Application" and choose "Cornix"
-
Set your API Key Name
-
Permissions: Check "Read" and "Trade"
-
Set your Passphrase (*make sure to remember it for the next steps)
-
Click 'Next'
-
Complete the following verification step with a verification code sent to your email address.
# Connect your BloFin API to Cornix
## Step 4 - Copy Your Keys to Cornix
After a verification step, your API Key and Secret Key will become visible. Copy and paste them to your Cornix Create Account page (elaborated in the next step).


## Step 5 - Create Account in the Cornix Dashboard
-
Navigate to your Cornix Dashboard > Accounts > '+Create Account' > BloFin (or click this link instead)
-
Choose the sub-exchanges you would like to connect (Futures/Inverse/Both)
-
Choose a name for your BloFin account
-
Past the API Key, Secret and Passphrase
-
Click 'Connect to BloFin'.

Once the connection is done, you'll get a pop up notification at the top of your screen mentioning the account was created successfully.
You can now start trading with your BloFin account.
