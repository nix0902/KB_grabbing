# Connect your Bitget Account (Create API Keys) - Quick Connect ⚡
How to automatically create encrypted and IP-restricted API keys when connecting to Bitget
Written by Hadar Cornix Updated over 2 months agoTable of contents
Step 1 - Navigate to your Web DashboardStep 2 - Select the Bitget Quick Connect Option and Click Connect to BitgetStep 3 - Security Verification
Note: Bitget Unified Accounts are not supported. Instead, use Classic Account Mode.

# Step 1 - Navigate to your Web Dashboard
Select 'Accounts & Groups' and tap on 'Create Account'.


# Step 2 - Select the Bitget Quick Connect Option and Click Connect to Bitget
Once you select your choice, tap on 'Connect To Bitget'.

Then click "Next":


# Step 3 - Security Verification
After completing the security verification, Cornix will automatically create two exchange account types, for Spot and Futures (USDT):

And you're all set!
