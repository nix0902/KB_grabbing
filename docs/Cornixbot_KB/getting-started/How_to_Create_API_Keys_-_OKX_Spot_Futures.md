# How to Create API Keys - OKX Spot & Futures
Create your API Keys on OKX and connect them with your Cornix account
Written by Hadar Cornix Updated over 8 months agoTable of contents
Step 1 - Enter the API Keys sectionStep 2 - Click on 'Create V5 API key'Step 3 - Choose the 'Linking third-party apps'Step 4 - Copy your API Keys and insert them into Cornix.
## Step 1 - Enter the API Keys section
## Step 2 - Click on 'Create V5 API key'
## Step 3 - Choose the 'Linking third-party apps'
1. Choose Cornix from the drop-down list.
2. Set a name for your API.
3. Choose your Passphrase.
4. Check ✔ next to 'Trade' in the Permissions section.
5. Click 'Confirm' and go through the verification process, after which you will be provided with your API keys.
## Step 4 - Copy your API Keys and insert them into Cornix.
Please make sure you copy your API Key, Secret Key, and Passphrase to create your client on Cornix.

That's it,
you're all set!
