# What Is a Cornix Account (Client)?
  
Creating a Cornix Account (Client) connects your exchange account to Cornix via secure API keys. This setup is the foundation of auto-trading, monitoring, and managing trades, all from one dashboard.

This connection between Cornix to your exchange account is what we call "an  Account  ( Client)" .

#  How It Works 
  
Without linking an account, Cornix can’t execute trades or fetch your portfolio. It’s the key that unlocks full functionality, including:
   -  
Viewing your portfolio on all your connected exchange accounts.
 
-  
Place orders in all your exchanges from one place.
 
-  
Execute trades automatically via bot or channel signals
 
-  
Set up complex DCA or grid trades that run 24/7
 
-  
Etc.

#  Next Step: How To Set It Up
