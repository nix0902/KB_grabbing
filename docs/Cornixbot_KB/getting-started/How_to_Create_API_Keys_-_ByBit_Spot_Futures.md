# How to Create API Keys - ByBit Spot & Futures
Create your API Keys on ByBit and connect them with your Cornix account
Written by Hadar Cornix Updated over 8 months agoTable of contents
ByBit UTA (Unified Trading Account) additional permissions required:Step 1 - Navigate to your Web DashboardStep 2 - Select the Exchange that you want to connectStep 3 - Choose the sub-exchanges you would like to connectStep 4 - Connect with API Keys Click 'Create New Key'Choose 'Connect to Third-Party Applications' and select 'Cornix'Set the API PermissionsSecurity VerificationCopy your keys to Cornix
Before we start,
If supported, we highly recommend using the ⚡ByBit Quick Connect⚡ feature, for fast, easy and seamless process!

⚡Use ByBit Quick Connect⚡
ByBit UTA Note:
Connecting a ByBit UTA (Unified Trading Account) will require additional permissions as detailed here:

## ByBit UTA (Unified Trading Account) additional permissions required:

## Step 1 - Navigate to your Web Dashboard
Under 'Portfolio' select 'Accounts' and tap on '+Create'.


## Step 2 - Select the Exchange that you want to connect
Click 'ByBit'


## Step 3 - Choose the sub-exchanges you would like to connect
Note: By default, all sub exchanges types are checked (ByBit Spot, ByBit USDT and ByBit Inverse). Uncheck the ones that you do not want to connect.
In case a sub exchange type you selected does not exist in your account yet, we will direct you to ByBit in order to create one.

## Step 4 - Connect with API Keys
Switch to 'Connect with API Keys' to manually connect your APIs, and follow the instructions on the left side of the page


1. Login to ByBit
2. Click on the user icon in the upper right corner
3. Click 'API' and then 'Create New Key'
4. Choose the 'Connect to Third-Party Applications'
5. select 'Cornix' in the Application name dropdown
6. Select the 'Read-Write' permissions
7. Select 'Trade' under SPOT and 'Orders' and 'Positions' under Contracts
8. Click 'Submit' and Enter your 2FA code
9. Copy & Paste the API Key and Secret key

##
Click 'Create New Key'
Under the API management, click 'Create New Key' and choose 'System-generated API Keys'.

## Choose 'Connect to Third-Party Applications' and select 'Cornix'
Choose the 'Connect to Third-Party Applications' option and use the dropdown to select 'Cornix'.


##
Set the API Permissions
Proceed to set up the API connection permissions under the 'API permissions (check all that apply)' section.
-
Select the 'Read-Write' option
-
Under 'Contract' choose 'Orders' and 'Positions'
-
Under 'Spot' choose 'Trade'
-
Click 'Submit'

##
Security Verification
You will be requested to pass a Security Verification:


##
Copy your keys to Cornix
Copy the 'API Key' and 'Secret Key' to your Cornix bot, by clicking the copy button next to the keys.
​
After making sure everything is set properly, click the "Understood" button.
​

That's it,
​You are all set!
