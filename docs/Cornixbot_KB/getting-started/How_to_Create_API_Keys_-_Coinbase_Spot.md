# How to Create API Keys - Coinbase Spot
Create your API Keys on Coinbase Spot and connect them with your Cornix account
Written by Hadar Cornix Updated over 8 months agoTable of contents
Step 1 - Turn On 'Advanced' Settings on Your Coinbase AccountStep 2 - Go to Your 'API' Section on the Menu Step 3 - Click 'Create API Key' Step 4 - Set your API KeyStep 5 - Copy your API Keys to Cornix
In order to link your Coinbase account to Cornix, you'll have to create an API Key on your Coinbase account, and insert this key to your Cornix account.
Here are the steps you should follow:

## Step 1 - Turn On 'Advanced' Settings on Your Coinbase Account
Search for the 'Advances' toggel on the left side of the screen, and turn it on.
​

## Step 2 - Go to Your 'API' Section on the Menu

## Step 3 - Click 'Create API Key'

## Step 4 - Set your API Key
-
Choose a name for your API Key.
-
Choose the Portfolio for your API Key if you have any specific portfolio you wish to link to your API. If not, keep it as 'Default'.
-
Allow 'View' and 'Trade' permissions.
-
Skip the 'IP Whitelist' section.
-
Click 'Create & download' to see your keys.


## Step 5 - Copy your API Keys to Cornix
Copy and paste the API key name and Private key to your Cornix account.
​Note: Make sure to copy it entirely, do not subtract any part of the keys.


This is it!
Once you're done creating your Keys, click 'I've saved my key' and you'll get a pop up message at the buttom of the screen, saying 'API Key succesfully created'.
You can now go back to your Cornix account and start trading!
