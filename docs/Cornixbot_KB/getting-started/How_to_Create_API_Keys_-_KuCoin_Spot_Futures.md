# How to Create API Keys - KuCoin Spot & Futures
Create your API Keys on KuCoin and connect them with your Cornix account
Written by Hadar Cornix Updated over 8 months agoTable of contents
Step 1 - Enter the API Management sectionStep 2 - Click on 'Create API'Step 3 - Switch to the 'Link Third Party Applications' tabStep 4 - Copy your API Keys and insert them into Cornix.
## Step 1 - Enter the API Management section

## Step 2 - Click on 'Create API'

##
Step 3 - Switch to the 'Link Third Party Applications' tab
1. Choose Cornix from the drop-down list.
2. Provide a name for your API.
3. Choose your API Passphrase (The passphrase should be the unique password for the API key. It is not the same as your account's password or Trading password.)
4. Check ✔ for 'Future Trading' & 'Spot Trading' in the Permissions section.
5. Click 'Next' and go through the verification process, after which you will be provided with your API keys.



## Step 4 - Copy your API Keys and insert them into Cornix.
Make sure you copy your API Key, Secret Key, and API Passphrase to create your KuCoin account on Cornix.



That's it, you're all set :)
