# Connect your OKX Account (Create API Keys) - Quick Connect ⚡
How to automatically create encrypted and IP-restricted API keys when connecting to OKX
Written by Hadar Cornix Updated over 8 months agoTable of contents
Step 1 - Navigate to your Web DashboardStep 2 - Select the OKX Quick Connect Option and Click Connect to OKX
To enable API connection with Cornix, ensure your OKX exchange account is configured for "Single-Currency Margin" mode. If it's not set up this way, please refer to this article for instructions on how to change your account mode.

# Step 1 - Navigate to your Web Dashboard
Select 'Accounts & Groups' and tap on 'Create Account'.

# Step 2 - Select the OKX Quick Connect Option and Click Connect to OKX
Once you select your choice, tap on 'Connect To OKX'.
Then click "Approve":

Cornix will automatically create three exchange account types, for Spot and Futures (USDT) and Inverse:

And you're all set!
