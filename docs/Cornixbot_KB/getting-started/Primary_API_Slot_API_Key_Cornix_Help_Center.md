# 🔌 How Do API Slots Work?
  
When connecting an exchange to Cornix, each unique API Key is counted as  1 Primary API Slot .
  
You can connect  more than one account  using a  single API Key  if you select multiple account types (e.g., Spot, Futures, etc.) during the Exchange connection process.

Each Cornix subscription plan includes a different quota of Primary API Slots. You can review the details on our ' Plans & Pricing ' page.

To check your current usage, visit your ' Billing ' page and look for the  Primary API Slots  under the  Current Usage  section. 

# ➕ Purchasing Additional API Slots
  
If you've reached your plan's API Slot limit and wish to connect more exchange APIs, you can purchase additional slots at any time.
   -  
 Price : $200 per year (per additional slot)
 
-  
 How to purchase :
   -  
Go to the  Billing  page
 
-  
Click   Purchase   next to "Primary API Slots"
 
-  
Select your payment method and complete the checkout

## ⚠️ Important Notes
   -  
Additional API Slots are available as  annual purchases only , and cannot be purchased on a monthly basis.
 
-  
Promo codes cannot be used for purchasing API Slots.
