# 🔹 What is Cornix?
  
Cornix is a trading automation platform that executes trades on your behalf. You can:
   -  
 Auto-trade signals  from channels (via Telegram/ Discord)
 
-  
 Use Smart Trading Bots  (Trading View, DCA and Grid Bots)
 
-  
 Manually open trades  using Telegram, Web Dashboard or the Mobile app
