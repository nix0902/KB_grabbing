# What's a Promo Code ?
Promo codes offer a discount on Cornix subscription plans and are available through select sources. You can apply them during checkout to get the reduced price for your purchase.

⚠️ Note: Promo codes and any kind of discount will only apply on Annual Subscription Plans. Monthly subscriptions are not eligible for a discount.


# How Do I Get a Promo Code?
## Signal Groups Promo Codes
If you’re a member of a signal group that's integrated with Cornix, ask the channel admins if they offer a promo code. Admins who are part of our affiliation program can provide their own promo codes to their members, offering an additional discount.
​
## Special Cornix Promotions
Cornix occasionally runs limited-time promotions.
💡 Make sure to stay updated! new promotions will always be announced via:
-
Our Website
-
The Cornix Mobile App
-
Our Telegram Announcements Channel
-
Personalized Emails
Note:
Cornix currently offers 33% off for all annual subscription plans, no code is needed.

## New Cornix User? Special Offer 40% OFF 🎉
Cornix is offering a 40% discount for all new users on our annual plans.
To apply the discount, enter the promo code ​FIRSTTIME40 at the checkout.
-
The offer can only be used once, and on the account's first payment.
-
The discounted price will appear automatically once the code is applied.
-
The discount is reduces from the full price of the selected plan.

# How to Apply a Promo Code
You can enter a promo code at the checkout, after selecting your plan and duration, and before completing your payment. If the code is valid, the discount will be applied automatically to your subscription plan and you'll see it on the payment preview.
​
You can find the 'Add Promo Code' button right above the 'Pay Now' option when paying with PayPal/ Credit Card.
​
If you select 'Crypto' it will be right above the network selection drop-down list.
​

# Do Promo Codes Have an Expiration Date?
Yes. Cornix Promo Codes usually expires after 1 week of promotion. If your promo code was given by a channel admin (signal provider), and is not part of a Cornix special promotion, it will remain valid as long as you're still a member of this channel.


# Promo Codes & Referral Links – What You Need to Know
If you have both a referral link and a promo code, here’s how it works:
-
The referral link should always be used first.
This ensures the referring person receives their commission and it also includes the basic referral discount automatically.
-
If you have a special promo code as well, make sure to use it at the checkout.
-
If later on, you decide to use a promo code from someone else, you can still use it, however, the referral commission will still go to the first person who referred you to Cornix, and not the one who provided the new promo code.
