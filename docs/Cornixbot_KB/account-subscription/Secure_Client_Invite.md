# Secure Client Invite
A secure and seamless way for asset managers to manage user accounts without exposing API keys
Written by Hadar Cornix Updated over a week agoTable of contents
🔗 How Does It Work?🛒 API Slots & Pricing🧭 How to Invite Users📋 The "Manage Invites" Page Explained⚠️ Important Notes
This feature Secure Client Invite gives you the ability to manage users’ trades without requiring them to publicly share their API keys. It’s a seamless and secure way to connect your users exchange accounts to your own Cornix account, allowing for smarter trade management and deeper engagement.


## 🔗 How Does It Work?
With Secure Client Invites, you can invite users to connect their exchange account to your Cornix account via a private invite link. Once connected, you’ll have access to trade on their behalf and view account stats, without users exposing their API credentials publicly.

## 🛒 API Slots & Pricing
To start inviting users, you'll need to have some Primary API Slots available on your account. If you do not have them, you'll need to purchase additional API Slots:
-
Each API Slot costs $200 per year.
-
A single slot covers one user connection.
-
A user can connect multiple sub-exchanges (e.g., Binance Spot, Futures, and Coin-M) using a single invite link - as long as they connect them at the same time when using the link.
-
Slots are valid for 12 months and can be reused if a user is disconnected.
## 🧭 How to Invite Users
Here’s how to create your invitations:
-
Go to User Interface > Accounts > Create > 'Secure Client Invite' at the bottom right.
-
Click on Manage Invites > Create Invite
-
Give your invitation a clear name, like ‘Daniel’. You can always rename it later.
-
Set the Asset Manager Access and the Client Access:
-
Asset Manager Access - the permissions you will have for this client's account as the asset manager:
-
Read Only - you'll only be able to view the client's account, monitor its performance and generate reports and invoices.
-
Read & Write - you'll have full access to create and manage trades for this client.
-
Client Access - the permissions your client will have to Cornix:
-
No Access - your client will have no access to the Cornix dashboard, and you'll be the only one who can see and manage their account.
-
Read Only - your client will be directed to register to Cornix, and will have a read only access to their account. They will not be able to create trades or perform any action on the account.
-
Read & Write - your client will be directed to register to Cornix, and will have access to create and manage their trades.
-
Click Create to generate your unique invitation link.
-
Copy and share the link with your user.





### What's next?
When your user opens the link, they’ll see your Full name and Telegram username to help them verify the request.

They will create their accounts following the basic Create Account flow, choosing the Exchange, Sub-exchanges, Setting their APIs etc.

Note: you can define which exchanges will appear, and how your Asset Manager profile will appear in the system and to your clients, through the Asset Manager Settings page.
Read more about how to set your Asset Manager profile.
​
### The user's view once clicking your link 📷



Once the user connects their API keys:
-
A new account will appear on your 'Accounts' page.
-
You can start trading on their behalf.
-
You gain visibility into their portfolio and account statistics.
-
They can track their account’s performance from their exchange.

## 📋 The "Manage Invites" Page Explained
-
Account Name: The custom name you gave to the invite (must be unique)
-
Creation: The timestamp when the invite was generated
-
Expiration Date: Invite links expire after 24 hours
If the link expires, simply delete it and generate a new one.
-
Status: Indicates the current state of the invite:
Active - The link is still valid and waiting to be used
Expired - The link is no longer valid
Used - The link has been used by the client
-
Client Access: shows the permissions your client will have with this invite link.
-
Manager Access: shown the permissions you will have for this client.

## ⚠️ Important Notes
-
This feature is only available for live trading, and not on demo accounts.
-
If all API Slots are used, users trying to connect will receive an alert mentioning they should contact the asset manager. To resolve this, purchase additional API Slots and share a new invite link.

Go to 'Create Account' page and start managing invites
