# Assets Manager Subscription Plan
Overview of the Cornix Assets Manager subscription plan, pricing and key features
Written by Hadar Cornix Updated over 3 months agoTable of contents
Plan DetailsGeneralHigh CapacityExtensive BacktestingSecure ConnectDiscounted Annual OptionHow to SubscribeHow to Manage The Accounts? Dedicated Trading Channel (Telegram / Discord)Bulk Edit BotsBulk Close TradesFAQs
The Assets Manager plan is a new Cornix subscription plan that provides expanded capacity and advanced usage quotas for asset managers who need to manage accounts and strategies in high scales. This plan includes increased API slots, higher backtesting limits, and a secured API connection featured.

# Plan Details
## General
-
Monthly price: $500
-
Annual price: $4,000 (includes a 33% discount)
-
Primary API Slots: 200
-
Backtesting Runs: 1,000 backtests per month
-
Secure Connect: Invite external users to securely connect their Exchange APIs to your assets manager.
➤ Learn more about Secure Client Invite
## High Capacity
With 200 Primary API Slots, you can manage a large number of exchange accounts and strategies simultaneously, making it ideal for asset managers or power users.

## Extensive Backtesting
Run up to 1,000 backtests per month to fine-tune your strategies and make data-driven trading decisions.

## Secure Connect
Easily collaborate with your clients using Secure Client Invites. This feature lets you securely grant access while maintaining full control over your trading setup.
➤ Visit this page to learn more about the Secure Client Invite

## Discounted Annual Option
Choose the annual plan for $4,000 and save 33% compared to monthly billing.

## How to Subscribe
You'll find the Assets Manager Plan on your Cornix Web Dashboard under Billing > View Plans > Asset Manager, or visit our plans page to explore all available options.


# How to Manage The Accounts?
Check out this demo to get a full overview of what the Asset Manager account includes:
In order to simplify the accounts management, and efficiently handle a large number of accounts and strategies at scale, several tools are available:

## Dedicated Trading Channel (Telegram / Discord)
Create a dedicated trading channel on Telegram or Discord to share signals, edit or update them, and set basic trading configurations. This ensures that all accounts follow the same signals and trade configurations in real time.

## Bulk Edit Bots
When using DCA, TV or Grid bots, you can edit configurations for multiple bots in a single action. Bulk editing allows you to keep everything aligned and consistent in one click.
Note: At this point, Bulk Edits are not available for Signals Bots.
​
## Bulk Close Trades
Manage trades more effectively by closing multiple trades at once across multiple accounts. This tool helps you react quickly to market events and protect client funds in fast-moving conditions.

# FAQs
### What if I need more than 200 API Slots?
If more than 200 APIs are required, please reach out to the Cornix team through one of our support channels - in the app, on the web, or via Telegram.
