# Bulk Actions on Bots
How to Bulk Deactivate, Activate, Edit and Delete your bots as well as close all trades in one action
Written by Hadar Cornix Updated over 8 months agoTable of contents
GlossaryBulk ActivateBulk Deactivate Bulk Edit Bulk DeleteBulk Close - Trades
Bulk Actions is a powerful new tool that can help you save time and effort. For example, if you have a large number of DCA bots that you want to deactivate, you can now do so with a single click. Similarly, if you want to make changes to a large number of TradingView bots, you can now do so without having to edit each bot individually!

This is especially useful in times of market volatility when it is important to be able to react quickly to changes in the market. With Bulk Actions, users can quickly deactivate or activate bots as needed, without having to worry about manually adjusting each bot.

This can save you a significant amount of time and effort, allowing you to focus on other aspects of your trading. In addition, Bulk Actions can help to reduce errors, as you are less likely to make mistakes when making changes to multiple bots at once.

Overall, the Bulk Feature is a valuable addition to the Cornix platform.

## Glossary
• Bulk Activate
• Bulk Deactivate
• Bulk Edit
• Bulk Delete
• Bulk Close - Trades

Here are some advantages of the Bulk Actions:

-
Increased efficiency: Bulk Actions can help you to be more efficient by allowing you to manage your bots more quickly and easily.

-
Reduced risk: Bulk Actions can help you to reduce risk by allowing you to quickly deactivate or activate bots in response to market changes.

-
Improved trading results: Bulk Actions can help you to improve your trading results by allowing you to make more informed decisions about your bots.

Let's see how we can easily use the Bulk Actions on our Web Dashboard:


## Bulk Activate
-
Go to the Bots page.
-
Select the Inactive bots that you want to activate.
-
Click on the Activate button.
-
Click on the Yes, Activate Bots button to confirm.

## Bulk Deactivate
-
Go to the Bots page.
-
Select the Active bots that you want to deactivate.
-
Click on the Deactivate button.
-
Choose if you want to keep your open trades or close them.
-
Click on the Yes, Deactivate Bots button to confirm.

## Bulk Edit
-
Go to the Bots page.
-
Select the bots that you want to edit.
-
Click on the Edit button.
-
Make the changes that you want to the bots.
-
Click on the Save Changes button.
-
Click on the Continue button.

## Bulk Delete
-
Go to the Bots page.
-
Select the bots that you want to delete.
-
Click on the Delete button.
-
Click on the OK button to confirm.

## Bulk Close - Trades
-
Go to the Trades page.
-
Select the trades you want to close.
-
Click Close Trades.
-
Choose if you want to keep the coins or sell them at the current price.
-
Click Close Trades again.
