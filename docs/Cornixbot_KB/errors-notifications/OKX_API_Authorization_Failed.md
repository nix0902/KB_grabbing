# OKX API Authorization Failed
Why do I get this error when I try to connect my OKX API Keys?
Written by Hadar Cornix Updated over 4 months agoTable of contents
Step 1: Verify Your API Key Creation ProcessStep 2: Try the Quick Connect FeatureStep 3: Other Causes
If you’re trying to connect your OKX API keys and see the error message “API Authorization Failed,” follow these steps to identify and resolve the issue.

## Step 1: Verify Your API Key Creation Process
Make sure you’ve created your API keys correctly by following this guide:
👉 How to Create API Keys for OKX (Spot & Futures)

## Step 2: Try the Quick Connect Feature
To avoid manual setup errors, you can use the Quick Connect feature.
This method automatically creates and connects your keys safely, with no manual action required.
👉 Connect Your OKX Account Using Quick Connect

## Step 3: Other Causes
If your keys were created correctly and you still encounter the same error, it may be caused by one of the following:
-
Unsupported Region
Cornix only supports the Global OKX website.
Accounts created under us.okx or eu.okx are not supported.

-
Unsupported Account Type
Some OKX account types cannot be connected to Cornix.
You can find more details here:
👉 OKX Account Type Is Not Supported

-
Other Issues
There may be temporary exchange server issues or other account related issues.
If you’ve verified all the points above and still get this error, please reach out to our support team for further assistance.
