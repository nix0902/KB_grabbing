# Allowed Number of Orders per Exchange Exceeded
Written by Hadar Cornix Updated over 2 years ago
Each exchange allows a different number of active orders.

If a trade didn't open or an order was canceled with this status, it means that you reached the exchange limit of the allowed number of active orders.
