# How to Create a Telegram Channel
Written by Dor Updated over 3 years ago
If you already have a Telegram channel then skip this part and start with Add the Cornix Trading Bot and Enable Auto-Trading in your Channel.

To create a new channel, you can use any Telegram interface, like Telegram Desktop or similar, and select New Channel. It’s very important to create a channel and not a group, as Cornix supports Telegram channels and not groups.
