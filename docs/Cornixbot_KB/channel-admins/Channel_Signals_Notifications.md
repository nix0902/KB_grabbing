# 📢 Update: New Cleaner Notification System
Starting December 3rd, we introduced an improved channel notification system, offering clearer and more concise updates.

## What’s New?
-
Notifications are now shorter, clearer, and easier to follow.
-
Notifications no longer include updates from multiple exchanges at once.
-
Instead, the system selects one main exchange for the signal and sends notifications only for that exchange.
-
Other exchanges will not send direct notifications in the channel.
Members who trade on other exchanges can still track the signal’s progress at any time by viewing the Signal Details Page and selecting their preferred exchange.

## Exchange Prioritization Order
Notifications will be sent based on the first available exchange in the following priority list:
Binance Futures, Bybit USDT, OKX Futures, KuCoin Futures, Bitget Futures, Hyperliquid Futures, BingX Futures, Bybit Inverse, Binance Coin-M, OKX Inverse, BitMEX Inverse, Bitget Inverse, Binance Spot, Bybit Spot, KuCoin Spot, Huobi Spot, OKX Spot, Bitget Spot, Coinbase Spot, BingX Spot.
-
The new notification system is enabled by default for new channels that integrated Cornix after December 3rd.
-
Existing channels can request to enable the new version at any time.

# What are channel signal notifications?
Channel signal notifications provide real-time updates to your channel members about the progress of each signal. These updates include information such as target achievements, potential profit/loss, and other real-time status changes of the signals.

Please note that all channel signal notifications are theoretical.
Notifications appear when the price reaches a target on the exchange, but they do not indicate whether a user’s real order has been filled. Execution depends on market conditions, liquidity, and individual user settings.
Therefore, if your channel members use auto-trading, order execution times may differ from the notification times due to supply and demand in the order book.
For more details, please see our article about Trade Results vs. Channel Results.

# Notification Types
The following types of notifications may be sent to your channel:
-
Signal Cancelled (by the admin)
-
Entry Target Achieved
-
All Entry Targets Achieved
-
Signal Entered the Entry Zone
-
Take-Profit Target Achieved
-
All Take-Profit Targets Achieved
-
Stop Loss Target Achieved
-
Stop Loss Hit After Take-Profit
-
Trailing Stop Loss Hit After Take-Profit
-
Trailing Take-Profit
-
Trailing Stop Loss
-
Signal Closed Due to Opposite Direction Signal Opened

# How to Disable or Customize Notifications
Some notification types can be disabled upon request, these are:
-
Take-Profit Notifications
-
'Take-Profit Target Achieved', 'All Take-Profit Targets Achieved' and 'Trailing Take-Profit'.
-
Entry Notifications
-
'Entry Target Achieved', 'All Entry Targets Achieved', 'Signal Entered the Entry Zone'.
-
Stop Loss Notifications
-
'Stop Loss Target Achieved', 'Stop Loss Hit After Take-Profit', 'Trailing Stop Loss', 'Trailing Stop Loss Hit After Take-Profit'.

Channel admins can request changes through Cornix Support using any of the platforms: Web Dashboard, Telegram or Mobile app.
