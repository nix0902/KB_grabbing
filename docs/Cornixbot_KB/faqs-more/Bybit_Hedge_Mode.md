# Bybit Hedge Mode
How to change from one-way mode to two-way mode on ByBit
Written by Hadar Cornix Updated over 3 years ago
In this article, we will demonstrate how to change your trading preference from one-way mode to two-way mode (Hedge Mode).

For individuals who are using a Bybit Unified Margin Account (please start here), in order to switch between one-way mode to two-way mode (Hedge Mode), you simply need to switch to your 'Subaccount'.

To enable the Hedge mode on the Bybit exchange you need to do the following:

-
Select the settings icon in your Bybit trading terminal.

*Please note that this change is per pair (Since 28.11.2022).
2. Select 'Hedge Mode' to trade this pair as seen in the picture below.



You're all set :)
