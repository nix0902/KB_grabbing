# Supported Exchanges
Written by Hadar Cornix Updated over 3 months agoTable of contents
Cornix currently supports 10 spot and futures exchanges:

# Spot Exchanges:
-
Binance
-
ByBit
-
OKX (only okx.com)
-
Bitget
-
KuCoin
-
BingX
-
Coinbase
-
Huobi.pro

# Futures Exchanges:
-
Binance
-
ByBit
-
OKX (only okx.com)
-
Bitget
-
KuCoin
-
BingX
-
HyperLiquid
-
BloFin

# Inverse Exchanges:
-
Binance
-
ByBit
-
OKX(only okx.com)
-
Bitget
-
BitMEX
-
BloFin

Note - the following are currently not supported by Cornix:
-
okx.eu and okx.us
-
ByBit.eu
-
Binance.us

Looking for an exchange that's not on the list?
Let us know by submitting a feature request on our official feedback page!
