# Binance Hedge Mode
How to change from one-way mode to two-way mode on Binance
Written by Hadar Cornix Updated over 3 years ago
In this article, we will demonstrate how to change your trading preference from one-way mode to two-way mode (Hedge Mode).

To enable the Hedge mode on the Binance exchange you need to do the following:

-
Select the 'Filter' icon in your Binance trading terminal.

2. Select 'Preference' in the options list.




3. Tap on 'Position Mode' and choose the 'Hedge Mode' option.

You're all set :)
