<h1>User Data Streams for Binance Spot TESTNET</h1>
<h2 class="anchor anchorWithStickyNavbar_fMI7" id="general-information">General information<a class="hash-link" aria-label="Direct link to General information" title="Direct link to General information" href="/docs/binance-spot-api-docs/testnet/user-data-stream#general-information">​</a></h2>
<ul>
<li>Subscribe via the <a href="/docs/binance-spot-api-docs/testnet/websocket-api/user-data-stream-requests#user-data-stream-subscribe">WebSocket API</a> using an API Key.</li>
<li>Both <a href="/docs/binance-spot-api-docs/faqs/sbe_faq">SBE</a> and JSON output are supported.</li>
<li>Account events are pushed in <strong>real-time</strong>.</li>
<li>All timestamps in JSON payloads are in <strong>milliseconds by default</strong>.</li>
<li>Events may contain non-ASCII characters encoded in UTF-8 if you own or trade any assets or symbols whose names contain non-ASCII characters.</li>
</ul>

<h4 class="anchor anchorWithStickyNavbar_fMI7" id="execution-types">Execution types<a class="hash-link" aria-label="Direct link to Execution types" title="Direct link to Execution types" href="/docs/binance-spot-api-docs/testnet/user-data-stream#execution-types">​</a></h4>
<ul>
<li><code>NEW</code> - The order has been accepted into the engine.</li>
<li><code>CANCELED</code> - The order has been canceled by the user.</li>
<li><code>REPLACED</code> - The order has been amended.</li>
<li><code>REJECTED</code> - The order has been rejected and was not processed. (e.g. Cancel Replace orders where the new order placement was rejected even if the cancel request succeeded.)</li>
<li><code>TRADE</code> - Part of the order or all of the order&#x27;s quantity has filled.</li>
<li><code>EXPIRED</code> - The order was canceled according to the order type&#x27;s rules (e.g. LIMIT FOK orders with no fill, LIMIT IOC or MARKET orders that partially fill) or by the exchange, (e.g. orders canceled during liquidation, orders canceled during maintenance).</li>
<li><code>TRADE_PREVENTION</code> - The order has expired due to STP.</li>
</ul>
<p>Check the <a href="/docs/binance-spot-api-docs/testnet/enums">Enums Documentation</a> for more relevant enum definitions.</p>
